# Recruiter/TA Stack

Talent acquisition and recruitment workflow automation for Claude Code.

## What's Here

- **skills/** — Job description, candidate screening, interview prep, offer workflows
- **commands/** — Custom CLI tools for TA processes
- **hooks/** — Automated candidate updates, interview reminders, offer tracking
- **mcp/** — LinkedIn, Greenhouse, Workable, ATS integrations
- **session-log.md** — Workflow session history

## Quick Start

1. Browse available skills in `skills/`
2. Use `/skill-name` to invoke in Claude Code
3. See `CLAUDE.md` for detailed naming and structure

## Integration Points

- LinkedIn API (candidate sourcing)
- ATS systems (Greenhouse, Workable, Lever)
- Calendar systems (interview scheduling)
- Email (candidate communication)
- Slack (hiring team notifications)
