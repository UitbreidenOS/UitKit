# audit-seo Command

Run comprehensive SEO audit on existing content.

## Usage

```
/audit-seo --url "https://example.com/post" --report-type "full|quick"
```

## Parameters

- `--url` — Content URL to audit
- `--report-type` — Audit depth: "full" or "quick" (default: full)

## Output

SEO audit report with scores, recommendations, and priority fixes.
