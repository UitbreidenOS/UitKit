# generate-post Command

Create a blog post from outline and target keywords.

## Usage

```
/generate-post --title "Post Title" --keywords "kw1, kw2" --outline "Section 1, Section 2"
```

## Parameters

- `--title` — Blog post title
- `--keywords` — Comma-separated SEO keywords
- `--outline` — Content structure (sections)

## Output

Formatted blog post with frontmatter and SEO metadata.
