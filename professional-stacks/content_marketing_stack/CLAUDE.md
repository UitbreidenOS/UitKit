# Content Marketing Stack — Workspace Config

## Key Principles

1. **Content Quality First** — All generated content undergoes SEO validation and brand voice review before publication.
2. **Data-Driven Decisions** — Campaign performance is tracked via integrated analytics; optimization decisions backed by metrics.
3. **Distribution Automation** — Content is synchronized across channels (email, social, blog) via hooks and MCP integrations.
