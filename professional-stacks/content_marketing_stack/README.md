# Content Marketing Stack

Fast, integrated tooling for creating, optimizing, and distributing marketing content.

## Skills

- `seo-optimization.md` — Validate and optimize content for search engines
- `copy-quality.md` — Review brand voice, tone, and messaging clarity
- `email-campaign.md` — Generate and schedule email campaigns
- `social-media.md` — Create platform-specific social content
- `content-calendar.md` — Plan and coordinate multi-channel campaigns
- `analytics-sync.md` — Pull metrics and generate performance reports
- `competitor-analysis.md` — Track competitor content and positioning
- `audience-segmentation.md` — Define and test audience targeting

## Commands

- `generate-post` — Create blog post from outline
- `publish-batch` — Schedule multiple pieces across channels
- `audit-seo` — Run full SEO audit on existing content

## Hooks

- `on-publish` — Sync content to distribution channels
- `on-schedule` — Trigger campaign deployment
- `on-analytics-sync` — Update performance dashboards

## MCP Integrations

- `wordpress-plugin` — Blog publishing and scheduling
- `analytics-connector` — GA4, Stripe, customer data APIs
