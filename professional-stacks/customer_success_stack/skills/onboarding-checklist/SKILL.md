# Skill: Onboarding Checklist

## When to activate

New customer sign-up or upgrade to new plan tier.

## When NOT to use

Do not apply to existing customers mid-contract (use success plan instead).

## Instructions

1. Generate role-based checklist (admin, user, integrator)
2. Track completion via automated or manual checkmarks
3. Alert on blockers (missing integration, unpaid invoice)
4. Measure time-to-value (days to first use)

## Example

```
/onboard-checklist --customer=startup-xyz --plan=pro
→ 12 items; 8 complete, 4 pending
→ Blocker: Salesforce integration auth
```
