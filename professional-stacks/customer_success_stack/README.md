# Customer Success Stack

Tooling and workflows for proactive customer health, retention, and expansion.

## Skills

- `account-health` — Health score aggregation and trending
- `renewal-forecast` — SaaS renewal prediction and pipeline
- `churn-detection` — Engagement drop detection and alerts
- `escalation-routing` — Issue routing by severity and team
- `satisfaction-tracking` — NPS and feedback aggregation
- `expansion-opportunity` — Upsell/cross-sell identification
- `onboarding-checklist` — New customer setup verification
- `retention-playbook` — At-risk customer engagement scripts

## Quick Start

```bash
claude /cs-health-check --account=acme-corp
claude /renewal-forecast --cohort=2024-q3
```
