# Mobile Developer Stack

Cross-platform and native mobile development workspace — React Native, Flutter, SwiftUI, Jetpack Compose, and app deployment.

## Quick Start

```bash
npx claudient add mobile_developer_stack
```

## Skills

| Skill | Description |
|---|---|
| app-architecture | Design mobile app architecture and navigation |
| react-native-expo | React Native + Expo development |
| flutter-widgets | Flutter widget composition and state management |
| ios-swift-ui | SwiftUI and iOS platform integration |
| android-compose | Jetpack Compose and Android architecture |
| push-notifications | Push notification implementation |
| offline-first | Offline-first data sync strategies |
| app-store-deploy | App Store and Play Store submission |
