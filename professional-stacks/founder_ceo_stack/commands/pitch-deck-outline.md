# Command: pitch-deck-outline

## Purpose
Quick outline and structure for investor pitch deck. Returns slide order, key talking points, and visual guidance.

## Usage
```
/pitch-deck-outline [--round seed|seriesA|seriesB] [--audience vcs|corporate|angels]
```

## Output Format
1. Problem & Opportunity
2. Solution & Product
3. Team & Traction
4. Market Size & Go-to-Market
5. Financials & Ask
6. Key Talking Points per Slide
7. Visual Guidance (charts, demos, narrative flow)

## Notes
- Assumes ~10-12 slides for seed/Series A
- Customize for different audience focus
- See `fundraising-strategy` skill for deeper messaging
