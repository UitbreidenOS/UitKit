# Founder/CEO Stack

A tactical intelligence stack for founders and CEOs. Skills, commands, and integrations for fundraising, team building, strategic planning, and operational execution.

## Quick Start
1. Review `CLAUDE.md` for workspace principles and model guidance
2. Invoke a skill: `/fundraising-strategy`, `/team-hiring-guide`, `/market-analysis`
3. Run a command: `/pitch-deck-outline`, `/quarterly-review`
4. Enable hooks via `settings.json` for automatic: weekly digests, pre-meeting prep, investor tracking

## Skills (8 domains)

| Skill | Purpose |
|-------|---------|
| fundraising-strategy | Pitch strategy, cap table, investor targeting |
| product-strategy | Product-market fit, roadmap prioritization, positioning |
| team-building | Hiring, culture, scaling from 5→50 people |
| investor-relations | Investor communication, board meetings, reporting |
| market-analysis | Competitive landscape, TAM/SAM, go-to-market sizing |
| go-to-market | Launch planning, sales strategy, pricing |
| financial-planning | Runway, unit economics, financial modeling |
| hiring-strategy | Recruiting funnel, compensation, retention |

## Commands (3 quick tools)

- `pitch-deck-outline` — Structure and outline for investor pitch
- `investor-tracker` — Log, track, and follow up with investors
- `quarterly-review` — Q-end review template and decision frame

## Hooks (3 automations)

- `pre-meeting-prep` — Auto-brief before investor/board meetings
- `weekly-digest` — Weekly ops summary and action items
- `investor-outreach` — Outreach reminder and tracking

## MCP Tools

- **CRM Integration** — Log investor interactions, pipeline management
- **Financial Tools** — Revenue projections, cap table calculator
