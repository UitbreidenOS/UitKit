# Product Strategy

## When to activate
- Defining product roadmap or priorities
- Evaluating product-market fit signals
- Deciding on feature/market focus
- Positioning product against competitors

## When NOT to use
- Technical implementation details (use engineering)
- UX/design critiques (use design review)
- Customer support decisions

## Instructions
1. Map current product state and usage data
2. Identify market/customer segment opportunities
3. Prioritize via impact, effort, and strategic fit
4. Draft roadmap with key milestones
5. Define success metrics per phase

## Example
User asks: "Should we expand to SMB or double down on enterprise?"
Skill outputs: TAM/SAM comparison, resource cost, go-to-market implications, recommendation framework
