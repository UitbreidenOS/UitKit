# Financial Planning

## When to activate
- Modeling revenue, costs, and runway
- Planning unit economics and margin targets
- Forecasting cash flow and capital needs
- Evaluating cost structure and efficiency

## When NOT to use
- Accounting/tax questions (use accountant)
- Personal finance advice
- Detailed P&L reconciliation with actuals

## Instructions
1. Map current financial state (revenue, burn, runway)
2. Build model with assumptions (growth, unit economics, cost drivers)
3. Project 12-24 month scenarios (conservative, base, aggressive)
4. Identify breakeven point and capital needs
5. Stress-test key assumptions

## Example
User asks: "How long is our runway with current burn?"
Skill outputs: Runway calculation, scenario analysis, unit economics breakdown, recommended cost actions
