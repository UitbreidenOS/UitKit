# Founder/CEO Stack — Workspace Config

## Purpose
A curated set of skills, commands, hooks, and MCP tools for founders and CEOs managing startups, fundraising, team building, and strategic operations.

## Key Principles

1. **Decision Clarity** — Every skill surfaces the core decision or trade-off, with supporting data. No fluff.
2. **Execution Speed** — Commands prioritize iteration speed and measurable outcomes over perfection.
3. **Scalable Mental Model** — Tools and frameworks remain useful from pre-seed through Series A+.

## Model Guidance
- Default: Sonnet (multi-step reasoning for strategic questions)
- Haiku for: quick lookups, log entries, routine status updates
- Opus for: deep competitive analysis, financial modeling, board-level strategic synthesis

## When to Use This Stack
- Building/operating a startup (founder, CEO, operator)
- Preparing investor materials or pitches
- Making critical hiring or capital allocation decisions
- Planning quarterly/annual strategy
