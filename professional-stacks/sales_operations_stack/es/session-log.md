# Registro de Sesiones — Sales Operations Stack

> Registro de actividades actualizado automáticamente para revisiones de pipeline, análisis de territorios, pronósticos, seguimiento de cuotas y decisiones operacionales. Indexado por fecha para referencia histórica.

---

## Cómo Usar Este Registro

**Auto-logging:** Las acciones clave se registran automáticamente cuando ejecutas comandos (`/analyze-pipeline`, `/optimize-territory`, `/build-forecast`).

**Entradas manuales:** Agrega observaciones, decisiones o seguimientos manualmente en el formato a continuación.

**Búsqueda:** Usa Ctrl+F para encontrar representantes, cuentas o decisiones por nombre o fecha.

**Archivo:** Al final del mes, este archivo se convierte en tu registro de auditoría para documentación de cumplimiento y decisiones.
