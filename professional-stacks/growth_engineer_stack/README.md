# Growth Engineer Stack

> Autonomous growth metrics analysis — SaaS funnel optimization, retention cohort analysis, viral loop design, and channel ROI audit.

---

## Quick Start

1. **Copy this folder** into your Claude Code workspace.
2. **Run `/analyze-funnel`** — Get AARRR breakdown with drop-off analysis.
3. **Run `/analyze-retention`** — View cohort curves and churn patterns.
4. **Run `/audit-channels`** — Compare CAC/LTV by acquisition channel.

## Skills (5)

| Skill | Trigger | Purpose |
|---|---|---|
| `funnel-analyzer` | `/analyze-funnel` | AARRR metrics, drop-off analysis, top 3 levers |
| `retention-researcher` | `/analyze-retention` | Cohort curves, churn analysis, L7/L30/L90 trends |
| `growth-channel-auditor` | `/audit-channels` | CAC/LTV by channel, reallocation recommendations |
| `viral-loop-designer` | `/design-viral` | Referral mechanics, viral coefficient, network effects |
| `retention-lever-finder` | `/find-levers` | Top 3 retention drivers, prioritized test plan |

---

## Commands (5)

| Command | What It Does |
|---|---|
| `/analyze-funnel` | Acquisition → Activation → Retention → Revenue → Referral breakdown. Identifies drop-offs. |
| `/analyze-retention` | Cohort curves (L7, L30, L90), churn reasons, payback period analysis. |
| `/audit-channels` | CAC, LTV, payback by channel. Recommends budget reallocation. |
| `/design-viral` | Referral loop audit, viral coefficient (K), network effect analysis. |
| `/find-levers` | Identifies and prioritizes top 3 retention drivers with test plan and success metrics. |

---

## Best Practices

1. **Always include current metric → target metric → ROI** in any recommendation.
2. **Prioritize by effort/impact.** Frame as hypothesis and test plan.
3. **Use cohort analysis** to isolate seasonality and product changes.
4. **Log all findings to `session-log.md`** for decision audit trail.
