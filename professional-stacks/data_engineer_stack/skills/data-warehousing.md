# Data Warehousing

## When to activate

Designing star schemas, OLAP tables, partitioning strategies, or cost optimization.

## When NOT to use

Not for OLTP design; focus on analytic workloads.

## Instructions

1. Design dimension and fact tables
2. Plan partitioning (date, region, customer)
3. Estimate storage and cost
4. Document refresh cadence
