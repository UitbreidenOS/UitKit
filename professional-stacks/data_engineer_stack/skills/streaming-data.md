# Streaming Data

## When to activate

Designing event streams, Kafka topics, Pub/Sub subscriptions, or real-time ingestion.

## When NOT to use

Not for batch-only systems; focus on low-latency architectures.

## Instructions

1. Define event schema (Avro, Protobuf)
2. Plan partitioning and retention
3. Size consumer groups
4. Handle backpressure and ordering
