# Data Engineer Stack

A Claude Code workspace for data pipeline design, orchestration, debugging, and optimization.

## Skills

- `sql-optimization` — Query performance tuning and indexing strategies
- `pipeline-design` — DAG design and orchestration patterns
- `etl-debugging` — Data flow debugging and bottleneck analysis
- `schema-migration` — Safe schema changes and backwards compatibility
- `performance-tuning` — CPU, memory, and I/O optimization
- `data-validation` — Quality checks and anomaly detection
- `streaming-data` — Event stream design and Kafka/Pub-Sub patterns
- `data-warehousing` — OLAP design, partitioning, and cost optimization
