# Blockchain & Web3 Developer Stack

Decentralized application development workspace — smart contracts, DeFi, NFTs, DAOs, bridges, and token economics.

## Quick Start

```bash
npx claudient add blockchain_web3_stack
```

| Skill | Description |

| cross-chain-bridge | Cross-chain messaging and asset bridging |
| wallet-integration | Web3 wallet connection and signing |
| gas-optimization | Solidity gas cost reduction techniques |
| token-economics | Tokenomics design and modeling |
