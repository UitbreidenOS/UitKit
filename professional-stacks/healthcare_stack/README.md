# Healthcare Stack

Clinical operations, compliance, and patient care workflow — from intake to outcomes.

## Overview

This stack provides specialized tooling and guidance for:

- **Clinical decision support** — CDS rule design with alert fatigue mitigation
- **HIPAA compliance** — Privacy and Security Rule gap analysis
- **Patient intake** — Workflow optimization and bottleneck elimination
- **EHR integration** — HL7 FHIR, C-CDA, X12 interface design
- **Telehealth** — Virtual care program design with licensing and reimbursement
- **Medical coding** — ICD-10, CPT, HCPCS accuracy validation
- **Clinical trials** — Enrollment tracking, safety monitoring, regulatory milestones
- **Population health** — Risk stratification, disparity analysis, intervention design
