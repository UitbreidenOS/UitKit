# API Developer Stack

A collection of skills, agents, guides, and workflows for API development in Claude Code.

## Directory Structure

- **guides/** — Human-readable documentation and tutorials for API development patterns
- **skills/** — Reusable slash commands and skill definitions for API tasks
- **agents/** — Specialized subagents for API development roles and domains
- **workflows/** — End-to-end API development process documentation
- **prompts/** — Reusable prompt templates for API-related tasks
- **rules/** — Guidelines and best practices for API development

## Getting Started

Add skills and agents from this stack to your Claude Code configuration to enhance your API development workflow.
