# Embedded & IoT Developer Stack

Embedded systems and IoT development workspace — firmware, RTOS, BLE, sensors, low-power design, and cloud connectivity.

## Quick Start

```bash
npx claudient add embedded_iot_stack
```

## Skills

| Skill | Description |
|---|---|
| firmware-architecture | HAL, drivers, and application layers |
| rtos-patterns | FreeRTOS/Zephyr task design and IPC |
| ble-connectivity | BLE GATT, pairing, and mesh |
| iot-cloud-platform | AWS IoT, Azure IoT, MQTT, fleet management |
| sensor-integration | Sensor drivers, calibration, and fusion |
| low-power-design | Power profiling and battery optimization |
| ota-updates | Secure over-the-air firmware updates |
| embedded-testing | Unit testing, HIL, and CI/CD for firmware |
