# Bio-Research Stack

A rigorous, reproducible biological research workflow — from hypothesis to publication.

## Overview

This stack provides specialized tooling and guidance for:

- **Experimental design** — Power analysis, controls, randomization, blinding
- **Literature review** — Systematic search strategies, evidence synthesis
- **Statistical analysis** — Test selection, assumption checks, effect sizes
- **Lab documentation** — Structured notebook entries following FAIR principles
- **Bioinformatics** — NGS pipeline design, QC checkpoints, reproducibility
- **Grant writing** — NIH/NSF-style proposals with Specific Aims structure
- **Peer review** — Simulated reviewer feedback before submission
- **Reproducibility** — Pre-submission audits for data, code, and methods

## Skills

| Skill | Purpose |
|-------|---------|
| experiment-protocol-designer | Design experiments with power analysis and controls |
| literature-reviewer | Systematic literature reviews with evidence tables |
| statistical-analysis-advisor | Statistical test selection and result interpretation |
| lab-notebook-writer | Structured electronic lab notebook entries |
| bioinformatics-pipeline-builder | NGS analysis pipelines with QC checkpoints |
| grant-proposal-drafter | NIH/NSF-style grant proposals |
| peer-review-simulator | Simulated peer review with severity-rated comments |
| reproducibility-auditor | Pre-submission reproducibility compliance scorecard |
