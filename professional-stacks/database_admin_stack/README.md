# Database Admin Stack

A Claude Code workspace for database user management, performance optimization, backup/recovery, and compliance monitoring.

## Skills

- `user-access-control` — Create/drop users, manage roles, audit permissions
- `performance-tuning` — Identify slow queries, optimize indexes, tune parameters
- `backup-recovery` — Schedule backups, verify integrity, test recovery procedures
- `replication-monitoring` — Set up and monitor primary/replica replication, lag alerts
- `compliance-auditing` — Verify access logs, encryption, retention policies
- `incident-response` — Diagnose outages, failed connections, corruption detection
- `schema-migration` — Safe DDL changes, rollback procedures, zero-downtime deployment
- `capacity-planning` — Monitor resources, forecast growth, scaling needs
