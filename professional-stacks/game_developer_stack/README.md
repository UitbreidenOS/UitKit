# Game Developer Stack

Game development workspace — Unity, Unreal, Godot, networking, physics, level design, and performance optimization.

## Quick Start

```bash
npx claudient add game_developer_stack
```

## Skills

| Skill | Description |
|---|---|
| game-architecture | Game architecture patterns (ECS, FSM, components) |
| unity-csharp | Unity C# development and DOTS/ECS |
| unreal-cpp | Unreal Engine C++ and Blueprint integration |
| godot-gdscript | Godot 4 GDScript/C# development |
| game-networking | Multiplayer networking and rollback netcode |
| game-physics | Physics systems and collision design |
| level-design | Level design methodology and documentation |
| game-performance | Performance profiling and optimization |
