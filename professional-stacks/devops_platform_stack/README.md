# DevOps/Platform Stack

Unified workspace for Kubernetes, infrastructure, CI/CD, monitoring, and platform reliability.

## Skills

- `kubernetes-deployment` — Deploy and manage Kubernetes workloads
- `terraform-modules` — Write and review IaC configurations
- `docker-optimization` — Optimize container images and runtime
- `ci-cd-pipelines` — Design and debug deployment pipelines
- `prometheus-alerting` — Configure observability and alerts
- `elk-logging` — Deploy and query logging infrastructure
- `security-scanning` — Run and interpret security tools
- `infrastructure-disaster-recovery` — Design and test failover scenarios

## Commands

- `/k8s-debug` — Diagnose pod and cluster issues
- `/terraform-plan` — Review infrastructure changes
- `/logs-search` — Query logs across services

## Hooks

- `pre-deploy` — Validate infrastructure before applying changes
- `post-incident` — Generate incident postmortems
- `daily-health-check` — Run automated cluster health scans

## MCP Tools

- Kubernetes API integrations
- Terraform state inspection tools
