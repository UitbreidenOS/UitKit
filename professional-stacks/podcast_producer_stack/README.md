# Podcast Producer Stack

A collection of skills, agents, guides, and workflows for podcast production in Claude Code.

## Directory Structure

- **guides/** — Human-readable documentation and tutorials for podcast production workflows
- **skills/** — Reusable slash commands and skill definitions for podcast tasks
- **agents/** — Specialized subagents for podcast production roles and domains
- **workflows/** — End-to-end podcast production process documentation
- **prompts/** — Reusable prompt templates for podcast-related tasks
- **rules/** — Guidelines and best practices for podcast production

## Getting Started

Add skills and agents from this stack to your Claude Code configuration to enhance your podcast production workflow.
