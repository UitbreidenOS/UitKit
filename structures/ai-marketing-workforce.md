# 📂 AI Marketing Workforce

> The canonical workspace for a multi-agent SaaS platform where marketing and growth operations are completely automated by virtual AI employees, designed to drive top-of-funnel traffic and manage website assets autonomously.

📄 `growth-architecture-brief.md`  # Canonical brief: Target audience, customer acquisition cost (CAC) limits, and omni-channel distribution strategy
🧠 `active-campaign-memory.md`     # Session memory: Dynamic context tracking for current A/B tests and live social media engagement rates
🤖 `CLAUDE.md`                     # Operating rules: Strict instructions to enforce absolute color preservation during image processing and ban hallucinated branding from generated visual assets

## 📁 virtual-employee-personas/ (4 skills - The Team)
📄 `content-strategist-agent.md`   # The supervisor agent that orchestrates the weekly content calendar and delegates tasks to specialized sub-agents
📄 `technical-copywriter-bot.md`   # Translates dense system updates into highly readable Medium blog posts and scroll-stopping LinkedIn hooks
📄 `video-storyboarder-agent.md`   # Drafts high-energy scripts and shot lists for Instagram Reels, specifically optimizing technical unboxings (e.g., revealing an Nvidia DGX Spark or a Bambulab H2d Pro)
📄 `zaltaclaw-execution-loop.md`   # The autonomous engine running locally on a dedicated Mac mini, executing background SEO audits and link-building outreach overnight

## 📁 website-asset-engine/ (3 skills - Visual Integrity)
📄 `painting-color-preserver.md`   # Strict image-enhancement guardrails: absolutely guarantees that when upscaling product images or paintings for the website, the AI only improves resolution and 100% preserves the original colors
📄 `texture-and-branding-rules.md` # Guardrails for 3D/lifestyle rendering: explicitly forces hyper-realistic surfaces (e.g., a natural green coconut texture) and ruthlessly scrubs out any hallucinated third-party branding or monster logos
📄 `dynamic-og-image-gen.md`       # Automatically generates customized OpenGraph metadata images for every new blog post or product page

## 📁 distribution-and-social/ (3 skills - Syndication)
📄 `cross-platform-scheduler.md`   # API mappings that format and push a single core message natively to LinkedIn, Twitter, and Instagram simultaneously
📄 `youtube-blender-pipeline.md`   # Prompts and scripts for autonomous agents to generate 3D background scenes and tutorial thumbnails using Blender
📄 `community-engagement-bot.md`   # Drafts authentic, non-robotic replies to comments on YouTube and Medium to boost algorithmic reach

## 📁 analytics-and-optimization/ (3 skills - Measurement)
📄 `funnel-conversion-tracker.md`  # Maps social media impressions directly to sign-ups on your frontend dashboard
📄 `ab-test-orchestrator.md`       # Automatically rotates landing page headlines and CTA buttons to find the highest converting variant
📄 `token-roi-calculator.md`       # Compares the API cost of the virtual marketing employees against the actual revenue generated from their campaigns

## 📁 core-platform-sync/ (3 skills - CI/CD)
📄 `frontend-health-ui-sync.md`    # Automated scripts allowing the AI marketing agents to securely push updated landing page copy and optimized images directly to the frontend repository
📄 `backend-health-api-sync.md`    # Logic for syncing new email capture endpoints and webhook analytics to the core backend
📄 `github-final-sync.md`          # Final CI/CD action that validates the autonomous agents' code and merges it cleanly into the Github final repos
