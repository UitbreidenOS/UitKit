# 📂 SDR Prospecting Hub

> The canonical workspace for a Sales Development Representative (SDR) or BDR, designed to automate account research, list building, and hyper-personalized outreach sequences.

📄 `icp-definition.md`         # Canonical brief: Ideal Customer Profile, buyer personas, and negative signals
🧠 `active-pipeline.md`        # Session memory: Dynamic context tracking for prospects currently in sequence
🤖 `CLAUDE.md`                 # Operating rules: Strict instructions to avoid corporate jargon and keep cold emails under 75 words

## 📁 account-research/ (4 skills - Pre-Call Intelligence)
📄 `intent-signal-parser.md`   # Tracks job changes, funding rounds, and recent news mentions
📄 `10k-analyzer.md`           # Pulls strategic priorities from annual reports (for enterprise targets)
📄 `tech-stack-detector.md`    # Identifies current tools the prospect uses via BuiltWith/Wappalyzer data
📄 `persona-mapper.md`         # Maps specific pain points to the prospect's exact job title

## 📁 outreach-sequences/ (3 skills - Multi-Channel Touches)
📄 `cold-email-generator.md`   # Drafts personalized first-touch emails using the account research data
📄 `linkedin-dm-drafter.md`    # Creates casual, non-salesy connection requests and voice note scripts
📄 `cold-call-scripts.md`      # Dynamic talk tracks and opening hooks based on industry

## 📁 objection-handling/ (3 skills - Battlecards)
📄 `competitor-kill-sheets.md` # Quick-reference bullet points on how to differentiate from top 3 rivals
📄 `budget-pushback.md`        # Rebuttals for "we don't have budget" or "call me in Q4"
📄 `security-concerns.md`      # One-liners addressing SOC2 and data privacy hesitations

## 📁 pipeline-management/ (3 skills - CRM Sync)
📄 `salesforce-sync.md`        # Automated formatting of meeting notes for direct CRM paste
📄 `follow-up-scheduler.md`    # Triggers for 3-day and 7-day bump emails
📄 `disqualification-log.md`   # Tracks exactly why an account was closed-lost for future marketing alignment

## 📁 weekly-reporting/ (2 skills - Performance)
📄 `conversion-tracker.md`     # Tracks open rates, reply rates, and meeting booked metrics
📄 `a-b-test-evaluator.md`     # Compares Subject Line A vs Subject Line B performance

---
**Configuration Files**
⚙️ `apollo-filters.json`       # Saved boolean search strings for automated list building
📦 `sequence-timeline.yaml`    # The exact day-by-day cadence map (Email -> Call -> LinkedIn)
