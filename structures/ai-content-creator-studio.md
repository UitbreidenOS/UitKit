# 📂 AI Content Creator Studio

> The canonical workspace for a Technical Content Creator and AI Builder, designed to automate multi-channel syndication, streamline video unboxings, and maintain strict brand aesthetics across YouTube, Instagram, Medium, and LinkedIn.

📄 `content-strategy-brief.md` # Canonical brief: Defines audience personas, core technical pillars, and omni-channel release schedules
🧠 `active-campaign-memory.md` # Session memory: Dynamic context tracking for current draft statuses and trending tech news
🤖 `CLAUDE.md`                 # Operating rules: Strict instructions to never hallucinate technical facts and to explicitly preserve original baseline colors when enhancing or upscaling product reference images (e.g., website paintings or textures)

## 📁 video-and-reels-engine/ (4 skills - Visual Storytelling)

📄 `blender-youtube-scenes.md` # Prompts and Python scripts for auto-generating Blender animations and 3D scenes for deep-dive YouTube tutorials
📄 `retention-hook-writer.md`  # Frameworks for the first 3 seconds of a Reel to maximize viewer retention and prevent scrolling

## 📁 written-content-pipeline/ (3 skills - Copy & Blogs)

📄 `linkedin-growth-hooks.md`  # Generates punchy, scroll-stopping LinkedIn text posts and custom banner dimension specs to drive professional engagement
📄 `technical-tutorial-drafter.md` # Converts raw API documentation and code snippets into step-by-step written guides

## 📁 asset-and-design-ops/ (3 skills - Branding)
📄 `image-enhancement-guardrails.md` # Strict rules for AI upscaling workflows to ensure original art and painting colors are 100% preserved while only improving resolution for website product listings
📄 `thumbnail-ab-tester.md`    # Frameworks for generating high-contrast, clickable YouTube thumbnails with varying text overlays
📄 `brand-voice-aligner.md`    # Checks all generated copy to ensure the tone matches the creator's authentic, developer-focused voice rather than generic AI speak

## 📁 publishing-and-analytics/ (3 skills - Distribution)
📄 `cross-platform-scheduler.md`# API mapping to automatically push the same core message formatted natively for Twitter, LinkedIn, and Instagram
