# Content Repurposer Skill

## When to activate
- You have a blog post, podcast transcript, or article and want to get more from it
- Creating a week's worth of social content from a single source
- Turning a client case study into multiple formats
- Repurposing a long-form video into clips, captions, and a newsletter
- You have 30 minutes and need content for multiple channels

## When NOT to use
- Creating brand-new content from scratch — describe what you want instead
- Repurposing competitor content — legal risk
- Content that's more than 2 years old — check accuracy first

## Instructions

### The one-to-many workflow

Give Claude your source content and the outputs you need:

```
Here's a blog post I wrote about [topic]: [paste content]

Please create:
1. 5 LinkedIn posts (professional, one insight per post)
2. 5 Instagram captions with hashtag suggestions
3. An email newsletter version (300 words, conversational)
4. A Google Ads headline and description
5. A 60-second video script
```

### Formats Claude can produce

| Format | What to specify |
|---|---|
| LinkedIn post | Tone (professional/personal), length, include a question? |
| Instagram caption | Vibe (inspirational/educational/fun), hashtag count |
| Email newsletter | Length, subject line, CTA |
| Twitter/X thread | Number of tweets, hook style |
| Facebook post | Community-focused, conversational |
| Google Ads | Character limits auto-applied |
| YouTube description | SEO-focused, with timestamps |
| Video script | Duration, talking-head or voiceover |
| Podcast show notes | Summary + key points + timestamps |
| Pinterest pin | Description + keyword-rich |

### For a service business

```
I'm a plumber. Here's a case study about a job I did last week:
"Fixed a burst pipe for a homeowner at 2am on a Sunday, saved their kitchen from flooding"

Turn this into:
- A Google review request (to send to the customer)
- A Facebook post for my business page
- A Google Business post
- A short email to my newsletter list
```

### For a product business

```
Here's my new product description: [paste]

Create:
- 3 different Instagram captions (educational, testimonial-style, promotional)
- Email subject line options (5 variations)
- Ad copy for Facebook (awareness) and Google (search intent)
- A product launch announcement for my email list
```

### Consistent brand voice

```
Here's an example of content I've written before that sounds like me: [paste example]

Now repurpose this [new content] in the same voice and style.
```

### Content calendar from one piece

```
I wrote this article: [paste]

Create a 2-week social media content plan using only this article as the source.
Mix formats: tips, quotes, questions, behind-the-scenes takes.
Platform: LinkedIn + Instagram.
```

## Example

**You paste a 600-word blog post about "5 reasons local businesses lose customers"**

**Claude produces:**
- 5 LinkedIn posts (one per reason, each with a hook + insight + question)
- Email: subject line "Are you making these 5 mistakes?" + newsletter version
- 3 Instagram captions with different angles
- Google Ads: 3 headline + description combinations
- 60-second script: "If you're a local business owner and you're losing customers, here's what's probably happening..."
