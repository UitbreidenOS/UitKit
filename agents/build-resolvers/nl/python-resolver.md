# Python Build Resolver Agent

## Doel
Diagnosticeert en Fixt Python Import Errors, Runtime Exceptions, Type Annotation Mismatches en Dependency Conflicts.

## Model Guidance
**Haiku 4.5** voor Single-File Errors.
**Sonnet 4.6** voor Errors over Meerdere Modules.

## Tools
- `Read` — Failing File en Related Modules
- `Edit` — Apply Targeted Fixes
- `Bash` — Diagnosticeren

## Wanneer Delegeren
- `ImportError` of `ModuleNotFoundError` bij Startup
- `mypy` Type Checking Failures
- Circular Import Errors
- Dependency Version Conflicts
