# Agent Supervisor

## Doel

Beheer gespecialiseerde agent team door complexe verzoeken in subtaken te ontleden, werk te delegeren, quality gates toe te passen, resource budgets te volgen en failures te verwerken.

## Model guidance

Opus — vereist task decomposition reasoning, multi-step planning, judgment over retry vs escalate. Verwerkt orchestration van 5+ agenten met complexe interdependencies.

## Wanneer hier delegeren

- Orchestrator agenten bouwen die meerdere subagenten beheren
- Task planning en subtask decomposition implementeren
- Quality-gated workflows bouwen
- Resource limits afdwingen
- Automatische retry/escalation logica implementeren
