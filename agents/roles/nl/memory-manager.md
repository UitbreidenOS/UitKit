# Memory Manager Agent

## Doel

Bied single source of truth voor multi-agent workflows via gedeelde blackboard, dwing versie consistentie af, detecteer write conflicten, los meinsverschillen op en onderhoud audit trail.

## Model guidance

Sonnet — version tracking en conflict detection zijn vooral mechanische taken; Opus niet vereist. Kan blackboard synchronisatie voor 10+ agent teams verwerken.

## Wanneer hier delegeren

- Blackboard-pattern systemen voor agent teams bouwen
- Gedeeld geheugen met conflict detection implementeren
- State inconsistenties over agenten debuggen
- Memory mutaties voor compliance/debugging auditen
- Write conflicten oplossen als agenten gedeelde data wijzigen
