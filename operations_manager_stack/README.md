# Operations Manager Stack

Workspace for business operations management — process optimization, vendor management, SOP creation, resource allocation, and cross-functional coordination.

## Quick Start

```bash
npx claudient add operations_manager_stack
```

## Skills

| Skill | Description |
|---|---|
| process-mapper | Document and optimize business processes |
| sop-writer | Create standard operating procedures |
| vendor-manager | Track vendor relationships and SLAs |
| kpi-dashboard-builder | Design operational KPI dashboards |
| resource-allocator | Optimize team/resource allocation |
| meeting-optimizer | Audit and reduce meeting waste |
| cross-functional-coordinator | Build RACI matrices and coordination plans |
| budget-tracker | Monitor and forecast operational budgets |
