# User Research Stack

Workspace for user research workflows — study design, participant recruitment, interview facilitation, synthesis, and research operations.

## Quick Start

```bash
npx claudient add user_research_stack
```

## Skills

| Skill | Description |
|---|---|
| research-planner | Design research studies and plans |
| interview-guide-writer | Create semi-structured interview guides |
| screener-designer | Build participant screening questionnaires |
| synthesis-facilitator | Analyze qualitative data and generate insights |
| persona-builder | Create data-driven user personas |
| journey-mapper | Map user journeys with pain points and opportunities |
| usability-test-designer | Design task-based usability studies |
| research-ops-coordinator | Manage research operations infrastructure |
