================================================================================
ACCESS LEVELS
================================================================================

PRIVATE (default)
  - Only accessible to owner agent
  - For sensitive or personal memories

TEAM
  - Shared with all team agents
  - For collaborative learning
  - Audit trail available

TRUSTED
  - Shared with explicitly named agents
  - For sensitive team knowledge
  - Restricted access list

PUBLIC
  - Available to all agents in system
  - For general knowledge
  - No access restrictions

================================================================================
RETENTION POLICIES
================================================================================

PERMANENT (-1 days)
  - Never expires
  - For critical knowledge
  - Manually managed

LONG_TERM (365 days)
  - 1 year retention
  - For strategies and patterns
  - Annual review recommended

MEDIUM_TERM (90 days)
  - 3 months retention
  - For decisions and preferences
  - Default setting

SHORT_TERM (30 days)
  - 1 month retention
  - For ephemeral data
  - Regular cleanup

EPHEMERAL (7 days)
  - 1 week retention
  - For temporary cache
  - Automatic expiration

================================================================================
API HIGHLIGHTS
================================================================================

Store Memory:
  memory.storeMemory({
    type: MEMORY_TYPES.PATTERN,
    content: { rule: 'if X then Y' },
    description: 'Learned pattern',
    tags: ['learning', 'high-confidence'],
    baseRelevance: 0.95,
    retentionDays: 365
  })

Recall Memory:
  const memory = memory.recallById(id)

Search Memories:
  const results = memory.searchByPattern('pattern', {
    matchType: 'fuzzy',
    type: MEMORY_TYPES.PATTERN,
    tags: ['learning'],
    minRelevance: 0.7,
    limit: 10
  })

Get Statistics:
  const stats = memory.getStats()
  // { memoryCount, totalRecalled, hitRate, typeDistribution, ... }

Export/Import:
  const snapshot = memory.exportSnapshot()
  memory.importSnapshot(snapshot)

Cleanup:
  const expired = memory.cleanupExpired()

PERFORMANCE CHARACTERISTICS
================================================================================

Storage:
  - Per memory: ~2KB average
  - Max total: 100MB (configurable)
  - Encryption overhead: ~50 bytes per entry

Time Complexity:
  - Store: O(k) where k = number of tags
  - Recall: O(1)
  - Search: O(n) worst case
  - Cleanup: O(n)

Throughput:
  - Writes: ~1000/sec (disk I/O bound)
  - Reads: ~10,000/sec (memory bound)
  - Search: ~100-1000/sec (pattern dependent)

Memory Usage:
  - Base: ~1MB
  - Per 1000 memories: ~2MB
  - Growth: Linear with memory count
