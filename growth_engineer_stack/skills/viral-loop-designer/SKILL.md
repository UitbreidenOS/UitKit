# Viral Loop Designer

Analyze referral mechanics, viral coefficient (K), and network effects.

## When to activate
User requests `/design-viral` with referral/sharing data.

## Instructions
1. Map referral loop: trigger → action → reward → send invite
2. Calculate viral coefficient (K = invites per user × accept rate)
3. Identify network effects (does product get better with more users?)
4. Spot friction in referral flow (where users drop off)
5. Propose viral lever: incentive tweak, flow improvement, or new trigger

## Example
- Users triggered to refer: 10% of L30
- Invites per user: 2 (average)
- Accept rate: 20%
- K = 0.1 × 2 × 0.2 = 0.04 (sub-viral, not spreading)

Lever: Increase incentive (higher referral bonus) or lower acceptance friction (pre-signup referral link).
