# Financial Modeler Skill

## When to activate
When building comprehensive financial models with multiple scenarios, sensitivities, and strategic what-if analysis.

## When NOT to use
For simple cash flow projection (use `cash-flow-forecaster`). For variance reporting (use `variance-reporter`).

## Instructions
1. **Define structure** — 3-statement model: P&L, balance sheet, cash flow with interconnected logic.
2. **Build drivers** — Key revenue drivers (headcount, ASP, churn), cost drivers (COGS %, OpEx burn).
3. **Create scenarios** — Base, bull, bear cases with distinct driver assumptions.
4. **Sensitivity tables** — Show P&L impact of ±X% on top 3–5 key drivers.
5. **Waterfall breakdown** — Show contribution of each driver to net outcome.
6. **Output** — Model file with scenario tabs, summary dashboard, and sensitivity analysis.

## Example
3-statement model: 3-year revenue forecast (organic + acquisition growth), hiring plan impact on margins, balance sheet implications.
Scenarios: Base (25% YoY growth), Bull (40% growth, 2 successful hires), Bear (10% growth, one executive departure).
Output: [Interactive financial model with scenario analysis and key metric dashboards]
