# Compétence Confluence Expert

## Quand l'activer
- Restructurer Confluence pour une équipe croissante (organisation par espace, hiérarchie)
- Concevoir une gouvernance documentaire (qui publie ? révision avant publication ? archivage ?)
- Implémenter des modèles d'équipe pour la décision, les ADR, les incidents
- Automatiser Confluence avec les webhooks, API, les créateurs de pages
- Intégrer Confluence avec Slack, Jira, GitHub

## Quand NE PAS l'utiliser
- Recherche de code — utiliser GitHub, not Confluence
- Gestion de tâches — utiliser Jira, pas Confluence
- Dépôt de conversations — utiliser Slack threads, pas Confluence
- Secrets ou credentials — jamais à Confluence
