# Habilidad Research Dossier

## Cuándo activar
- Construcción de un dossier de investigación sobre una empresa antes de una asociación, adquisición o inversión
- Investigación de una persona antes de una junta directiva, decisión de contratación o negociación
- Mapeo de un panorama competitivo antes de entrar a un mercado
- Análisis profundo de una tecnología antes de una decisión build-vs-buy
- Preparación para una reunión de alto riesgo donde necesita contexto completo rápidamente

## Cuándo NO usar
- Verificación de antecedentes rápida (Google es suficiente para nivel superficial)
- Revisión de literatura académica — usar la habilidad lit-review
- Panorama de patentes — usar la habilidad patent-analysis
- Monitoreo de noticias en tiempo real — usar un agregador de noticias dedicado
