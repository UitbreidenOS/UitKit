# Remix Skill

## Wanneer activeren
- Bouwing of uitbreiding Remix applicatie (`app/routes/`, `app/root.tsx`)
- Implementeren data ophalen met loaders of mutations met actions
- Setting up sessie-based auth, cookie management, of form verwerking Remix
- Wiring up resource routes voor API's of webhooks Remix project
- Enig taak references `useLoaderData`, `useActionData`, `Form`, `redirect`, of `@remix-run/*`

## Wanneer NIET gebruiken
- Gebruiker wil pure React SPA zonder server-side rendering — gebruik react skill
- Gebruiker op Next.js — gebruik nextjs skill
- Server-only API (geen UI) — gebruik express of fastify skill

Zie Remix documentatie voor complete implementatie voorbeelden en patterns.
