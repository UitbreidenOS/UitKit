# Agent Supervisor Vaardigheid

## Wanneer activeren

- Orchestrator agent bouwen die meerdere subagenten beheert
- Timeout/retry/escalation logica voor multi-agent workflows
- Resource limits (tokens, latency, cost) over agent team afdwingen
- Task decompositie en werkverdelinge tussen agenten
- Quality gates en validatie tussen agent handoffs
