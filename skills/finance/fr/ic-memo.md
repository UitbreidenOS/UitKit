# Compétence Mémo IC

## Quand activer

- Rédiger un mémorandum Comité d'investissement pour investissement PE, growth equity ou VC
- Structurer une recommandation de transaction pour approbation interne
- Documenter la thèse d'investissement, les résultats due diligence et les rendements financiers
- Préparer réunion IC et anticiper les questions

## Quand NE PAS utiliser

- Prospectus formels ou documents prospectifs destinés aux investisseurs — réglementés
- Recherche d'investissement sur marchés publics — format et exigences divulgation différents
- Mémos de transaction informels (une page) — adapter plutôt à un format plus court

## ⚠️ Important

Les mémos IC présentent des recommandations d'investissement à enjeux élevés. Chaque chiffre financier doit être marqué `[VERIFY]` et sourcé. Les hypothèses doivent être divulguées. Le mémo doit permettre la prise de décision informée, non plaider sans reconnaître les risques.

## Instructions

### Structure du mémo IC à 9 sections

```
I. Résumé exécutif
II. Aperçu de l'opportunité d'investissement
III. Description du métier et proposition de valeur
IV. Opportunité de marché
V. Thèse d'investissement
VI. Résumé de la due diligence
VII. Aperçu financier et analyse de rendements
VIII. Facteurs risque et atténuants
IX. Recommandation
```

### Section I — Résumé exécutif

```
Rédigez le Résumé exécutif d'un mémo IC.

Transaction : [nom entreprise], [type : rachat / growth equity / participation minoritaire]
Montant investissement : $[X]M pour [X]% de participation
Valorisation : [EV/Chiffre d'affaires ou EV/EBITDA multiple], EV = $[X]M
Rendements : [X]x MOIC, [X]% IRR sur [X]-an hold

Éléments clés d'investissement (3-4 bullets) :
- [Résumé chiffre d'affaires/croissance]
- [Avantage concurrentiel]
- [Profil financier]
- [Pourquoi maintenant / pourquoi nous]

Recommandation résumée : [Approuver / Approuver sous conditions / Refuser]
```

### Section V — Thèse d'investissement

```
Rédigez la section Thèse d'investissement.

La thèse doit répondre à :
1. Pourquoi ce métier est-il fondamentalement bon ?
2. Pourquoi sera-t-il plus cher à la sortie ?
3. Pourquoi est-ce le bon moment ?
4. Pourquoi sommes-nous le bon partenaire ?

Piliers thèse : [lister 3-4 thèmes clés]
Leviers création de valeur que nous contrôlons : [améliorations opérationnelles, M&A, expansion géographique, etc.]
Scénarios de sortie : [vente stratégique, IPO, sponsor-to-sponsor]
Pourquoi maintenant : [conditions marché, motivation vendeur, notre avantage concurrentiel]
```

### Section VII — Analyse de rendements

```
Rédigez la section Analyse de rendements.

Hypothèses d'entrée :
- EV d'entrée : $[X]M ([X]x EBITDA LTM)
- Capitaux propres d'entrée : $[X]M (après dette : $[X]M)
- Date d'entrée : [année]

Scénario opérationnel cas de base :
- CAGR chiffre d'affaires : [X]% sur [X]-an hold
- Expansion marge EBITDA : [X]% → [X]%
- Multiple sortie : [X]x EBITDA ([rationale])

Produits sortie :
- EV sortie : $[X]M
- Endettement net sortie : $[X]M
- Capitaux propres sortie : $[X]M

Rendements :
- MOIC brut : [X,Xx]
- IRR brut : [X]%
- IRR net (après frais/carry) : [X]%

Tableau sensibilité (MOIC à différents multiples sortie × périodes hold) :
[X]-an hold  [X]-an hold  [X]-an hold
Sortie [X]x:  [X,X]x       [X,X]x       [X,X]x
Sortie [X]x:  [X,X]x*      [X,X]x       [X,X]x  ← cas de base
Sortie [X]x:  [X,X]x       [X,X]x       [X,X]x

[VERIFY] tous les calculs rendement contre le modèle financier.
```

### Section VIII — Facteurs risque

```
Rédigez la section Facteurs risque et atténuants.

Pour chaque risque : énoncer le risque clairement, quantifier l'impact si possible, décrire l'atténuant.

Risques PE courants à traiter :
1. Risque équipe direction : dépendance clé personne, rétention post-clôture
2. Concentration client : top 3-5 clients en % du chiffre d'affaires
3. Dynamique concurrentiale : défensibilité du moat, menace nouveaux entrants
4. Risque technologie : obsolescence plateforme, disruption IA
5. Risque levier : couverture service de la dette à EBITDA stressé
6. Sensibilité macro : cyclicité, exposition taux d'intérêt
7. Risque exécution : intégration si M&A, entrée nouveau marché

Format : [Risque] | [Sévérité : Élevée/Moyenne/Basse] | [Atténuant]
```

### Génération mémo complet

```
Générez un brouillon de mémo IC complet.

Entreprise : [nom]
Type transaction : [rachat / growth equity / SPAC]
Industrie : [secteur]
Métriques financières clés : [chiffre d'affaires, EBITDA, croissance, marges]
Structure transaction : [montant check, levier, multiple entrée]
Thèse investissement en une phrase : [décrire]
Risques principaux : [lister 2-3]

Générez les 9 sections. Marquez tous chiffres financiers avec [VERIFY].
Signalez toute section où vous besoin plus d'information pour compléter avec précision.
```

## Exemple

**Utilisateur :** Rédigez un mémo IC pour investissement $75M growth equity dans une entreprise B2B HR software à 8x ARR.

**Résultat attendu :** Les 9 sections avec marqueurs `[VERIFY]` appropriés sur tous les chiffres. Résumé exécutif avec cibles MOIC/IRR. Thèse d'investissement axée sur tailwinds HR tech, growth produit-led, et chemin vers profitabilité. Analyse rendements avec cas base 5 ans (3x MOIC, 25% IRR) et tableau sensibilité. Section risque couvrant churn SaaS, menaces concurrentives et dépendance direction.
