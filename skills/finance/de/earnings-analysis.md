# Kompetenz Gewinnanalyse

## Wann zu aktivieren
- Ein Unternehmen, das Sie abdecken, hat gerade Gewinne gemeldet und Sie müssen diese schnell analysieren
- Aktualisierung Ihres Finanzmodells nach einer vierteljährlichen Gewinnveröffentlichung
- Identifizieren Sie die wichtigsten Überraschungen gegenüber dem Konsens (Beat/Miss-Analyse)
- Entwurf einer Morgen-Note zu Gewinnen oder Kundenupdate
- Vorbereitung auf einen Gewinn-Call und Erstellung von Schlüsselfragen

## Wann NICHT zu verwenden
- Gewinn-Streaming in Echtzeit — benötigt einen Live-Daten-Feed
- Formal veröffentlichte Forschungsnotizen — erfordert Compliance-Überprüfung
- Handelsbestimmungen basierend auf dieser Analyse — geregelte Aktivität

## ⚠️ Wichtig

Alle Gewinnanalyse-Ausgaben sind `[ÜBERPRÜFUNG]` erforderlich — validieren Sie jede Zahl gegen die tatsächliche Anmeldung oder das offizielle Transkript, bevor Sie diese verwenden. Die Gewinnanalyse treibt erhebliche Entscheidungen an; Fehler haben echte Konsequenzen.

## Anweisungen

### Schritt 1 — Transkript analysieren

```
Analysen Sie dieses Transkript zum Gewinncall und extrahieren Sie:

[Transkript einfügen oder wichtige Management-Zitate]

Extrahieren Sie:
1. Umsatz tatsächlich vs. Leitfaden vs. Konsensprognose
2. Bruttogewinn tatsächlich vs. Leitfaden vs. Vorjahr
3. EBITDA / Betriebseinkommen tatsächlich vs. Konsens
4. EPS tatsächlich vs. Konsens
5. Aktualisierung der Jahresguidance (erhöht / gesenkt / beibehalten / zurückgezogen)
6. Wichtige Kommentare des Managements (was sich geändert hat, was ist die Erzählung)
7. Analysten Fragen, die Problembereiche offenbaren
8. Alle einmaligen Positionen oder Non-GAAP-Anpassungen, die gekennzeichnet werden sollen

Zusammenfassung: Beat/Miss bei Umsatz, Margen, EPS. Einsatzurteil.
[ÜBERPRÜFEN] Sie alle Zahlen gegenüber den tatsächlichen 10-Q- oder Pressemitteilungen.
```

### Schritt 2 — Beat/Miss-Analyse

```
Analysieren Sie die Gewinnergebnisse gegenüber Erwartungen:

Tatsächliche Ergebnisse:
- Umsatz: $[X]M (Konsens: $[X]M, Leitfaden: $[X]M)
- Bruttogewinnspanne: [X]% (Konsens: [X]%, vorheriger Quartal: [X]%)
- EBITDA: $[X]M (Konsens: $[X]M)
- EPS: $[X] (Konsens: $[X])

Berechnen Sie:
- Umsatz-Beat/Miss: $[X]M ([X]% vs. Konsens)
- Margin-Überraschung: [X]pp gegenüber Konsens
- EPS-Beat/Miss: $[X] ([X]% vs. Konsens)

Qualität des Schlags:
- War es umsatz- oder kostengetrieben?
- Margen erweitert oder kontrahiert?
- Wurde die Anleitung für das nächste Quartal / das Gesamtjahr erhöht?
- Inflationieren einmalige Positionen den Schlag?

Gesamtbewertung: Starker Schlag / Gemischt / Fehlschlag / [nuancierte Einnahme]
[ÜBERPRÜFEN] Sie Konsenszahlen gegen FactSet / Bloomberg, bevor Sie abschließen.
```

### Schritt 3 — Modellaktualisierung

```
Aktualisieren Sie mein Finanzmodell basierend auf diesen Ergebnissen:

Frühere Schätzungen:
- Gesamtjahresumsatz: $[X]M, Gesamtjahres-EBITDA: $[X]M, Gesamtjahres-EPS: $[X]
- Q[X] spezifisch: Umsatz $[X]M, EBITDA $[X]M

Neue Anleitung des Unternehmens:
- Nächstes Quartal: Umsatz $[X]-$[X]M, EPS $[X]-$[X]
- Gesamtjahr: Umsatz $[X]-$[X]M (war $[X]-$[X]M davor)
- Margin-Leitfaden: [X]%-[X]%

Was sollte ich in meinem Modell ändern?
- Q[X+1]-Schätzung: Umsatz $[X]M → $[X]M (Änderung: [X]%)
- Gesamtjahresschätzung: Umsatz $[X]M → $[X]M (Änderung: [X]%)
- Wichtigster Änderungsfaktor: [Volumen / Preis / Marge / Kosten / Einmalig]

[ÜBERPRÜFEN] Modelländerungen gegen tatsächliche Leitungssprache in der Pressemitteilung.
```

### Schritt 4 — Morgen-Note entwerfen

```
Entwerfen Sie eine Morgen-Note zu Gewinnen für Kunden.

Format: [Blitznachricht (1-2 Absätze) / Vollständige Notiz (strukturiert)]
Publikum: [institutionelle Anleger / Portfolio-Manager / Einzelhandel]
Bewertung: [Kaufen / Halten / Verkaufen / Unter Überprüfung]
Kursziel: $[X] (war $[X] — [erhöht/gesenkt/unverändert])

Zu behandelnde Schlüsselpunkte:
1. Überschrift: Beat/Miss, Einsatzurteil
2. Wichtige Positivpunkte (2-3)
3. Wichtige Bedenken (1-2)
4. Leitungsänderung und Auswirkungen auf das Jahr
5. Modelländerungen (EPS-Schätzungsänderung)
6. Bewertung: bei [X]x Geschäftsjahr[X] EV/EBITDA handelt die Aktie mit [Prämie/Rabatt] zu Mitbewerbern
7. Empfehlung

Ton: direkt, keine Absicherung, drücke eine Ansicht aus.
[ÜBERPRÜFEN] Sie alle Zahlen, bevor Sie an Kunden senden.
```

### Schritt 5 — Vorbereitung auf Gewinn-Call

```
Ich decke den Gewinn-Call für [Unternehmen] am [Datum] ab.
Konsensprognosen: Umsatz $[X]M, EPS $[X]
Meine Schätzungen: Umsatz $[X]M, EPS $[X] (vs. Konsens, weil [Grund])

Schlüsselthemen, die ich erforschen möchte:
1. [Wachstumsfaktor oder Risiko, das Sie beobachten]
2. [Margin-Frage]
3. [Geografische oder Produktmischung]

Entwerfen Sie 5-7 Analysten-Fragen für die Q&A-Sitzung.
Machen Sie sie spezifisch, nicht abfallend und so konzipiert, dass sie Informationen extrahieren, die das Management nicht freiwillig mitgeteilt hat.
```

## Beispiel

**Benutzer:** Analysieren Sie diesen Ausschnitt aus einem Transkript zum Gewinn-Call eines SaaS-Unternehmens:
„Der Umsatz belief sich auf 142 Millionen Dollar, ein Anstieg von 28 % gegenüber dem Vorjahr, über unserer Anleitung von 136-138 Millionen Dollar. ARR schloss das Quartal bei 580 Millionen Dollar. Die Netto-Einnahmen-Retention betrug 112 %. Wir leiten Q3 zu 147-149 Millionen Dollar und erhöhen die Gesamtjahresprognose auf 570-580 Millionen Dollar von 550-565 Millionen Dollar."

**Claudes Analyse:**

Umsatz-Beat: 142 Millionen Dollar gegen 137 Millionen Dollar Leitpunkt = 5 Millionen Dollar Beat (3,6%)
NRR 112% stark — Expansion übertrifft Churn
Gesamtjahr erhöht: Mittelpunkt 575 Millionen Dollar gegenüber früherer 557 Millionen Dollar = 18 Millionen Dollar Erhöhung (+3,2%)

**Einsatzurteil:** Clean Beat mit Leitungserhöhung — Ausführung solide, keine Zeichen von Nachfrageschwäche.

**Modelländerungen [ÜBERPRÜFEN]:** Gesamtjahr erhöht auf 575 Millionen Dollar von 557 Millionen Dollar. Unter der Annahme einer ähnlichen Beat-Kadenz ist ein realistischer Bereich 580-590 Millionen Dollar für das Geschäftsjahr.

**Morgennoten-Lead:** „[Unternehmen] lieferte einen sauberen Q2-Beat von 5 Millionen Dollar (3,6%) auf der Top-Line mit 112% NRR, das eine gesunde Expansion signalisiert. Das Management erhöhte die Gesamtjahresprognose um 18 Millionen Dollar am Mittelpunkt — eine glaubwürdige Erhöhung angesichts des Q2-Beat. Kaufen beibehalten, PT auf $X bei höheren Gesamtjahrsschätzungen erhöht."
