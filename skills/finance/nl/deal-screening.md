# Vaardigheid Deal Screening

## Wanneer activeren
- Snel evalueren van een inkomende deal of CIM (vertrouwelijk informatie memorandum)
- Eerste controle uitvoeren voordat je je naar een volledig due diligence-proces engageert
- Een gestandaardiseerde deal-scorecard voor je investeringscomité samenstellen
- Bepalen of je voorbij gaat of naar managementvergaderingen gaat

## Wanneer NIET gebruiken
- Volledige due diligence — screening is een eerste pass, geen definitieve beoordeling
- Openbare markttrade-investeringsbeslissingen — ander framework
- Formele IC-presentaties — gebruik de IC Memo-vaardigheid

## ⚠️ Belangrijk

Deal-schermen zijn richtingsgewijs beoordelingen op basis van beperkte informatie. Kritieke aannamen moeten als `[VERIFICATIE]` worden gemarkeerd. Veel schermen zijn verkeerd — het doel is om duidelijk slechte deals efficiënt uit te sluiten, niet de rest perfect te evalueren.

## Instructies

### Snel scherm (30-minuten versie)

```
Schermbeurs snel. Gebaseerd op de verstrekte CIM/teaser:

Bedrijf: [naam]
Sector: [industrie]
Inkomsten: $[X]M, EBITDA: $[X]M ([X]% marge)
Groeipercentage: [X]% YoY
Vraagprijs: $[X]M EV / [X]x EBITDA

Snelle schermprocedure:
1. GROTE PASVORM: Is de deal in ons doelbereik ($[X]M–$[X]M EV)?
2. SECTOR PASSEND: Past deze sector in ons mandaat en expertise?
3. BEDRIJFSMODEL: Is dit een verdedigbaar bedrijf (geen eenmalig projectbedrijf)?
4. FINANCIËLE GEZONDHEID: Zijn marges redelijk? Is groei organisch of acquisitief?
5. DUIDELIJKE PROBLEMEN: Eventuele rode vlaggen (klantconcentratie, sleutelperson, cyclicaliteit, tech disruptie)?

Uitspraak: [Onmiddellijk doorslaan / Interessant — meer info aanvragen / Managementvergadering aanvragen]
[VERIFICATIE] alle financials tegen CIM voordat escalatie.
```

### Volledige eerste pass-screening

```
Voer volledige eerste pass deal-screening uit.

Bedrijfsoverzicht: [beschrijf of plak CIM-samenvatting]

MARKT
- Marktgrootte: $[X]B TAM — is het groot genoeg voor onze rendementsvereisten?
- Groeipercentage: [X]% — groeit de markt, stabiel of afnemend?
- Cyclicaliteit: is vraag gekoppeld aan macro-omstandigheden?

CONCURRENTIEPOSITIE
- Wat is de gracht? [prijsmacht / schakkelkosten / netwerkeffecten / IP / regelgeving / schaal]
- Marktaandeel: #[X] op markt, [X]% aandeel
- Belangrijkste concurrenten: [noem 2-3] en differentiatie tegenover hen
- Barrières voor toetreders: [hoog / gemiddeld / laag]

MANAGEMENT
- Termijn van CEO en CFO: [X] jaren
- Oprichter-eigendom of reeds PE-ondersteund?
- Managementretentieplan na acquisitie?
- Sleutelpersonrisico?

FINANCIEEL PROFIEL
- Inkomsten: $[X]M ([X]% groei), EBITDA $[X]M ([X]% marge)
- Inkomstenkwaliteit: [terugkerend / transactioneel / projectgebaseerd]
- Klantconcentratie: topklant [X]% van inkomsten, top 5 [X]%
- Churn of retentie: [X]% bruto / [X]% netto
- CapEx-intensiteit: [X]% van inkomsten — licht of zwaar kapitaal?
- Schuld: $[X]M ([X]x EBITDA) — leningswaardig?

WAARDECREATIEKANSEN
- Organische groeihendels: [prijzen / expansie / nieuwe producten]
- M&A-potentieel: [versnipperde markt voor add-ons?]
- Margeverbeteringen: [kostenefficiëntie / prijsstelling / mixshift]
- PE-meerwaarde: [procesverbetering / managementupgrade / governance]

INVOER/UITGANG
- Waarderingsingang: [X]x EBITDA — hoe verhoudt dit zich tot vergelijkbare transacties?
- Uitgangswegen: [strategische verkoop / sponsor-naar-sponsor / IPO]
- Doelstelling IRR: [X]% bij [X]-jarige bedrijf — haalbaar bij deze ingang?

VOORLOPIG OORDEEL
Elke dimensie scoren: 5 = uitstekend, 3 = acceptabel, 1 = zwak

Markt: [1-5]
Concurrentiepositie: [1-5]
Management: [1-5]
Financiën: [1-5]
Waardegroei: [1-5]
Invoer/Uitgang: [1-5]

Gemiddelde score: [X]/5
Aanbeveling: [Doorslaan / Soft doorslaan / Voorzichtig voortgaan / Sterk belang]
Sleutel due diligence vragen voor managementvergadering: [lijst 3-5]

[VERIFICATIE] alle financiële cijfers en marktgegevens voordat presentatie aan IC.
```

### Rode vlag checklist

```
Scan deze deal voor rode vlaggen:

[ ] Klantconcentratie > 25% bij één klant
[ ] Sleutelperson-afhankelijkheid (oprichter is heel verkoopteam)
[ ] Technische disruptie (AI vervangt kernproduct)
[ ] Cyclische inkomsten met 30%+ daling in laatste recessie
[ ] Dalende marges gedurende 3+ opeenvolgende jaren
[ ] Negatieve vrije cashflow ondanks positieve EBITDA
[ ] Geschil, regelgevingsonderzoek of milieuaansprakelijkheid
[ ] Gerelateerde transacties die management voordelen
[ ] Verkoopermotivatie onduidelijk of verdacht
[ ] Vorige PE-sponsor uit onder doel-IRR

Markeer alle aanwezige items en beoordeel ernst.
```

## Voorbeeld

**Teaser samenvatting:** B2B vlootbeheersoftware, $12M ARR, 85% brutowinst, 25% YoY groei, oprichter-eigenaar, vraagt 10x ARR ($120M EV).

**Schermuitgang:**

MARKT: Groot ($8B TAM, vlootbeheer groeit met elektrificatierugwind) — STERK (5/5)
CONCURRENTIE: In competitie met Samsara en Verizon Connect — differentiatie moet worden uitgelegd — ONZEKER (3/5)
MANAGEMENT: Alleen oprichter-eigenaar — sleutelpersonrisico gemarkeerd — GEMIDDELD (3/5)
FINANCIËN: 85% brutowinst is uitstekend; 25% groei aantrekkelijk; $12M ARR op 10x = $120M — agressief in pre-rentabiliteitsstadium — ACCEPTABEL (3/5)
WAARDEGROEI: Land-and-expand potentieel; EV-laadintegratie mogelijkheid — INTERESSANT (4/5)
INVOER/UITGANG: 10x ARR is marktkoers voor SaaS op deze groei; strategische kopers (OEM's, telematica cos) = realistische exit — ACCEPTABEL (3/5)

**Score: 3,5/5 — Voorzichtig voortgaan**

**Sleutelvragen voor managementvergadering:**
1. Wie zijn je top 3 klanten en welk % van ARR vertegenwoordigen zij?
2. Hoe ziet de rol van oprichter er uit na acquisitie?
3. Waarom verkoop je nu versus op $30-40M ARR?
4. Hoe win je tegen Samsara in een competitieve deal?
5. Wat is je netto inkomsten retention en churn rate?
