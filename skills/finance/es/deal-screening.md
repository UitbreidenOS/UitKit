# Habilidad de Selección de Transacciones

## Cuándo activar
- Evaluar rápidamente una transacción entrante o CIM (Memorándum de información confidencial)
- Ejecutar un primer análisis antes de comprometerse con un proceso de due diligence completo
- Construir una tarjeta de puntuación de transacciones estandarizada para tu comité de inversión
- Decidir pasar o proceder a reuniones de gestión

## Cuándo NO usar
- Due diligence completa — el análisis es un primer paso, no una evaluación definitiva
- Decisiones de inversión del mercado público — marco diferente
- Presentaciones formales de comité — usa la habilidad Memo de Comité

## ⚠️ Importante

Las evaluaciones de transacciones son evaluaciones direccionales basadas en información limitada. Las suposiciones críticas deben marcarse como `[VERIFICAR]`. Muchas evaluaciones son incorrectas — el objetivo es eliminar de manera eficiente las transacciones claramente malas, no evaluar perfectamente el resto.

## Instrucciones

### Análisis rápido (versión 30 minutos)

```
Evalúa rápidamente esta transacción. Basado en el CIM/teaser proporcionado:

Empresa: [nombre]
Sector: [industria]
Ingresos: $[X]M, EBITDA: $[X]M ([X]% margen)
Tasa de crecimiento: [X]% YoY
Precio pedido: $[X]M EV / [X]x EBITDA

Marco de análisis rápido:
1. ADECUACIÓN DE TAMAÑO: ¿Está la transacción dentro de nuestro rango objetivo ($[X]M–$[X]M EV)?
2. ADECUACIÓN DEL SECTOR: ¿Este sector se ajusta a nuestro mandato y experiencia?
3. MODELO DE NEGOCIO: ¿Este es un negocio defendible (no una tienda de proyectos de una sola vez)?
4. SALUD FINANCIERA: ¿Los márgenes son razonables? ¿Es el crecimiento orgánico o adquisitivo?
5. PROBLEMAS OBVIOS: ¿Alguna bandera roja (concentración de clientes, persona clave, ciclicalidad, disrupción tecnológica)?

Veredicto: [Pasar inmediatamente / Interesante — solicitar más información / Solicitar reunión de gestión]
[VERIFICAR] todos los datos financieros contra el CIM antes de escalar.
```

### Análisis de primer paso completo

```
Ejecuta un análisis de primer paso de transacción completo.

Descripción de la empresa: [describe o pega resumen del CIM]

MERCADO
- Tamaño del mercado: $[X]B TAM — ¿es suficientemente grande para nuestros requisitos de retorno?
- Tasa de crecimiento: [X]% — ¿está creciendo, estable o en declive?
- Ciclicalidad: ¿está la demanda vinculada a las condiciones macroeconómicas?

POSICIÓN COMPETITIVA
- ¿Cuál es el foso competitivo? [poder de precios / costos de cambio / efectos de red / PI / regulación / escala]
- Cuota de mercado: #[X] en mercado, [X]% de cuota
- Competidores principales: [enumera 2-3] y diferenciación versus ellos
- Barreras para nuevos entrantes: [alta / media / baja]

GESTIÓN
- Antigüedad del CEO y CFO: [X] años
- ¿Propiedad del fundador o ya respaldado por PE?
- ¿Plan de retención de gestión post-adquisición?
- ¿Riesgo de persona clave?

PERFIL FINANCIERO
- Ingresos: $[X]M ([X]% crecimiento), EBITDA $[X]M ([X]% margen)
- Calidad de ingresos: [recurrentes / transaccionales / basados en proyectos]
- Concentración de clientes: cliente principal [X]% de ingresos, top 5 [X]%
- Churn o retención: [X]% bruto / [X]% neto
- Intensidad de CapEx: [X]% de ingresos — ¿capital ligero o pesado?
- Deuda: $[X]M ([X]x EBITDA) — ¿apalancable?

OPORTUNIDADES DE CREACIÓN DE VALOR
- Palancas de crecimiento orgánico: [precios / expansión / nuevos productos]
- Potencial de M&A: [¿mercado fragmentado para add-ons?]
- Mejora de márgenes: [eficiencia de costos / precios / cambio de mezcla]
- Valor añadido de PE: [mejora de procesos / mejora de gestión / gobierno]

ENTRADA/SALIDA
- Valoración de entrada: [X]x EBITDA — ¿cómo se compara esto con transacciones comparables?
- Vías de salida: [venta estratégica / patrocinador a patrocinador / IPO]
- TIR objetivo: [X]% a [X] años — ¿alcanzable a esta entrada?

VEREDICTO PRELIMINAR
Puntúa cada dimensión: 5 = excelente, 3 = aceptable, 1 = débil

Mercado: [1-5]
Posición competitiva: [1-5]
Gestión: [1-5]
Finanzas: [1-5]
Creación de valor: [1-5]
Entrada/Salida: [1-5]

Puntuación promedio: [X]/5
Recomendación: [Pasar / Paso suave / Proceder con cautela / Interés fuerte]
Preguntas clave de due diligence para reunión de gestión: [enumera 3-5]

[VERIFICAR] todos los datos financieros y de mercado antes de presentar al comité de inversión.
```

### Lista de verificación de banderas rojas

```
Escanea esta transacción para banderas rojas:

[ ] Concentración de clientes > 25% en un único cliente
[ ] Dependencia de persona clave (fundador es todo el equipo de ventas)
[ ] Riesgo de disrupción tecnológica (IA reemplaza el producto principal)
[ ] Ingresos cíclicos con caída de 30%+ en última recesión
[ ] Márgenes declinantes durante 3+ años consecutivos
[ ] Flujo de caja libre negativo a pesar de EBITDA positivo
[ ] Litigio, investigación regulatoria o responsabilidad ambiental
[ ] Transacciones relacionadas que benefician a la gestión
[ ] Motivación del vendedor poco clara o sospechosa
[ ] Patrocinador PE anterior salió por debajo de retornos objetivo

Marca todos los elementos presentes y evalúa la severidad.
```

## Ejemplo

**Resumen del teaser:** Software de gestión de flota B2B, $12M ARR, margen bruto 85%, crecimiento 25% YoY, propietario fundador, pidiendo 10x ARR ($120M EV).

**Resultado de análisis:**

MERCADO: Grande ($8B TAM, gestión de flota creciendo con impulso de electrificación) — FUERTE (5/5)
COMPETENCIA: Compitiendo con Samsara y Verizon Connect — diferenciación debe explicarse — INCIERTO (3/5)
GESTIÓN: Propietario fundador único — riesgo de persona clave marcado — MEDIO (3/5)
FINANZAS: Margen bruto 85% es excelente; crecimiento 25% atractivo; $12M ARR en 10x = $120M — agresivo en etapa pre-rentabilidad — ACEPTABLE (3/5)
CREACIÓN DE VALOR: Potencial de expansión; oportunidad de integración de carga EV — INTERESANTE (4/5)
ENTRADA/SALIDA: 10x ARR es tasa de mercado para SaaS con este crecimiento; compradores estratégicos (OEM, compañías telemáticas) = salida realista — ACEPTABLE (3/5)

**Puntuación: 3,5/5 — Proceder con cautela**

**Preguntas clave para reunión de gestión:**
1. ¿Quiénes son tus 3 clientes principales y qué % de ARR representan?
2. ¿Cómo se ve el rol del fundador post-adquisición?
3. ¿Por qué vendes ahora versus en $30-40M ARR?
4. ¿Cómo ganas contra Samsara en una transacción competitiva?
5. ¿Cuál es tu retención neta de ingresos y tasa de churn?
