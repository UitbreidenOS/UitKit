# Habilidad de Despeje de PI

## Cuándo activar
- Verificar si un nombre de marca, nombre de producto o dominio está disponible antes del lanzamiento
- Evaluar la libertad de operación (FTO) para un nuevo producto o característica
- Revisar dependencias de código abierto para cumplimiento de licencia antes de lanzamiento comercial
- Identificar conflictos de marca registrada en nuevos mercados o geografías
- Evaluación de riesgo de PI anterior a la presentación
