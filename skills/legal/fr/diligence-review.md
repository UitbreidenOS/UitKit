# Compétence d'examen d'audit

## Quand l'activer
- Conduite d'un audit juridique pour une transaction M&A ou un investissement
- Examen d'une salle de données et production d'un résumé d'audit
- Signalisation des risques clés dans les contrats, PI ou questions d'emploi de l'entreprise cible
- Création d'une liste de contrôle d'audit pour un type d'accord spécifique
- Résumé des conclusions d'un grand ensemble de documents dans un format tabulaire
