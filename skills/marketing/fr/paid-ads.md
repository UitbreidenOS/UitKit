# Paid Ads Skill

## Quand activer
- Configuration d'une nouvelle campagne Google Ads, Meta Ads ou LinkedIn Ads à partir de zéro
- Diagnostic de ROAS faible ou CPA élevé sur les campagnes existantes
- Structuration de groupes d'annonces et de campagnes pour un meilleur Quality Score
- Rédaction de copie d'annonce et de brèves créatives pour les campagnes payantes
- Conception de tests A/B pour les annonces (créatif, audience, page d'accueil)
- Décision de l'allocation budgétaire entre les canaux

## Quand NE PAS utiliser
- SEO organique — utilisez la compétence seo-audit
- Marketing par e-mail — utilisez la compétence email-sequence
- Stratégie de contenu — utilisez la compétence content-strategy
- Industries sensibles au conformité (santé, finance) — les politiques publicitaires varient; vérifier manuellement

## Instructions

### Audit de structure de campagne

```
Auditez la structure de ma campagne publicitaire et suggérez des améliorations.

Canal: [Google Ads / Meta Ads / LinkedIn Ads / TikTok Ads]
Structure actuelle: [décrire — campagnes, groupes d'annonces, ciblage]
Budget mensuel: $[X]
ROAS/CPA actuel: [X]
Objectif: [sensibilisation / leads / achats / inscriptions]
Valeur moyenne de commande / valeur de lead: $[X]

Audit:
1. Structure de la campagne — les campagnes sont-elles organisées par objectif, pas juste par produit?
2. Granularité du groupe d'annonces — le ciblage et les créations sont-ils étroitement appariés?
3. Types de correspondance de mots-clés (Google) — trop large? trop étroit?
4. Stratification des audiences — excluez-vous les clients existants des campagnes d'acquisition?
5. Stratégie d'enchères — correspond-elle à votre objectif et volume de données?
6. Allocation budgétaire — les campagnes performantes sont-elles limitées en budget?

Produire: liste des problèmes classés par impact sur les revenus + corrections spécifiques.
```

### Configuration de campagne Google Ads

```
Configurer une campagne Google Ads pour [produit/service].

Objectif: [leads / ventes ecommerce / installations d'applications]
Mots-clés cibles: [liste ou décrire l'intention]
Budget: $[X]/jour
Géo: [pays / villes]
Noms de concurrents à considérer: [liste]

Conception:
1. Type de campagne: [Recherche / Performance Max / Shopping / Display]
2. Structure de la campagne:
   - Campagne 1: [Marque] — correspondance exacte sur les propres termes de marque
   - Campagne 2: [Intention non-marque] — cibler les mots-clés par thème
   - Campagne 3: [Concurrent] — termes concurrents (si budget permet)

3. Structure du groupe d'annonces par campagne:
   - Groupes d'annonces serrés: 5-15 mots-clés par groupe, même intention
   - Groupes d'annonces distincts pour différentes étapes d'achat (conscient du problème vs. conscient de la solution)

4. Stratégie d'enchères:
   - < 30 conversions/mois: Maximiser les conversions (phase d'apprentissage)
   - 30-100 conversions/mois: CPA cible
   - > 100 conversions/mois: ROAS cible

5. Copie d'annonce (3 titres × 30 caractères, 2 descriptions × 90 caractères):
   - Titre 1: [correspondance à l'intention de recherche]
   - Titre 2: [différenciateur clé]
   - Titre 3: [CTA ou offre]

6. Extensions: Liens de site, Callouts, Extraits structurés minimum

Écrivez 3 variations d'annonces pour les tests A/B.
```

### Stratégie de ciblage Meta Ads

```
Concevoir la stratégie de ciblage pour Meta Ads (Facebook/Instagram).

Produit: [décrire]
Client cible: [données démographiques, intérêts, comportements]
Budget: $[X]/jour
Objectif: [sensibilisation / trafic / leads / achats]
LTV client moyen: $[X]

Construire les couches de ciblage:
1. Audiences principales:
   - Ciblage des intérêts: [intérêts spécifiques, pas de catégories larges]
   - Ciblage comportemental: [comportements d'achat, utilisation d'appareil]
   - Données démographiques: [âge, sexe, localisation, titre du poste si pertinent]

2. Audiences personnalisées (télécharger/créer):
   - Visiteurs du site Web (pixel — fenêtres 30d, 60d, 180d)
   - Téléchargement de liste de clients (correspondance e-mail)
   - Observateurs de vidéos (regardé 50%+)

3. Audiences lookalike:
   - LAL 1%: basé sur [acheteurs / clients à LTV élevé / liste de courrier électronique]
   - LAL 2-3%: expansion plus large

4. Exclusions:
   - Clients existants (exclure de l'acquisition)
   - Convertisseurs récents (exclure pendant 30 jours après achat)

Entonnoir de campagne:
- TOF: Audience large/intérêts → créatif vidéo ou image → impulse sensibilisation
- MOF: Visiteurs du site Web + engagants → créatif testimonial → impulse considération
- BOF: Abandonnistes de panier + observateurs de produit → offre/urgence → impulse conversion

Répartition budgétaire pour $[X]/jour: [allouer entre entonnoir]
```

### Brief créatif d'annonce

```
Écrivez un brief créatif pour la campagne publicitaire [canal].

Produit: [décrire]
Audience cible: [qui sont-ils, ce qui les intéresse]
Message clé: [la seule chose que l'annonce doit communiquer]
Offre: [rabais, essai, garantie, etc.]
CTA: [ce que vous voulez qu'ils fassent]
Format: [image statique / vidéo / carrousel / story]
Ton: [professionnel / ludique / urgent / aspirationnel]

Sections du brief:
1. Hook (premières 3 secondes / au-dessus du pli):
   - Interruption de schéma: [ce qui arrête le défilement?]
   - Signal de valeur immédiate: [pourquoi devraient-ils continuer à regarder?]

2. Corps (problème → solution → preuve):
   - Énoncé du problème (les faire se sentir compris)
   - Positionnement de la solution (comment vous la résolvez différemment)
   - Preuve sociale (nombre, témoignage ou résultat)

3. CTA (action claire):
   - CTA principal: [texte de bouton + page d'accueil]
   - Élément d'urgence: [le cas échéant]

Écrivez 3 concepts créatifs dans ce format. Évaluez chacun pour le potentiel CTR estimé.
```

### Optimisation ROAS

```
Diagnostiquer un ROAS faible et créer un plan d'optimisation.

Canal: [Google / Meta / LinkedIn]
ROAS actuel: [X] (cible: [X])
Dépense mensuelle: $[X]
Suivi de conversion: [fonctionnement / interrompu / partiel]
Étape de l'entonnoir avec la plus grande chute: [sensibilisation → clic / clic → page d'accueil / page d'accueil → conversion]

Diagnostiquer par ordre:
1. Suivi: les données de conversion sont-elles exactes? (pixel tire, view-through vs. click-through, fenêtre d'attribution)
2. Audience: le ciblage est-il trop large (faible intention) ou trop étroit (portée limitée)?
3. Créatif: CTR par rapport à la référence du secteur? CTR faible = problème créatif
4. Page d'accueil: taux de conversion par rapport à la référence? CVR faible = problème de page
5. Offre: y a-t-il une raison convaincante d'agir maintenant?
6. Enchères: l'algorithme optimise-t-il vers le bon signal?

Produire: liste des hypothèses classées + expériences à exécuter, chacune avec impact attendu et effort.
```

## Exemple

**Utilisateur:** Mes campagnes Google Ads pour un outil SaaS ($99/mois) ont un CPA de $480 — bien trop élevé. Dépense: $8,000/mois. Qu'est-ce qui ne va pas?

**Diagnostic de Claude:**

À $99/mois, vous avez besoin CPA < $200 pour avoir une période de remboursement viable (en supposant une durée d'abonnement moyenne de 6 mois). À un CPA de $480, vous brûlez ~2.4x ce que vous pouvez vous permettre.

**Suspects les plus probables dans l'ordre:**

1. **Problème de suivi de conversion** (vérifier d'abord) — confirmez que votre objectif est "inscription d'essai" pas "affichage de page". Si vous comptez n'importe quel événement, le CPA semble gonflé mais ne l'est pas vraiment.

2. **Budget alloué à la correspondance large** — la correspondance large sur les mots-clés SaaS attire les recherches non pertinentes. Exécutez le rapport des termes de recherche pour les 30 derniers jours. Si >30% des clics sont non pertinents, passez à phrase/exact et ajoutez les mots-clés négatifs.

3. **Aucune spécificité de page d'accueil** — si vous envoyez tous les groupes d'annonces à votre page d'accueil, le taux de conversion sera 1-2% au lieu de 5-8%. Créez des pages d'accueil dédiées par campagne.

4. **Campagne de marque manquante** — les termes de marque se convertissent à 10-20x CPA inférieur au non-marque. Si vous n'enchérissez pas sur le nom de votre marque, les concurrents capturent votre trafic le plus chaud.

**Gains rapides cette semaine:**
- Tirez le rapport des termes de recherche, ajoutez 20+ négatifs
- Créez une page d'accueil d'inscription à la version d'essai dédiée pour vos 2 meilleurs groupes d'annonces
- Ajoutez une campagne de marque avec correspondance exacte — budget $500/mois
