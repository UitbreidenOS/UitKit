# Compétence Gestionnaire des Médias Sociaux

## Quand l'activer
- Planifier un calendrier de contenu mensuel ou hebdomadaire
- Écrire des posts optimisés pour la plate-forme (LinkedIn ≠ Instagram ≠ X ≠ TikTok)
- Construire une stratégie de hashtags pour votre niche
- Répondre aux commentaires et aux DM à l'échelle
- Analyser les performances des médias sociaux et ajuster la stratégie

## Quand ne pas l'utiliser
- Publicité sociale payante — utiliser la compétence paid-ads
- Outreach d'influenceurs — flux de travail différent
- Communications de crise — implique un jugement RP au-delà de la création de contenu

## Instructions

### Calendrier de contenu mensuel

```
Construire un calendrier de contenu mensuel de médias sociaux.

Entreprise : [type et brève description]
Plates-formes : [LinkedIn / Instagram / X / TikTok / Facebook — choisir lesquels]
Fréquence de publication : [X fois/semaine par plate-forme]
Piliers de contenu (3-5 thèmes sur lesquels votre marque publie) : [listez ou laissez-moi suggérer]
Événements/campagnes à venir : [lancements de produits, saisonniers, annonces]
Mois : [spécifiez]

Pour chaque semaine, planifiez :
- Thèmes de contenu par jour
- Mélange de formats (éducatif / promotionnel / personnel / réutilisé / engagement)
- Adaptations spécifiques à la plate-forme (ce qui fonctionne sur LinkedIn vs Instagram)
- Un post "hero" par semaine (effort maximal, boosté si payant)

Résultat : plan semaine par semaine avec idées de posts pour chaque plate-forme.
```

### Copy spécifique à la plate-forme

**LinkedIn :**
```
Écrire un post LinkedIn sur [sujet].
Audience : [décrivez — fondateurs / spécialistes du marketing / développeurs / RH]
Objectif : [pensée leader / lead gen / embauche / engagement]
Ton : [professionnel / personnel / contrarian / storytelling]

Règles de format LinkedIn :
- Hook en ligne 1 (avant la coupure "...plus") — faitesles cliquer
- Paragraphes courts (1-2 phrases max)
- Espace blanc entre les paragraphes
- Terminez par une question ou un CTA clair
- Pas plus de 3 hashtags (LinkedIn récompense la profondeur, pas la découverte)
- Longueur optimale : 150-300 mots pour l'engagement
```

**Instagram :**
```
Écrire une légende Instagram pour [description du contenu/image].
Vibe : [inspirant / éducatif / coulisse / produit]
Voix de marque : [décrivez]

Format Instagram :
- Accrocher avec le hook (premiers 125 caractères visibles avant "plus")
- Emojis utilisés naturellement, pas comme décoration
- Hashtags : 5-10 ciblés, 2-3 larges (placer à la fin ou en premier commentaire)
- CTA : "lien dans la bio" ou question pour générer des commentaires
- Variante de story : adapter la même idée en 3-5 cadres d'histoire
```

**X (Twitter) :**
```
Écrire un tweet / fil sur [sujet].
Style : [insight / hot take / fil listicle / question]

Règles tweet :
- < 280 caractères pour tweets autonomes
- Pour les fils : tweet accroche + 4-8 tweets de soutien + tweet résumé
- Punchyincisif, pas de couverture — X récompense les opinions fortes
- Ne commencez pas par "Je" ou le nom de votre entreprise
- Retweet-citez votre propre contenu pour ajouter du commentaire
```

**TikTok/Reels script :**
```
Écrire un script de [30/60/90] secondes pour [sujet].
Hook (3 premières secondes) : [ce qui arrête le scroll]
Format : [talking head / voiceover + texte / tutoriel]

Structure TikTok :
- Seconde 0-3 : Hook (dire ou montrer la chose la plus intéressante en premier)
- Seconde 4-15 : Configuration / contexte
- Seconde 16-45 : Contenu central / valeur
- Seconde 46-60 : CTA ou punchline
- Superpositions de texte : 1-2 points clés à l'écran à tout moment
```

### Stratégie de hashtag

```
Construire une stratégie de hashtag pour [plate-forme] et [niche].

Plate-forme : [Instagram / LinkedIn / X / TikTok]
Niche : [décrivez entreprise et sujets de contenu]
Taille actuelle du compte : [followers]

Pour chaque plate-forme, créez 3 niveaux :
1. Hashtags de niche (petit — < 100K posts) : chance de classement la plus élevée
2. Hashtags de taille moyenne (100K-1M posts) : portée + découvrabilité
3. Hashtags larges (1M+ posts) : faible chance de classement mais signal de marque

Recommander : [X] hashtags par post, [distribution entre les niveaux]
Organiser en 3-5 ensembles de hashtags à faire pivoter (éviter le shadowban par répétition).
```

### Scripts d'engagement

```
Écrire les modèles de réponse d'engagement pour [plate-forme].

Situations à couvrir :
1. Commentaire générique positif ("J'adore ça !")
2. Commentaire réfléchi qui mérite une vraie réponse
3. Question qui peut être répondue publiquement
4. Critique ou retour négatif (non-abusif)
5. Spam ou commentaire de faible qualité (quand ignorer/masquer)

Ton : [voix de marque]
Objectif : [construire la communauté / entraîner vers les DM / augmenter la portée / montrer l'expertise]
```

### Interprétation d'analyse

```
Interpréter mes performances des médias sociaux pour [mois].

Plate-forme : [laquelle]
Métriques clés :
- Portée : [X] (vs le mois dernier : [X])
- Impressions : [X]
- Taux d'engagement : [X]%
- Followers gagnés : [X]
- Top post : [décrivez-le]
- Pire performer : [décrivez-le]

Dites-moi :
1. Quel type de contenu a le mieux fonctionné et pourquoi ?
2. Qu'est-ce que je devrais faire plus / moins de ?
3. Quels jours/heures ont conduit au plus d'engagement ?
4. Quel est mon expérience 3-post pour le mois prochain basée sur ces données ?
```

## Exemple

**Utilisateur :** Je suis fondateur d'une startup B2B SaaS. J'ai besoin d'une semaine de contenu LinkedIn. Les sujets dont je me préoccupe : croissance dirigée par les produits, métriques SaaS et construction en public.

**Claude génère 5 posts LinkedIn :**

**Lundi (Insight) :**
> La plus grande erreur que les fondateurs SaaS commettent avec la conversion essai-payant :
> La mesurer hebdomadairement.
>
> La conversion d'essai est une métrique de cohorte. Vous avez besoin de 30-60 jours avant que le nombre n'ait du sens.
>
> Voici le framework que j'utilise à la place : [3 bullets]
>
> Quel est votre taux actuel essai-payant ?

[continue pour les 5 jours avec variété de format/jour]
