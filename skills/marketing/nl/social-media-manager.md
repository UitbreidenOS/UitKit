# Social Media Manager-vaardigheid

## Wanneer activeren
- Planning van een maandelijkse of wekelijkse social media inhoudskalender
- Schrijven van platform-geoptimaliseerde berichten (LinkedIn ≠ Instagram ≠ X ≠ TikTok)
- Het bouwen van een hashtag-strategie voor uw niche
- Reageren op opmerkingen en DM's op schaal
- Analyse van social media prestaties en aanpassing van strategie

## Wanneer NIET gebruiken
- Betaalde social media-advertenties — gebruik de paid-ads vaardigheid
- Influencer outreach — ander workflow
- Crisiscommunicatie — gaat verder dan contentcreatie

## Instructies

### Maandelijkse inhoudskalender

```
Bouw een maandelijkse social media inhoudskalender.

Bedrijf: [type en korte beschrijving]
Platforms: [LinkedIn / Instagram / X / TikTok / Facebook — kies welke]
Posting frequentie: [X keer/week per platform]
Inhoudspilaars (3-5 thema's waar uw merk over post): [lijst of laat me suggereren]
Aankomende evenementen/campagnes: [productlanceringen, seizoensgebonden, aankondigingen]
Maand: [opgeven]

Plan voor elke week:
- Inhoudsfocus per dag
- Mix van formaten (educatief / promotioneel / persoonlijk / hergebruikt / engagement)
- Platform-specifieke aanpassingen (wat werkt op LinkedIn vs. Instagram)
- Eén "hero" bericht per week (meeste inspanning, geboost als betaald)

Uitgave: week-voor-week plan met post-ideeën voor elk platform.
```

### Platform-specifieke copy

**LinkedIn:**
```
Schrijf een LinkedIn bericht over [onderwerp].
Publiek: [beschrijf — oprichters / marketeers / ontwikkelaars / HR]
Doel: [gedachtenleiderschap / lead gen / inhuurtaken / engagement]
Toon: [professioneel / persoonlijk / contrair / storytelling]

LinkedIn formaat regels:
- Hook in lijn 1 (voor "...meer" cut-off) — maak hen klikken
- Korte alinea's (max 1-2 zinnen)
- Witruimte tussen alinea's
- Eindigen met vraag of duidelijke CTA
- Niet meer dan 3 hashtags (LinkedIn beloont diepte, niet ontdekking)
- Optimale lengte: 150-300 woorden voor engagement
```

**Instagram:**
```
Schrijf een Instagram bijschrift voor [inhoud/afbeeldingsbeschrijving].
Vibe: [inspirerend / educatief / achter-de-schermen / product]
Merksteem: [beschrijf]

Instagram format:
- Hook vooraan (eerste 125 tekens zichtbaar voor "meer")
- Emoji's natuurlijk gebruikt, niet als decoratie
- Hashtags: 5-10 gericht, 2-3 breed (aan einde of eerste reactie)
- CTA: "link in bio" of vraag om reacties uit te lokken
- Story variant: pas hetzelfde idee aan voor 3-5 Story frames
```

**X (Twitter):**
```
Schrijf een tweet / thread over [onderwerp].
Stijl: [inzicht / hete take / listicle thread / vraag]

Tweet regels:
- < 280 tekens voor standalone tweets
- Voor threads: hook tweet + 4-8 ondersteunende tweets + samenvatting tweet
- Puntig, geen aarzeling — X beloont sterke meningen
- Begin niet met "Ik" of uw bedrijfsnaam
- Quote-tweet je eigen inhoud om commentaar toe te voegen
```

**TikTok/Reels script:**
```
Schrijf een [30/60/90]-seconde script voor [onderwerp].
Hook (eerste 3 seconden): [wat stopt het scrollen]
Format: [talkend head / voiceover + tekst / tutorial]

TikTok structuur:
- Seconde 0-3: Hook (zeg of toon het meest interessante eerst)
- Seconde 4-15: Setup / context
- Seconde 16-45: Kerninhoud / waarde
- Seconde 46-60: CTA of grap
- Tekstoverlays: 1-2 sleutelpunten constant op scherm
```

### Hashtag strategie

```
Bouw een hashtag strategie voor [platform] en [niche].

Platform: [Instagram / LinkedIn / X / TikTok]
Niche: [beschrijf bedrijf en inhoudsonderwerpen]
Huidige accountgrootte: [volgers]

Voor elk platform, maak 3 niveaus:
1. Niche hashtags (klein — < 100K posts): beste kans op ranking
2. Mid-size hashtags (100K-1M posts): bereik + ontdekbaarheid
3. Brede hashtags (1M+ posts): lage kans op ranking maar merksignaal

Aanbeveling: [X] hashtags per post, [distributie over niveaus]
Organiseer in 3-5 hashtag sets om te roteren (vermijd shadowban van herhaling).
```

### Engagement scripts

```
Schrijf engagement response templates voor [platform].

Situaties om mee om te gaan:
1. Generieke positieve opmerking ("Love this!")
2. Doordachte opmerking die een echte reactie verdient
3. Vraag die openbaar kan worden beantwoord
4. Kritiek of negatieve feedback (niet beledigend)
5. Spam of lage kwaliteit opmerking (wanneer negeren/verbergen)

Toon: [merksteem]
Doel: [community bouwen / naar DM's sturen / bereik verhogen / expertise tonen]
```

### Analytics interpretatie

```
Interpreteer mijn social media prestaties voor [maand].

Platform: [welke]
Sleutelmetrieken:
- Bereik: [X] (vs. vorige maand: [X])
- Impressies: [X]
- Engagement rate: [X]%
- Volgers gewonnen: [X]
- Toppost: [beschrijf het]
- Slechtste performer: [beschrijf het]

Vertel me:
1. Welk inhoudstype werkte het best en waarom?
2. Wat moet ik meer / minder doen?
3. Welke dagen/tijden dreven het meeste engagement?
4. Wat zijn mijn 3-post experimenten voor volgende maand op basis van deze data?
```

## Voorbeeld

**Gebruiker:** Ik ben een B2B SaaS oprichter. Heb een week LinkedIn inhoud nodig. Onderwerpen waar ik om geef: product-led growth, SaaS metrics, en open bouwen.

**Claude genereert 5 LinkedIn posts:**

**Maandag (Inzicht):**
> De grootste fout SaaS oprichters maken met trial-to-paid conversie:
> Het wekelijks meten.
>
> Trial conversie is een cohort metriek. Je hebt 30-60 dagen nodig voordat het getal iets betekent.
>
> Hier is het framework dat ik in plaats daarvan gebruik: [3 punten]
>
> Wat is uw huidige trial-to-paid rate?

**Woensdag (Verhaal/Bouwen in het openbaar):**
> We hebben deze week $50K MRR bereikt. Hier ziet het dashboard er echt uit:
> [datapunt] [datapunt] [datapunt]
> Het getal dat ons het meest verraste: [inzicht]
> En het getal dat me nog steeds wakker houdt: [kwetsbaarheid]

[gaat door voor alle 5 dagen met dag-/formaatverbetering]
