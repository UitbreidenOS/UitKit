# Copywriting Skill

## Wann aktivieren
- Verfassen oder Neuschreiben einer Landing Page für höhere Konversion
- Verfassung von E-Mail-Betreffzeilen, Vorschautext und Exemplaren
- Schreiben von Anzeigenüberschriften und Beschreibungen
- Erstellen eines überzeugenden Wertversprechen oder Slogans
- Verfassung von Produktbeschreibungen, die verkaufen, nicht nur beschreiben
- Überprüfung vorhandener Exemplare auf Schwächen

## Wann NICHT verwenden
- Langforminhalte wie Blogartikel — verwende die content-strategy Skill
- Technische Dokumentation — schreibe für Klarheit, nicht Konversion
- Tonalitätsdefinition der Marke — das ist brand-guidelines, nicht copywriting Ausführung

## Anweisungen

### Wertversprechen

```
Verfassen Sie ein Wertversprechen für [Produkt/Service].

Produkt: [was es tut]
Zielkunde: [wer, ihre Rolle, ihre Situation]
Schlüsselschmerz, den sie gerade haben: [spezifische Frustration oder Kosten]
Das Ergebnis, das sie wollen: [wie Erfolg für sie aussieht]
Warum Sie sich von Alternativen unterscheiden: [ein wahrer Unterschied]
Beweis dieses Unterschieds: [Stat, Fallstudie oder Feature]

Wertversprechen-Formate zum Produzieren:

1. Slogan (7 Wörter oder weniger):
   [Verb] + [Ergebnis] + [für wen]
   Beispiel: "Schneller verschiffen. Nichts brechen. Besser schlafen."

2. Hero-Überschrift (Landing Page):
   Ergebnis zuerst, spezifisch, kein Jargon.
   Formel: [Tun Sie X] ohne [Frustration/Kompromiss]
   oder: [Zeit/Aufwand] zu [Ergebnis]

3. Unter-Überschrift (max 25 Wörter):
   Erweitern Sie, für wen es ist und wie es funktioniert.

4. Fahrstuhl-Pitch (2 Sätze):
   Für [Persona], die mit [Problem] kämpfen,
   [Produkt] ist [Kategorie], die [Schlüsselvorteil].
   Im Gegensatz zu [Alternativen], [Unterscheidungsmerkmal].

Schreiben Sie 3 Versionen von jedem. Markieren Sie, welche am stärksten ist und warum.
```

### Landing Page Exemplar

```
Verfassen Sie Landing Page Exemplare für [Produktziel/Seitenziel].

Seitenziel: [Trial-Anmeldung / Demo-Anfrage / Kauf / Lead Magnet Download]
Zielbesucher: [wer kommt hierher, von woher]
Ihr Bewusstseinsniveau: [Problemorientiert / Lösungsorientiert / Produktorientiert]
Aktuelle Konversionsrate (falls bekannt): [X%]
Schlüsselobjektionen, die sie haben: [Preis / Vertrauen / Komplexität / Timing]

Seitenstruktur:

1. Hero-Abschnitt:
   - Überschrift: [Ergebnis zuerst, spezifisch]
   - Unter-Überschrift: [für wen es ist + wie es funktioniert]
   - CTA-Knopf: [Aktionsverb + was sie bekommen — nicht einfach "Absenden"]
   - Hero-Bild/Video-Konzept: [was würde Vertrauen aufbauen oder das Produkt zeigen?]

2. Problem → Reizung:
   - Benennen Sie den Schmerz klar (verwenden Sie ihre Worte, nicht Marketing-Sprache)
   - Zeigen Sie, dass Sie die Kosten des Problems verstehen (Zeit, Geld, Frustration)

3. Lösung (wie Sie es beheben):
   - 3 Schlüsselvorteile (nicht Features) mit spezifischen Ergebnissen
   - Format: [Feature] → So können Sie [Vorteil] → Was bedeutet [Ergebnis]

4. Beweis:
   - Sozialbeweis: [Testimoniale, Logos, Stats]
   - Spezifität schlägt Generisches: "4 Stunden pro Woche gespart" schlägt "spart Zeit"

5. Einwandbehandlung (FAQ oder inline):
   - Top 3 Objektionen → vorbeugende Antworten

6. CTA-Abschnitt:
   - Wiederholtes Angebot
   - Reduziertes Risiko (kostenloser Trial, Geldback, keine Kreditkarte)
   - Finaler CTA-Knopf

Vollständiges Exemplar für jeden Abschnitt produzieren. Annahmen markieren.
```

### E-Mail-Betreffzeilen

```
Verfassen Sie E-Mail-Betreffzeilen für [E-Mail-Kampagne].

E-Mail-Ziel: [Öffnung und Klick / Öffnung und Antwort / Öffnung und Kauf]
Publikum: [Listen-Segment — neue Leads / bestehende Kunden / kalte Interessenten]
E-Mail-Inhaltsübersicht: [was die E-Mail enthält]
Name vom Absender: [Personenname oder Marke]

Verfassen Sie 10 Betreffzeilen mit verschiedenen Taktiken:

1. Neugier-Lücke: insinuieren Sie etwas ohne es zu offenbaren
2. Direkter Vorteil: sagen Sie genau, was sie durch Öffnen bekommen
3. Frage: fragen Sie etwas, das sie bereits denken
4. Sozialbeweis: Name-Drop oder Stat
5. Dringlichkeit/Knappheit: Frist oder begrenzte Verfügbarkeit
6. Personalisierung: [Vorname] oder situationsspezifisch
7. Kurz und prägnant: 3-5 Wörter maximal
8. Umstritten/Konträr: fordern Sie einen gemeinsamen Glauben heraus
9. Geschichte-Hook: "Ich habe einen Fehler gemacht..."
10. Listenformat: "3 Dinge, die X nicht möchte, dass Sie wissen"

Für jeden: Betreffzeile + Vorschautext (50 Zeichen), der sie ergänzt.

Bewerten Sie jeden 1-10 für wahrscheinliche Öffnungsrate für diese bestimmte Publikum.
```

### PAS-Framework Exemplar

```
Verfassen Sie Exemplare mit dem Problem-Reizung-Lösungs-Framework.

Problem: [das spezifische Problem, das der Leser hat]
Publikum: [wer hat dieses Problem, in welchem Kontext]
Lösung: [Ihr Produkt/Service]

PAS-Struktur:

P — Problem (benennen Sie es klar, in ihren Worten):
[1-2 Sätze, die sie nicken lassen und sagen "ja, genau das ist es"]

A — Reizung (machen Sie den Schmerz lebhaft):
[3-5 Sätze zeigen die Nebenwirkungen:
- Was es sie kostet (Zeit, Geld, Beziehung, Chance)
- Was passiert, wenn sich nichts ändert
- Die emotionale Frustration, nicht nur das funktionale Problem]

S — Lösung (stellen Sie Ihre Antwort vor):
[2-3 Sätze zeigen, wie Ihre Lösung das Problem beseitigt.
Seien Sie spezifisch: nicht "wir helfen Ihnen Zeit sparen" sondern "3 Stunden Freitag."]

CTA: [nächster klarer Schritt mit niedriger Reibung]

Verfassen Sie dies für: [E-Mail / Landing Page / Anzeige / LinkedIn Post]
2 Versionen produzieren: emotionaler Winkel und rationaler/ROI-Winkel.
```

### Anzeigen-Exemplar

```
Verfassen Sie Anzeigen-Exemplare für [Kanal].

Kanal: [Google Suche / Meta / LinkedIn / Twitter / YouTube Pre-Roll]
Produkt: [beschreiben]
Ziel: [wer diese Anzeige sieht]
Angebot: [was sie bekommen, wenn sie klicken — Trial / Demo / Download / Kauf]
Schlüsseldifferenzier: [ein wahres Ding, das Sie unterschiedlich macht]
Zeichenbegrenzer: [Google: 30/90 | Meta: 125/27/27 | LinkedIn: 150 Überschrift]

Für Google Ads (Responsive Search Ad):
- 15 Überschriften (je 30 Zeichen): Hook, Vorteil, CTA-Variationen
- 4 Beschreibungen (je 90 Zeichen): Erweitern Sie die Top 2 Vorteile + Sozialbeweis

Für Meta Ads (3 Variationen):
- Primärtext (125 Zeichen): Hook-Zeile
- Überschrift (27 Zeichen): Produkt/Angebot
- Beschreibung (27 Zeichen): Angebot verstärken

Markieren Sie: [PIN] für Überschriften, die in spezifischen Positionen erscheinen müssen.
Verfassen Sie für direkte Reaktion, nicht Marke — jede Anzeige muss einen CTA haben.
```

## Beispiel

**Benutzer:** Verfassen Sie unsere Landing Page Überschrift für ein SaaS für juristische Vertragsüberprüfung neu. Aktuelle Überschrift: "KI-gesteuerte Vertragsüberprüfung für moderne Teams." CVR ist 1,8%.

**Claudes Neuschreiben:**

Die aktuelle Überschrift ist Feature-geführt ("KI-gesteuert") und generisch ("moderne Teams"). Bei 1,8% CVR verbindet sich der Hero nicht mit dem, was der Besucher wirklich fürchtet.

**Die echte Angst:** Misse eine Klausel, die uns sechs Ziffern kostet oder Wochen Rechtsstreit.

**3 Hero-Überschrifts-Optionen:**

Option 1 (Ergebnis + Risikoreduktion):
> "Überprüfen Sie Verträge in 10 Minuten. Fangen Sie die Klauseln, die Ihr Anwalt 400$/Stunde verrechnet."

Option 2 (Direktes Problem → Lösung):
> "Hören Sie auf, 40-seitige Verträge Zeile für Zeile zu lesen.
> Finden Sie jeden riskanten Klausel in unter 15 Minuten."

Option 3 (Konträr + Spezifisch):
> "Ihr letzter Vertrag hatte wahrscheinlich 3 rote Fahnen darin.
> Du hast sie nicht gefunden. Wir hätten es getan."

**Unter-Überschrift (für Option 1):**
> Für Startups und wachsende Teams, die rechtliche Klarheit ohne Anwaltsrechnungen benötigen. Laden Sie einen Vertrag hoch — NDA, SaaS-Vereinbarung, Beschäftigung — und erhalten Sie eine Risikozusammenfassung in einfachen Worten.

**CTA-Knopf:**
Anstelle von "Starten" → **"Überprüfen Sie meinen ersten Vertrag kostenlos"**

Geschätzte CVR-Verbesserung: 2,5-3,5% (basierend auf Spezifität + Ergebnis-Erst-Struktur).
