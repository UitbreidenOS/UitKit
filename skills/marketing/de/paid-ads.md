# Paid Ads Skill

## Wann aktivieren
- Einrichtung einer neuen Google Ads, Meta Ads oder LinkedIn Ads Kampagne von Grund auf
- Diagnose von schlechtem ROAS oder hohem CPA bei bestehenden Kampagnen
- Strukturierung von Anzeigengruppen und Kampagnen für bessere Quality Score
- Verfassung von Anzeigen und kreativen Briefs für bezahlte Kampagnen
- Gestaltung von A/B-Tests für Anzeigen (kreativ, Publikum, Landingpage)
- Entscheidung über Budgetverteilung über Kanäle

## Wann NICHT verwenden
- Organische SEO — verwende den seo-audit Skill
- E-Mail-Marketing — verwende den email-sequence Skill
- Content-Strategie — verwende den content-strategy Skill
- Compliance-sensitive Branchen (Gesundheit, Finanzen) — Anzeigenrichtlinien variieren; manuelle Überprüfung

## Anweisungen

### Audit der Kampagnenstruktur

```
Prüfen Sie meine Kampagnenstruktur für bezahlte Anzeigen und schlagen Sie Verbesserungen vor.

Kanal: [Google Ads / Meta Ads / LinkedIn Ads / TikTok Ads]
Aktuelle Struktur: [beschreiben — Kampagnen, Anzeigengruppen, Targeting]
Monatliches Budget: $[X]
Aktueller ROAS / CPA: [X]
Ziel: [Bewusstsein / Leads / Käufe / Registrierungen]
Durchschnittlicher Bestellwert / Lead-Wert: $[X]

Audit:
1. Kampagnenstruktur — sind Kampagnen nach Ziel organisiert, nicht nur nach Produkt?
2. Granularität der Anzeigengruppe — sind Targeting und Kreativleistung eng aufeinander abgestimmt?
3. Keyword-Übereinstimmungstypen (Google) — zu breit? zu eng?
4. Publikumsschichtung — schließen Sie bestehende Kunden von Akquisitionskampagnen aus?
5. Gebotsstrategie — entspricht sie Ihrem Ziel und Datenvolumen?
6. Budgetverteilung — sind leistungsstarke Kampagnen budgetbegrenzt?

Produktion: Liste der Probleme, die nach Umsatzauswirkung eingestuft sind + spezifische Lösungen.
```

### Einrichtung der Google Ads Kampagne

```
Richten Sie eine Google Ads Kampagne für [Produkt/Service] ein.

Ziel: [Leads / Ecommerce-Verkauf / App-Installationen]
Ziel-Keywords: [Liste oder Beschreiben der Absicht]
Budget: $[X]/Tag
Geo: [Länder / Städte]
Zu berücksichtigende Konkurrentennamen: [Liste]

Gestaltung:
1. Kampagnentyp: [Suche / Performance Max / Shopping / Display]
2. Kampagnenstruktur:
   - Kampagne 1: [Marke] — exakte Übereinstimmung mit eigenen Markenbegriffen
   - Kampagne 2: [Nicht-Markierungsabsicht] — Ziel-Keywords nach Thema
   - Kampagne 3: [Konkurrenz] — Konkurrenzausdrücke (falls Budget erlaubt)

3. Anzeigengruppe Struktur pro Kampagne:
   - Enge Anzeigengruppen: 5-15 Keywords pro Gruppe, gleiche Absicht
   - Separate Anzeigengruppen für verschiedene Kaufphasen (Problemorientiert vs. Lösungsorientiert)

4. Gebotsstrategie:
   - < 30 Konversionen/Monat: Konversionen maximieren (Lernphase)
   - 30-100 Konversionen/Monat: Ziel-CPA
   - > 100 Konversionen/Monat: Ziel-ROAS

5. Anzeigentext (3 Überschriften × 30 Zeichen, 2 Beschreibungen × 90 Zeichen):
   - Überschrift 1: [Suchabsicht abgleichen]
   - Überschrift 2: [Schlüsseldifferenzier]
   - Überschrift 3: [CTA oder Angebot]

6. Erweiterungen: Sitelinks, Callouts, strukturierte Snippets minimum

Schreiben Sie 3 Anzeigenvariationen zum A/B-Testen.
```

### Meta Ads Targeting-Strategie

```
Entwerfen Sie die Targeting-Strategie für Meta Ads (Facebook/Instagram).

Produkt: [beschreiben]
Zielkunde: [Demografie, Interessen, Verhaltensweisen]
Budget: $[X]/Tag
Ziel: [Bewusstsein / Traffic / Leads / Käufe]
Durchschnittlicher Kunden-LTV: $[X]

Erstellen Sie Targeting-Ebenen:
1. Kernzielgruppen:
   - Interessens-Targeting: [spezifische Interessen, keine breiten Kategorien]
   - Verhaltens-Targeting: [Kaufverhalten, Gerätenutzung]
   - Demografie: [Alter, Geschlecht, Ort, Job-Titel, falls relevant]

2. Benutzerdefinierte Zielgruppen (Upload/Build):
   - Website-Besucher (Pixel — 30d, 60d, 180d Fenster)
   - Kundenliste hochladen (E-Mail-Abgleich)
   - Video-Zuschauer (50%+ angesehen)

3. Lookalike-Zielgruppen:
   - LAL 1%: basierend auf [Käufer / Kunden mit hohem LTV / E-Mail-Liste]
   - LAL 2-3%: breitere Erweiterung

4. Ausschlüsse:
   - Bestehende Kunden (aus Akquisition ausschließen)
   - Letzte Konverter (30 Tage nach Kauf ausschließen)

Kampagnen-Trichter:
- TOF: Breit/Interessen → Video- oder Bild-Kreativ → Bewusstsein antreiben
- MOF: Website-Besucher + Engagierte → Testimonial-Kreativ → Überlegung antreiben
- BOF: Warenkorbabbrecher + Produktbetrachter → Angebot/Dringlichkeit → Konversion antreiben

Budgetaufteilung für $[X]/Tag: [über Trichter verteilen]
```

```
Schreiben Sie einen kreativen Brief für die Werbekampagne [Kanal].

Produkt: [beschreiben]
Zielgruppe: [wer sie sind, was ihnen wichtig ist]
Schlüsselbotschaft: [das einzige, das die Anzeige kommunizieren muss]
Angebot: [Rabatt, Testversion, Garantie, etc.]

Format: [statisches Bild / Video / Carousel / Story]
Ton: [professionell / spielerisch / dringend / aspirierend]

Brief-Abschnitte:
1. Hook (erste 3 Sekunden / über der Falz):
   - Muster unterbrechen: [was stoppt das Scrollen?]

2. Körper (Problem → Lösung → Beweis):
   - Problemstellung (sie verstanden fühlen)
   - Lösungspositionierung (wie Sie es anders lösen)
   - Sozialbeweis (Nummer, Testimonial oder Ergebnis)

3. CTA (klare Aktion):
   - Primärer CTA: [Schaltflächentext + Landingpage]
   - Dringlichkeitselement: [falls zutreffend]

Schreiben Sie 3 kreative Konzepte in diesem Format. Bewerten Sie jeweils das geschätzte CTR-Potenzial.
```

### ROAS-Optimierung

```
Diagnose schlechter ROAS und erstellen Sie einen Optimierungsplan.

Kanal: [Google / Meta / LinkedIn]
Aktueller ROAS: [X] (Ziel: [X])
Monatliche Ausgaben: $[X]
Konversionsverfolgung: [funktioniert / unterbrochen / teilweise]
Trichterstadium mit größtem Rückgang: [Bewusstsein → Klick / Klick → Landingpage / Landingpage → Konversion]

Diagnose in Reihenfolge:
1. Verfolgung: sind Konversionsdaten genau? (Pixel feuert, View-Through vs. Click-Through, Attributionsfenster)
2. Publikum: ist das Targeting zu breit (schwache Absicht) oder zu eng (begrenzte Reichweite)?
3. Kreativ: CTR vs. Branchenbenchmark? Niedriger CTR = Kreativproblem
4. Landingpage: Konversionsrate vs. Benchmark? Niedriger CVR = Seitenproblem
5. Angebot: gibt es einen zwingenden Grund, jetzt zu handeln?
6. Gebote: optimiert der Algorithmus zum richtigen Signal?

Produktion: Rangfolge Hypothesen + Experimente zum Ausführen, jeweils mit erwartetem Einfluss und Aufwand.
```

## Beispiel

**Benutzer:** Meine Google Ads Kampagnen für ein SaaS-Tool ($99/Monat) haben einen CPA von $480 — viel zu hoch. Ausgaben: $8,000/Monat. Was ist falsch?

**Claudes Diagnose:**

Bei $99/Monat benötigen Sie CPA < $200, um eine rentable Amortisationsdauer zu haben (angenommen durchschnittliche Abonnementdauer von 6 Monaten). Bei $480 CPA verbrennen Sie ~2,4x was Sie sich leisten können.

**Wahrscheinlichste Schuldige in Reihenfolge:**

1. **Konversionsverfolgungsproblem** (zuerst überprüfen) — bestätigen Sie, dass Ihr Ziel "Testanmeldung" ist, nicht "Seitenbild". Wenn Sie Ereignisse zählen, sieht der CPA aufgeblasen aus, ist aber nicht real.

2. **Budget für breite Übereinstimmung zugewiesen** — breite Übereinstimmung bei SaaS-Keywords zieht irrelevante Suchen an. Führen Sie den Suchbegriff-Bericht für die letzten 30 Tage aus. Wenn >30% der Klicks irrelevant sind, wechseln Sie zu Phrase/Exact und fügen negative Keywords hinzu.

3. **Keine Spezifität der Landingpage** — wenn Sie alle Anzeigengruppen auf Ihre Startseite leiten, liegt die Konversionsrate bei 1-2% statt 5-8%. Erstellen Sie dedizierte Landingpages pro Kampagne.

4. **Kampagne zur fehlenden Marke** — Markenbegriffe werden 10-20x niedriger CPA konvertiert als Nicht-Marke. Wenn Sie nicht auf Ihren Markennamen bieten, erfassen Konkurrenten Ihren wärmsten Traffic.

**Schnelle Gewinne diese Woche:**
- Suchbegriff-Bericht ziehen, 20+ Negative hinzufügen
- Dedizierte Testanmeldungs-Landingpage für Ihre Top-2-Anzeigengruppen erstellen
- Markenkampagne mit exakter Übereinstimmung hinzufügen — Budget $500/Monat
