# Email-Sequenz-Skill

## Wann aktivieren
- Aufbau einer Willkommens-E-Mail-Serie für neue Abonnenten oder Kunden
- Erstellung einer Nurture-Sequenz zum Bewegen von Leads in Richtung Kauf
- Schreiben einer Wiederengagement-Kampagne für inaktive Abonnenten
- Entwurf eines Post-Kauf-Onboarding-E-Mail-Flows
- A/B-Test von Betreffzeilen und E-Mail-Kopien

## Wann nicht verwenden
- Cold-Outreach-Sequenzen — verwenden Sie die Skill Email-Automation (GTM)
- Transaktions-E-Mails (Quittungen, Bestätigungen) — normalerweise durch Plattformvorlage
- Newsletter One-Offs — dies ist für automatisierte Sequenzen

## Anleitung

### Willkommens-E-Mail-Serie (5 E-Mails)

```
Schreiben Sie eine 5-E-Mail-Willkommenserie für neue [Abonnenten / Kunden / Testnutzer].

Über mein Unternehmen/Produkt: [beschreiben]
Neue Abonnenten-Persona: [wer sind sie, was wollen sie]
Ziel der Serie: [informieren / Vertrauen aufbauen / erste Kauftreiber / Testaktivierung]

E-Mail 1 (sofort): Willkommen + Versprechen erfüllen
E-Mail 2 (Tag 2): Ihr Tipp #1 / Insight (Wert, kein Pitch)
E-Mail 3 (Tag 4): Sozialer Beweis (Kundenstory / Fallstudie)
E-Mail 4 (Tag 7): Adressieren Sie die Haupteinwand oder Angst
E-Mail 5 (Tag 10): Soft CTA (Test / Demo / erster Kauf)

Für jede E-Mail:
- Betreffzeile (+ A/B-Variante)
- Vorschau-Text (40-50 Zeichen)
- Öffnungs-Hook
- Kernbotschaft (150-200 Wörter)
- CTA-Text und URL-Platzhalter
```

### Lead-Nurture-Sequenz

```
Schreiben Sie eine Nurture-Sequenz für Leads, die [Lead Magnet / Webinar beigetreten] heruntergeladen haben.

Lead-Profil: [beschreiben, was sie heruntergeladen/beigetreten haben und was das über sie aussagt]
Verkaufszyklus: [X] Tage durchschnittlich
Haupteinwände zum Kauf: [2-3 aufzählen]
Mein Produkt/Service: [beschreiben]

Entwerfen Sie eine [4/6/8]-E-Mail-Sequenz über [X] Wochen.
Mischung aus: Bildungsinhalten, sozialem Beweis, Einwandbehandlung, CTA.
Schrittweise Steigerung der Kaufabsicht — nicht bei jeder E-Mail verkaufen.
```

### Wiederengagement-Kampagne

```
Ich habe [X] Abonnenten, die seit [X] Monaten keine E-Mail geöffnet haben.
Meine Liste: [beschreiben — B2B / B2C, wofür haben sie sich angemeldet]

Schreiben Sie eine 3-E-Mail-Wiederengagement-Sequenz:
E-Mail 1: "Wir vermissen dich" — erinnere an den Wert, gebe einen Grund zur Wiederaufnahme
E-Mail 2: "Letzte Chance" — Dringlichkeit ohne manipulativ zu sein
E-Mail 3: "Auf Wiedersehen für jetzt" — sauberer Schnitt, Reanmeldungs-CTA, Unsubscribe-Rest

Ziel: 5-10% der inaktiven Abonnenten zurückgewinnen, Liste für Zustellbarkeit bereinigen.
```

### A/B-Test von Betreffzeilen

```
Generieren Sie 10 Betreffzeilenvarianten für diese E-Mail:

E-Mail-Thema: [beschreiben Sie, worum es in der E-Mail geht]
Ziel: [Öffnungsrate / Klickrate / Konvertierung]
Publikum: [beschreiben]
Aktuelle durchschnittliche Öffnungsrate: [X]%

Schreiben Sie 10 Varianten zum Testen von:
- Neugier-Lücke
- Spezifische Zahl/Statistik
- Persönlich/direkt ("Sie")
- Frage
- Dringlichkeit/Knappheit
- Vorteilsfokussiert
- Emoji-Variante
- Kurz (< 30 Zeichen) vs. lang (> 50 Zeichen)

Für jeden: Bewerten Sie die Wahrscheinlichkeit, eine Öffnungsrate von 25% zu übertreffen.
```

### Post-Kauf-Onboarding

```
Schreiben Sie eine Post-Kauf-Onboarding-Sequenz für [Produkt/Service].

Kunde hat gerade gekauft: [was sie gekauft haben]
Ziel des Onboardings: [sie zum Produktgebrauch / Freundesempfehlung / Kauferweiterung bewegen]
Zeit bis zum Wert: [wie lange bis sie den Vorteil sehen]
Häufige Gründe für Rückerstattung/Churn in den ersten 30 Tagen: [beschreiben]

Tag 0: Kaufbestätigung + unmittelbarer Wert (nicht nur Quittung)
Tag 1: "Erste Schritte" — eine Aktion zum Ergreifen
Tag 3: Check-in — "Haben Sie X versucht?"
Tag 7: Feature-Spotlight — das wertvollste Feature, das die meisten verpassen
Tag 14: Sozialer Beweis + Community-Einladung
Tag 30: Meilenstein-Check + Upsell/Referral
```

## Beispiel

**Benutzer:** Schreiben Sie eine 5-E-Mail-Willkommenserie für ein SaaS-Projektmanagement-Tool. Neue Testnutzer melden sich an, aber 60% erstellen niemals ihr erstes Projekt.

**E-Mail 1 (Tag 0 — sofort):**
Betreffzeile: Ihr Workspace ist bereit — eine Sache, die Sie gleich tun sollten
"Willkommen bei [Produkt]. Hier ist eine Sache, die während Ihrer ersten Woche einen großen Unterschied macht: [Ihr erstes Projekt in 2 Minuten erstellen]. So geht's genau → [Link]. Es dauert 2 Minuten und Sie sehen sofort, warum [X] Teams [Produkt] zur [Ergebnis] verwenden."

**E-Mail 2 (Tag 2):**
Betreffzeile: Der größte Fehler neuer Benutzer
"Die meisten Menschen richten [Produkt] ein und versuchen sofort, alles auf einmal zu migrieren. Tun Sie das nicht. Beginnen Sie mit einem Projekt — Ihrem aktivsten. [Hier ist das 10-Minuten-Setup, das ich machen würde, wenn ich Sie wäre] → [Link]."

[setzt sich für alle 5 E-Mails mit Betreffzeilen und Copy fort]
