# Social-Media-Manager-Fähigkeit

## Wann aktivieren
- Planung eines monatlichen oder wöchentlichen Social-Media-Inhaltskalenders
- Schreiben von plattformoptimierten Beiträgen (LinkedIn ≠ Instagram ≠ X ≠ TikTok)
- Aufbau einer Hashtag-Strategie für Ihre Nische
- Reaktion auf Kommentare und DMs im großen Maßstab
- Analyse der Social-Media-Leistung und Anpassung der Strategie

## Wann NICHT verwenden
- Bezahlte Social-Media-Werbung — verwenden Sie die Skill „Paid Ads"
- Influencer-Outreach — anderer Workflow
- Krisenkommunikation — beinhaltet PR-Urteilsvermögen über das Inhaltserstellen hinaus

## Anleitung

### Monatlicher Inhaltskalender

```
Erstellen Sie einen monatlichen Social-Media-Inhaltskalender.

Geschäft: [Art und kurze Beschreibung]
Plattformen: [LinkedIn / Instagram / X / TikTok / Facebook — wählen Sie]
Posting-Häufigkeit: [X mal/Woche pro Plattform]
Inhaltssäulen (3-5 Themen, über die Ihre Marke postet): [Listenansicht oder lass mich vorschlagen]
Anstehende Veranstaltungen/Kampagnen: [Produktstarts, saisonal, Ankündigungen]
Monat: [angeben]

Planen Sie jede Woche:
- Inhaltsthemen nach Tag
- Mix von Formaten (pädagogisch / werblich / persönlich / wiederverwendet / Engagement)
- Plattformspezifische Anpassungen (was auf LinkedIn vs. Instagram funktioniert)
- Ein "Hero"-Beitrag pro Woche (höchster Aufwand, verstärkt, wenn bezahlt)

Ausgabe: Wochenplan mit Post-Ideen für jede Plattform.
```

### Plattformspezifisches Copywriting

**LinkedIn:**
```
Schreiben Sie einen LinkedIn-Beitrag über [Thema].
Publikum: [beschreiben — Gründer / Vermarkter / Entwickler / HR]
Ziel: [Gedankenführerschaft / Lead-Generierung / Einstellung / Engagement]
Ton: [professionell / persönlich / konträr / storytelling]

LinkedIn-Format-Regeln:
- Hook in Zeile 1 (vor "...mehr" Abschnitt) — bringen Sie sie zum Klicken
- Kurze Absätze (max. 1-2 Sätze)
- Leerraum zwischen Absätzen
- Mit Frage oder klarem CTA enden
- Nicht mehr als 3 Hashtags (LinkedIn belohnt Tiefe, nicht Discovery)
- Optimale Länge: 150-300 Wörter für Engagement
```

**Instagram:**
```
Schreiben Sie eine Instagram-Bildunterschrift für [Inhalt/Bildbeschreibung].
Stimmung: [inspirierend / pädagogisch / Behind-the-Scenes / Produkt]
Markenstimme: [beschreiben]

Instagram-Format:
- Mit Hook führen (erste 125 Zeichen sichtbar vor "mehr")
- Emojis natürlich verwendet, nicht als Dekoration
- Hashtags: 5-10 gezielt, 2-3 breit (am Ende oder im ersten Kommentar)
- CTA: "Link in Bio" oder Frage zum Fahren von Kommentaren
- Story-Variante: passen Sie dieselbe Idee in 3-5 Story-Frames an
```

**X (Twitter):**
```
Schreiben Sie einen Tweet / Thread über [Thema].
Stil: [Einsicht / heißer Gedanke / Listicle Thread / Frage]

Tweet-Regeln:
- < 280 Zeichen für Standalone Tweets
- Für Threads: Hook Tweet + 4-8 unterstützende Tweets + Summary Tweet
- Prägnant, keine Abschwächung — X belohnt starke Meinungen
- Nicht mit "Ich" oder Ihrem Unternehmensname starten
- Quote-Tweet Ihren eigenen Inhalt, um Kommentar hinzuzufügen
```

**TikTok/Reels-Skript:**
```
Schreiben Sie ein [30/60/90]-Sekunden-Skript für [Thema].
Hook (erste 3 Sekunden): [was dem Scroll entgegenwirkt]
Format: [sprechender Kopf / Voiceover + Text / Tutorial]

TikTok-Struktur:
- Sekunde 0-3: Hook (sagen oder zeigen Sie das Interessanteste zuerst)
- Sekunde 4-15: Setup / Kontext
- Sekunde 16-45: Kerninhalt / Wert
- Sekunde 46-60: CTA oder Pointe
- Textüberlagerungen: 1-2 Schlüsselpunkte ständig auf Bildschirm
```

### Hashtag-Strategie

```
Erstellen Sie eine Hashtag-Strategie für [Plattform] und [Nische].

Plattform: [Instagram / LinkedIn / X / TikTok]
Nische: [beschreiben Sie Geschäft und Inhaltsthemen]
Aktuelle Kontogröße: [Follower]

Erstellen Sie für jede Plattform 3 Ebenen:
1. Nischen-Hashtags (klein — < 100K Beiträge): beste Chance, Ranking zu erreichen
2. Mittlere Hashtags (100K-1M Beiträge): Reichweite + Auffindbarkeit
3. Breite Hashtags (1M+ Beiträge): niedrige Chance auf Ranking, aber Markenzeichen

Empfehlung: [X] Hashtags pro Beitrag, [Verteilung über Ebenen]
Organisieren Sie in 3-5 Hashtag-Sets zum Drehen (vermeiden Sie Shadowban durch Wiederholung).
```

### Engagement-Skripte

```
Schreiben Sie Engagement-Antworttemplates für [Plattform].

Situationen zum Abdecken:
1. Generischer positiver Kommentar ("Love this!")
2. Durchdachter Kommentar, der eine echte Antwort verdient
3. Frage, die öffentlich beantwortet werden kann
4. Kritik oder negatives Feedback (nicht beleidigend)
5. Spam oder minderwertiger Kommentar (wann ignorieren/verstecken)

Ton: [Markenstimme]
Ziel: [Community aufbauen / zu DMs fahren / Reichweite erhöhen / Sachverstand zeigen]
```

### Analysinterpretation

```
Interpretieren Sie meine Social-Media-Leistung für [Monat].

Plattform: [welche]
Schlüsselmetriken:
- Reichweite: [X] (vs. letzten Monat: [X])
- Impressionen: [X]
- Engagement-Rate: [X]%
- Follower gewonnen: [X]
- Top-Beitrag: [beschreiben Sie es]
- Schlechtester Performer: [beschreiben Sie es]

Sagen Sie mir:
1. Welcher Inhaltstyp funktionierte am besten und warum?
2. Was sollte ich mehr / weniger tun?
3. Welche Tage/Zeiten trieben die meisten Engagement?
4. Was sind meine 3-Beitrag-Experimente für nächsten Monat basierend auf diesen Daten?
```

## Beispiel

**Nutzer:** Ich bin Gründer eines B2B-SaaS. Brauche eine Woche LinkedIn-Inhalte. Themen, über die ich mich interessiere: produktgesteuertes Wachstum, SaaS-Metriken und Aufbau in der Öffentlichkeit.

**Claude generiert 5 LinkedIn-Beiträge:**

**Montag (Einsicht):**
> Der größte Fehler, den SaaS-Gründer bei der Umwandlung von Testversion zu bezahlt machen:
> Es wöchentlich messen.
>
> Die Testversionskonvertierung ist eine Kohortenmetrik. Sie benötigen 30-60 Tage, bevor die Zahl etwas bedeutet.
>
> Hier ist das Framework, das ich stattdessen verwende: [3 Aufzählungszeichen]
>
> Was ist Ihre aktuelle Testversion-zu-bezahlt-Rate?

**Mittwoch (Geschichte/Aufbau in der Öffentlichkeit):**
> Wir haben diese Woche $50K MRR erreicht. Hier sieht das Dashboard tatsächlich aus:
> [Datenpunkt] [Datenpunkt] [Datenpunkt]
> Die Zahl, die uns am meisten überrascht hat: [Einsicht]
> Und die, die mich immer noch wach hält: [Verletzlichkeit]

[setzt sich für alle 5 Tage mit Tages-/Format-Vielfalt fort]
