# Habilidad del Gerente de Redes Sociales

## Cuándo activar
- Planificación de un calendario de contenidos de redes sociales mensuales o semanales
- Escritura de publicaciones optimizadas para plataformas (LinkedIn ≠ Instagram ≠ X ≠ TikTok)
- Construcción de una estrategia de hashtags para su nicho
- Respuesta a comentarios y mensajes directos a escala
- Análisis del desempeño de redes sociales y ajuste de la estrategia

## Cuándo NO usar
- Publicidad pagada en redes sociales — use la habilidad de anuncios pagados
- Divulgación de influencers — flujo de trabajo diferente
- Comunicación de crisis — implica juicio de relaciones públicas más allá de la creación de contenido

## Instrucciones

### Calendario de contenidos mensual

```
Construya un calendario de contenidos mensuales de redes sociales.

Negocio: [tipo y breve descripción]
Plataformas: [LinkedIn / Instagram / X / TikTok / Facebook — elija cuáles]
Frecuencia de publicación: [X veces/semana por plataforma]
Pilares de contenido (3-5 temas sobre los que publica su marca): [lista o déjame sugerir]
Eventos/campañas próximos: [lanzamientos de productos, estacionales, anuncios]
Mes: [especificar]

Para cada semana, planifique:
- Temas de contenido por día
- Mezcla de formatos (educativo / promocional / personal / reutilizado / participación)
- Adaptaciones específicas de plataforma (qué funciona en LinkedIn vs. Instagram)
- Un publicación "hero" por semana (máximo esfuerzo, impulsada si se paga)

Salida: plan semana a semana con ideas de publicación para cada plataforma.
```

### Copia específica de plataforma

**LinkedIn:**
```
Escriba una publicación de LinkedIn sobre [tema].
Audiencia: [describa — fundadores / comercializadores / desarrolladores / RR.HH.]
Objetivo: [liderazgo de pensamiento / generación de leads / contratación / participación]
Tono: [profesional / personal / contrario / narración]

Reglas de formato de LinkedIn:
- Gancho en línea 1 (antes del corte "...más") — hágalos hacer clic
- Párrafos cortos (máximo 1-2 oraciones)
- Espacio en blanco entre párrafos
- Termina con una pregunta o CTA clara
- No más de 3 hashtags (LinkedIn recompensa la profundidad, no el descubrimiento)
- Longitud óptima: 150-300 palabras para participación
```

**Instagram:**
```
Escriba un pie de foto de Instagram para [descripción de contenido/imagen].
Vibe: [inspirador / educativo / detrás de escenas / producto]
Voz de marca: [describa]

Formato de Instagram:
- Lidere con el gancho (primeros 125 caracteres visibles antes de "más")
- Emojis usados naturalmente, no como decoración
- Hashtags: 5-10 dirigidos, 2-3 amplios (al final o primer comentario)
- CTA: "enlace en biografía" o pregunta para impulsar comentarios
- Variante de historia: adapte la misma idea en 3-5 fotogramas de historia
```

**X (Twitter):**
```
Escriba un tweet / hilo sobre [tema].
Estilo: [perspicacia / toma caliente / hilo listicle / pregunta]

Reglas de tweets:
- < 280 caracteres para tweets independientes
- Para hilos: tweet gancho + 4-8 tweets de apoyo + tweet de resumen
- Contundente, sin cobertura — X recompensa opiniones sólidas
- No comience con "Yo" o el nombre de su empresa
- Cita tu propio contenido para agregar comentario
```

**Script de TikTok/Reels:**
```
Escriba un script de [30/60/90] segundos para [tema].
Gancho (primeros 3 segundos): [qué detiene el desplazamiento]
Formato: [cabeza parlante / voz en off + texto / tutorial]

Estructura de TikTok:
- Segundo 0-3: Gancho (diga o muestre lo más interesante primero)
- Segundo 4-15: Configuración / contexto
- Segundo 16-45: Contenido central / valor
- Segundo 46-60: CTA o remate
- Superposiciones de texto: 1-2 puntos clave en pantalla en todo momento
```

### Estrategia de hashtags

```
Construya una estrategia de hashtags para [plataforma] y [nicho].

Plataforma: [Instagram / LinkedIn / X / TikTok]
Nicho: [describa negocio y temas de contenido]
Tamaño de cuenta actual: [seguidores]

Para cada plataforma, cree 3 niveles:
1. Hashtags de nicho (pequeño — < 100K publicaciones): máxima oportunidad de clasificación
2. Hashtags de tamaño medio (100K-1M publicaciones): alcance + descubribilidad
3. Hashtags amplios (1M+ publicaciones): baja probabilidad de clasificación pero señal de marca

Recomendación: [X] hashtags por publicación, [distribución en niveles]
Organícelos en 3-5 conjuntos de hashtags para rotar (evite la prohibición de sombra por repetición).
```

### Scripts de participación

```
Escriba plantillas de respuesta de participación para [plataforma].

Situaciones a cubrir:
1. Comentario positivo genérico ("¡Me encanta!")
2. Comentario reflexivo que merece una respuesta real
3. Pregunta que se puede responder públicamente
4. Crítica o comentario negativo (no abusivo)
5. Comentario de spam o baja calidad (cuándo ignorar/ocultar)

Tono: [voz de marca]
Objetivo: [construir comunidad / enviar a DM / aumentar alcance / mostrar experiencia]
```

### Interpretación de análisis

```
Interprete mi desempeño en redes sociales para [mes].

Plataforma: [cuál]
Métricas clave:
- Alcance: [X] (vs. mes pasado: [X])
- Impresiones: [X]
- Tasa de participación: [X]%
- Seguidores ganados: [X]
- Publicación superior: [describa]
- Peor desempeño: [describa]

Dígame:
1. ¿Qué tipo de contenido funcionó mejor y por qué?
2. ¿Qué debo hacer más / menos?
3. ¿Qué días/horas impulsaron la mayor participación?
4. ¿Cuál es mi experimento de 3 publicaciones para el próximo mes según estos datos?
```

## Ejemplo

**Usuario:** Soy fundador de SaaS B2B. Necesito una semana de contenido de LinkedIn. Temas que me interesan: crecimiento dirigido por productos, métricas de SaaS y construcción en público.

**Claude genera 5 publicaciones de LinkedIn:**

**Lunes (Perspicacia):**
> El error más grande que cometen los fundadores de SaaS con la conversión de prueba a pago:
> Medirlo semanalmente.
>
> La conversión de prueba es una métrica de cohorte. Necesita 30-60 días antes de que el número signifique algo.
>
> Aquí está el marco que uso en su lugar: [3 viñetas]
>
> ¿Cuál es su tasa actual de prueba a pago?

**Miércoles (Historia/Construcción en público):**
> Alcanzamos $50K MRR esta semana. Así es como se ve realmente el panel:
> [punto de datos] [punto de datos] [punto de datos]
> El número que más nos sorprendió: [perspicacia]
> Y el que aún me quita el sueño: [vulnerabilidad]

[continúa para los 5 días con variedad de día/formato]
