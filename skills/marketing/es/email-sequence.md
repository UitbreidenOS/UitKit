# Habilidad de Secuencia de Correo Electrónico

## Cuándo activar
- Crear una serie de correos electrónicos de bienvenida para nuevos suscriptores o clientes
- Crear una secuencia de nutrición para mover leads hacia la compra
- Escribir una campaña de reenganche para suscriptores inactivos
- Diseñar un flujo de incorporación posterior a la compra
- Prueba A/B de líneas de asunto y copy de correo electrónico

## Cuándo no usar
- Secuencias de alcance en frío — usa la habilidad de automatización de correo electrónico (GTM)
- Correos electrónicos transaccionales (recibos, confirmaciones) — generalmente basados en plantilla de plataforma
- One-offs de boletín informativo — esto es para secuencias automatizadas

## Instrucciones

### Serie de correos electrónicos de bienvenida (5 correos)

```
Escriba una serie de 5 correos electrónicos de bienvenida para [suscriptores / clientes / usuarios de prueba] nuevos.

Sobre mi negocio/producto: [describe]
Persona de nuevo suscriptor: [quiénes son, qué quieren]
Objetivo de la serie: [educar / generar confianza / impulsar primera compra / activar prueba]

Correo 1 (inmediatamente): Bienvenida + entrega la promesa
Correo 2 (día 2): Tu consejo #1 / insight (valor, sin pitch)
Correo 3 (día 4): Prueba social (historia de cliente / estudio de caso)
Correo 4 (día 7): Aborda la principal objeción o miedo
Correo 5 (día 10): CTA suave (prueba / demostración / primera compra)

Para cada correo:
- Línea de asunto (+ variante A/B)
- Texto de vista previa (40-50 caracteres)
- Hook de apertura
- Mensaje central (150-200 palabras)
- Texto CTA y marcador de posición URL
```

### Secuencia de nutrición de leads

```
Escriba una secuencia de nutrición para leads que descargaron [lead magnet / asistieron a webinar].

Perfil de lead: [describe qué descargaron/asistieron y qué dice sobre ellos]
Ciclo de ventas: [X] días promedio
Principales objeciones a la compra: [lista 2-3]
Mi producto/servicio: [describe]

Diseña una secuencia de [4/6/8] correos electrónicos sobre [X] semanas.
Mezcla de: contenido educativo, prueba social, manejo de objeciones, CTA.
Aumenta gradualmente la intención de compra — no vendas en cada correo.
```

### Campaña de reenganche

```
Tengo [X] suscriptores que no han abierto un correo en [X] meses.
Mi lista: [describe — B2B / B2C, para qué se inscribieron]

Escriba una secuencia de reenganche de 3 correos:
Correo 1: "Te extrañamos" — recuerda el valor, da una razón para reengancharse
Correo 2: "Última oportunidad" — urgencia sin ser manipulador
Correo 3: "Adiós por ahora" — ruptura limpia, resuscribirse CTA, desuscribir el resto

Objetivo: Recuperar 5-10% de suscriptores inactivos, limpiar lista para entregabilidad.
```

### Prueba A/B de líneas de asunto

```
Genere 10 variantes de línea de asunto para este correo:

Tema del correo: [describe de qué se trata el correo]
Objetivo: [tasa de apertura / clic / conversión]
Audiencia: [describe]
Tasa de apertura promedio actual: [X]%

Escriba 10 variantes probando:
- Brecha de curiosidad
- Número/estadística específica
- Personal/directo ("Tú")
- Pregunta
- Urgencia/escasez
- Enfocado en beneficios
- Variante emoji
- Corto (< 30 caracteres) vs. largo (> 50 caracteres)

Para cada: evalúa la probabilidad de superar una tasa de apertura del 25%.
```

### Incorporación posterior a la compra

```
Escriba una secuencia de incorporación posterior a la compra para [producto/servicio].

Cliente acaba de comprar: [qué compraron]
Objetivo de incorporación: [hacer que usen el producto / refieran un amigo / expandan compra]
Tiempo para valor: [cuánto tiempo hasta que vean beneficio]
Razones comunes de reembolso/churn en primeros 30 días: [describe]

Día 0: Confirmación de compra + valor inmediato (no solo recibo)
Día 1: "Primeros pasos" — una acción a tomar
Día 3: Check-in — "¿Probaste X?"
Día 7: Destacado de características — característica más valiosa que la mayoría se pierden
Día 14: Prueba social + invitación de comunidad
Día 30: Check de hito + upsell/referencia
```

## Ejemplo

**Usuario:** Escribe una serie de 5 correos electrónicos de bienvenida para una herramienta SaaS de gestión de proyectos. Los nuevos usuarios de prueba se registran pero el 60% nunca crea su primer proyecto.

**Correo 1 (Día 0 — inmediatamente):**
Asunto: Tu workspace está listo — una cosa para hacer ahora
"Bienvenido a [Producto]. Aquí hay una cosa que marca la diferencia en tu primera semana: [crear tu primer proyecto en 2 minutos]. Así es exactamente cómo → [enlace]. Te toma 2 minutos y verás instantáneamente por qué [X] equipos usan [Producto] para [resultado]."

**Correo 2 (Día 2):**
Asunto: El mayor error de los nuevos usuarios
"La mayoría de las personas configuran [Producto] e inmediatamente intentan migrar todo de una vez. No lo hagas. Comienza con un proyecto — el más activo. [Aquí está la configuración de 10 minutos que haría si fuera tú] → [enlace]."

[continúa para los 5 correos con líneas de asunto y copy]
