# LinkedIn SDR

## Quand activer

- Vous menez une campagne de prospection B2B et avez besoin d'adapter la portée des messages sur LinkedIn sans violer les conditions d'utilisation de la plateforme
- Vous optimisez les taux d'acceptation de connexion, les taux de réponse aux messages ou la conversion en réunions qualifiées
- Vous avez besoin d'auditer votre stratégie de vente LinkedIn pour la conformité en matière de sécurité, la décroissance de l'ISS ou la qualité de l'engagement
- Vous construisez une stratégie de prospection menée par le contenu où l'engagement précède la demande
- Vous avez besoin de logique de décision pour la segmentation Sales Navigator ou la séquence de préchauffage

N'activez PAS pour : le réseautage personnel, la formation des équipes de vente interne, les campagnes orientées vers le consommateur ou les comptes de moins de 2 semaines.

## Quand NE PAS utiliser

- Si vous n'avez pas un profil LinkedIn complet avec photo, titre et résumé (corrigez d'abord)
- Si vous envoyez 100+ demandes de connexion quotidiennement ou si vous automatisez avec des outils tiers qui violent les conditions d'utilisation de LinkedIn
- Si vos filtres Sales Navigator sont : séniorité large + aucun filtre d'activité (vous visiterez des prospects inactifs)
- Si vous n'avez pas validé votre ICP (profil de client idéal) — les approches au hasard échouent rapidement
- Si vous ciblez des prospects dans des pays avec des restrictions GDPR/CCPA sans mécanismes de consentement
- Si votre compte a moins de 30 jours ou a été signalé comme spam (LinkedIn bannit l'ombre de manière agressive)

## Instructions

### Phase 1 : Fondation de sécurité et de profil

**Liste de contrôle pré-prospection (non négociable) :**

1. Profil complet : Photo professionnelle, titre avec mot-clé (par exemple, « Accélérateur de revenus B2B SaaS »), résumé de 150+ caractères, 3+ messages ou partages récents, recommandations sur 5-7 compétences pertinentes.
2. ISS actuel (Indice de vente sociale) : Vérifiez sur sales.linkedin.com. Objectif : 70+. Si moins de 60, passez 2 semaines en engagement avant la prospection.
3. Préchauffez votre compte : 15-20 min/jour en engageant 3-5 messages d'industrie (comme + commentaire) pendant 7 jours avant le premier message. LinkedIn suit « l'établissement du modèle » et récompense les comptes avec un engagement cohérent en début de période.
4. Définissez les garde-fous de prospection dans votre calendrier/CRM : Max 20 demandes de connexion/jour, max 50 messages/jour, échelonner les envois (pas tous à 9h), mettre en pause si le taux d'acceptation tombe en dessous de 25% pendant 3 jours consécutifs.

**Formule d'amélioration de l'ISS (si le départ est en dessous de 70) :**
- Engagement quotidien : 5 mentions j'aime + 3 commentaires substantiels (5-15 mots, pas « super message ! ») = +2-3 points ISS/jour
- Un message hebdomadaire (texte, lien d'article ou document) : +10 ISS sur 2 semaines
- Taux d'acceptation de l'e-mail : Maintenez au-dessus de 30% (important pour la visibilité de l'algorithme LinkedIn)
- Complétude du profil : Photo, titre, résumé, compétences = +15 plancher ISS

### Phase 2 : Filtrage Sales Navigator (La couche de précision)

**Filtres non négociables :**

```
1. Séniorité : 
   - Pour l'entreprise : C-suite + VP + Directeur (3 titres seulement, pas « manager »)
   - Pour le marché intermédiaire : Directeur + Senior Manager
   - Pour les PME : Fondateur + VP + Directeur

2. Fonction : 
   - Correspondre au comité d'achat : Généralement VP Ventes, VP Marketing, VP Ops, CFO, CTO
   - Éviter : RH, Coordinateur des opérations, Recrutement (mauvaises conversations DM)

3. Taille de l'entreprise : 
   - Recherche : filtre « company_headcount » + votre plage ICP
   - Par exemple, 50-200 pour la haute touche, 500-5000 pour le marché intermédiaire

4. CRITIQUE — Filtre d'activité : 
   - « A publié sur LinkedIn au cours des 30 derniers jours » (case à cocher dans Sales Navigator)
   - Cela améliore 10x les taux de réponse ; les prospects inactifs coûtent des cycles
   - Supprimer : Quiconque inactif >60 jours

5. Affinements optionnels : 
   - Vertical industrie (le cas échéant)
   - Mot-clé de compétence (par exemple, « Salesforce » pour la prospection axée sur les opérations)
   - Taux d'engagement : Si le profil public montre <5 messages en 6 mois, déprioritiser
```

**Construction de liste hebdomadaire** : 30 minutes dimanche soir. Construisez 3 listes de 50 personnes chacune (150 au total pour la semaine). Vérifiez « publié récemment » sur chacun. Cela prévient la dissémination.

### Phase 3 : La séquence de préchauffage en 3 étapes

**Étape 1 : Engagement du contenu (Jours 1-3, avant tout message)**

Action : S'engager avec leurs 2-3 derniers messages. Aimer + commenter (non-spammeur).

Modèle pour les commentaires (être spécifique au contenu du message) :
```
« [Nom], cela résonne — j'ai vu [découverte spécifique de leur message] générer 30% [métrique] plus élevée dans [contexte]. 
Avez-vous mesuré [question de suivi] dans votre [industrie/équipe] ? »
```

Pourquoi : L'algorithme de LinkedIn fait apparaître votre profil dans leurs notifications. Vous devenez « un visage » avant de demander. Commentez comme engagement : 2-3 par message, substantiel, pas de spam emoji.

**Étape 2 : Demande de connexion avec un crochet spécifique (Jour 3-4)**

Règles (strictes) :
- Max 300 caractères y compris le message
- Zéro mention de produit
- Une et une seule référence spécifique : message récent, connexion mutuelle, groupe partagé ou accomplissement
- Ton conversationnel (pas « synchronisons-nous »)

Modèles par type de crochet :

**Crochet de message récent :**
```
Bonjour [Nom] — J'ai adoré votre message sur [sujet spécifique]. Nous travaillons avec [contexte similaire] et voyons [résultat similaire]. 
Ce serait formidable de rester connecté. -[Votre nom]
```
Nombre de caractères : ~120-150. Sûr.

**Crochet de connexion mutuelle :**
```
Bonjour [Nom] — [Contact mutuel] parle hautement de vous. Je suis [votre rôle] chez [entreprise] en mettant l'accent sur [votre domaine]. 
J'adorerais me connecter.
```
Nombre de caractères : ~100-130. Sûr.

**Crochet de groupe partagé ou d'événement :**
```
C'est super de vous voir dans [groupe LinkedIn/événement]. Aussi en suivant [sujet spécifique que vous engagez tous les deux]. 
Connectons-nous. -[Nom]
```
Nombre de caractères : ~90-110. Sûr.

**Crochet de réussite de l'entreprise (financement, nouvelle embauche, lancement de produit) :**
```
[Nom] — J'ai vu [Entreprise] juste [accomplissement]. Bravo ! Nous travaillons avec des équipes en augmentant un [domaine] similaire. 
Heureux de se connecter. -[Votre nom]
```
Nombre de caractères : ~110-140. Sûr.

Envoi : Répartissez 20 demandes sur 24 heures (2-3 par heure, heures aléatoires). 9h, 14h45, 19h30 (varier quotidiennement).

**Étape 3 : Message 48 heures après l'acceptation (Timing exact)**

Règle critique : Attendez au minimum 48 heures. LinkedIn signale le « messagerie instantanée » comme similaire à un bot. Le temps d'attente signale que vous êtes humain.

Cette structure de message :
- **Longueur** : Max absolu 50 mots (pré-écrit, compter, couper)
- **Format** : Une question ouverte, zéro pitch produit, zéro lien
- **Ton** : Conversationnel, spécifique à leur profil ou activité récente

Modèle:
```
Salut [Nom], merci de vous être connecté ! 

J'ai remarqué que vous avez été [activité spécifique — par exemple, « message sur l'IA dans les processus de vente »] dernièrement. 

Qu'est-ce qui vous motive en ce moment ?
```

Nombre de mots : 28 mots. Envoyez à l'heure aléatoire (pas heures de bureau : essayez 23h15, 6h47 — imite le comportement humain).

**Pourquoi ce timing et cette structure :**
- Attente de 48 heures : La notification d'acceptation efface leur boîte de réception ; votre message arrive frais
- Limite de 50 mots : 47% des DM LinkedIn ne sont pas ouverts ; la brièveté gagne
- Une question : Déclenche l'obligation de réponse (réciprocité psychologique)
- Pas de produit : Vous demandez, ne pas vendre — maintient la curiosité
- Pas de liens : LinkedIn supprime les messages avec des liens dans la priorité de livraison ; garder pour le message 4+

### Phase 4 : La tactique de la note vocale (levée de réponse 2-3x)

**Quand utiliser** : Après acceptation de la connexion, si le prospect est votre cible 1-1 (pas prospection de masse).

**Benchmark** : Les DM vocaux atteignent 45-60% de taux de réponse vs. 12-18% pour le texte. Ce n'est pas marginal.

**La mécanique :**
1. Envoyer le message texte d'abord (de Phase 3)
2. Attendre 2-4 heures (laissez-le lire le texte)
3. Envoyer une note vocale de 30 secondes via DM

**Ce qu'enregistrer** (structure de script) :

```
[Salutation — utilisez leur nom, ton chaleureux]
« Salut [Nom], c'est [Votre nom] de [Entreprise]. »

[Observation spécifique — doit référencer leur profil, message ou entreprise]
« J'ai vu votre [message récent sur X / entreprise vient d'annoncer Y / vous êtes chez la société Z]. 
Je suis vraiment impressionné par votre [chose spécifique qu'ils ont faite]. »

[Une question personnalisée — naturelle, pas vendeuse]
« Question rapide : [question spécifique à leur rôle/industrie/activité récente]. 
Comment pensez-vous à cela en ce moment ? »

[Fermeture douce — pas de demande]
« Heureux de continuer ici ou de sauter à une conversation rapide si vous préférez. À bientôt ! »
```

**Règles de livraison :**
- Parlez de façon conversationnelle, pas lire le script (un coup, authentique)
- 28-35 secondes (auto-coupe à 60 ; rester en dessous de la lecture complète)
- Appelez-les par leur prénom (l'informalité signale l'authenticité)
- Assez spécifique pour qu'il ne PUISSE PAS être modélisé (la personnalisation est la valeur entière)

**Exemple de note vocale (script complet — 32 secondes) :**

```
« Salut Sarah, c'est Mike de Velocity. Alors je scrollais sur LinkedIn et j'ai vu ton message la semaine dernière sur 
l'implémentation de l'IA dans le processus de vente — une perspective vraiment pointue. Nous avons travaillé avec des équipes 
qui font exactement cela. Question rapide : comment pensez-vous à la partie gestion du changement avec votre équipe de vente ? 
C'est où la plupart des gens hit friction. De toute façon, j'adorerais entendre vos pensées. À bientôt ! »
```

**Pourquoi cela fonctionne :**
- Intimité : La voix est 10x plus humaine que le texte
- Signal : Vous avez passé 60 secondes sur eux (les signaux de rareté du respect)
- Engagement : La voix est plus difficile à ignorer que le texte (45% de personnes écoutent vs. 80% qui lisent)
- Rappel : Ils se sentent obligés de répondre en nature (réciprocité)

Limiter à : 5-10 par jour (c'est une tactique de précision, pas un jeu de volume).

### Phase 5 : Flux de conversation LinkedIn DM (Messages 2-5)

**Benchmark de séquence de messages** : 18% de réponse au premier message → 42% répondent finalement si vous suivez cette séquence.

**Message 2 (12-18 heures après la réponse Q1) :**
- S'ils ont répondu : « J'aime que vous pensiez à [leur réponse]. Suivi rapide : [développer sur leur réponse] »
- Si pas de réponse à Q1 : Envoyer un angle différent — référence quelque chose de leur profil, pas la question.
- Max 40 mots, une question ouverte toujours.

**Message 3 (24 heures après M2) :**
- Passage des questions à l'insight
- « Étant donné ce que vous avez dit sur [leur contexte], j'ai vu les équipes se déplacer plus vite quand [modèle spécifique]. Certains de cela s'appliquent ? »
- Premier placement de valeur douce autorisé (pas produit, mais méthodologie/insight)
- 45 mots max, une question.

**Message 4+ (seulement s'ils s'engagent, espacement 2+ jours) :**
- « Cela pourrait valoir un appel de 15 min pour parler de [résultat commercial spécifique auquel ils se soucient]. 
Ouvrir [jour] ou [jour] ? »
- Première demande spécifique autorisée (réunion, pas produit)

**Règle de zéro lien** : N'envoyez pas de liens dans les messages 1-2. LinkedIn supprime la livraison. Message 3 en avant, un lien est acceptable s'il est intégré dans le contexte (« Notre rapport de recherche sur [sujet] » + lien).

**S'ils se taisent** (message 2+ pas de réponse) : 
- N'envoyez pas d'autre message pendant 7 jours
- S'engager avec leur prochain message publiquement (aimer + commentaire)
- Envoyer un DM supplémentaire 8 jours plus tard : « J'ai vu votre message sur [sujet], je suis certainement d'accord sur [point]. Je pense toujours qu'une conversation rapide pourrait être précieuse — 15 min ? »
- Après cela : Passer à l'allaitement (réengagement trimestriel), ne pas forcer.

### Phase 6 : Prospection menée par le contenu (Le multiplicateur)

**Cela inverse le script : ils vous envoient un message.**

**Le mécanisme :**

1. **Rythme de contenu hebdomadaire** (1 message tous les 4-7 jours) :
   - Sujet : Un thème central que vous possédez (par exemple, optimisation du processus de vente, IA en entreprise, intelligence des revenus)
   - Format : Publication de texte (3-4 phrases max) + lien OU article natif (Les articles LinkedIn obtiennent 2x la portée)
   - Angle : Partager des données spécifiques, un insight contrarian ou un cas client (anonyme)

   Exemple de message :
   ```
   « Analysé 200 équipes de vente le trimestre dernier. Blocker #1 empêchant l'adoption de l'IA : 
   Pas la technologie, mais la gestion du changement. 78% des équipes qui ont résolu cela en premier 
   ont vu une augmentation de productivité de 20% + en 60 jours. Le reste est coincé dans les pilotes. 
   Quel est votre plus gros point de friction ? »
   ```

2. **Identifier les engageurs** : Quiconque aime ou commente sur votre message est un prospect chaud (ils ont validé votre espace de problème).

3. **Prospection avec contexte** (envoyer dans les 24 heures de leur engagement) :
   ```
   Salut [Nom], 
   
   J'ai vu votre commentaire sur mon message sur [sujet]. Vous avez mentionné [leur commentaire spécifique]. 
   C'est exactement ce que nous voyons avec [votre contexte client]. 
   Ce serait précieux de comparer les notes — 15 min cette semaine ?
   ```
   
   Ce n'est PAS froid. C'est chaud parce qu'ils se sont engagés en premier. Taux d'acceptation : 40-50%.

**Pourquoi cela compose :**
- Vous postez 1x/semaine = 4 messages/mois = 20-40 engageurs/mois
- Chaque engageur est un prospect chaud (conversion 25-30% à la réunion vs. 3-5% froid)
- Votre contenu établit l'autorité (ISS +20-30 de la publication régulière)
- L'algorithme LinkedIn récompense les afficheurs constants → Votre prospection reçoit une meilleure livraison

**Garde-fou de fréquence** : 1 message par 4-7 jours. 3+ messages/semaine déclenche une pénalité d'« auto-promotion » (suppression de l'algorithme).

### Phase 7 : Sécurité et conformité (Règles absolues)

**Violations des conditions d'utilisation de LinkedIn qui déclenchent l'interdiction d'ombre (30-90 jours) :**

1. **Automatisation par lot** : 
   - N'envoyez jamais 50 messages à 9 h 00 h 00 (semble robotique)
   - Randomiser tous les temps d'envoi (utilisez un CRM comme HubSpot, Salesloft ou Outreach qui stagger-envoie)
   - Minimum 45-60 secondes d'écart entre chaque action

2. **Spam de demande de connexion** :
   - Max 20/jour (limite indiquée par LinkedIn ; aller plus haut = détection d'algorithme)
   - Si le taux d'acceptation tombe en dessous de 25%, ARRÊTEZ pendant 3 jours (pause/diagnostics)
   - Si moins de 20%, votre messagerie est faible — modèle d'audit

3. **Volume de messages** :
   - Max 50 DM/jour
   - Ne jamais envoyer à la même personne plus d'une fois si elle ne répond pas (attendre 7+ jours minimum)

4. **Outils d'automatisation à NE JAMAIS utiliser** :
   - Bots de message de masse qui envoient une copie identique (LinkedIn signale, bannira)
   - Outils d'endossement automatique
   - Bots de visualisation de profil
   - Générateurs d'activité falsifiée
   - Les outils qui « garantissent » l'engagement (tous signalés)

   **Outils sûrs** (alignés avec les conditions d'utilisation) :
   - HubSpot Sales Hub (intégration LinkedIn, respecte les limites de débit)
   - Salesloft (stagger-send natif, timing humain)
   - Outreach (grade entreprise, étouffe correctement)
   - Manuel + Google Sheets (si discipliné sur le timing)

5. **Signaux de santé du compte** (vérifier hebdomadaire) :
   - ISS en baisse? Réduire le volume de prospection de 30%, augmenter l'engagement à 10 min/jour
   - Le taux d'acceptation de connexion chute ? Modèle d'audit (référence trop vague ?), mettre en pause 2 jours
   - Les messages ne sont pas livrés ? (Ils vont au dossier « Autre » — signifie que la messagerie est vue comme spam) Arrêtez, engagez 1 semaine, redémarrez

6. **Arbre de décision d'escalade** :

   ```
   Si le taux d'acceptation < 20% pendant 3 jours :
     → PAUSE de prospection
     → Audit de messagerie de demande de connexion (la référence est-elle assez spécifique ?)
     → Augmenter l'engagement à 15 min/jour pendant 5 jours
     → Reprendre à 10/jour (pas 20)
   
   Si le taux de réponse < 8% aux DM :
     → La question n'est pas assez convaincante
     → Rotation vers différentes questions (voir modèles Phase 3-5)
     → Envisager la tactique de note vocale
   
   Si l'ISS chute >10 points en semaine :
     → Pénalité d'engagement (l'algorithme a détecté engagement de faible qualité)
     → Arrêter la prospection immédiatement
     → 2 semaines : 20 min/jour commentaires engagés sur les messages de l'industrie
     → Reconstruire ISS à 75+
     → Redémarrer à 50% du volume précédent
   
   Si le compte shadowbanné (les messages vont au dossier Autre, l'acceptation de connexion s'arrête) :
     → Pause de 30 jours sur TOUTE la prospection
     → Engagement de profil seulement (aimer, commenter, message)
     → Contacter le support LinkedIn si cela persiste
   ```

**Note géographie/conformité** : 
- RGPD (UE) : Message seulement les prospects qui ont explicitement opté pour ou travail dans B2B (B2C plus strict)
- CCPA (Californie) : LinkedIn est couvert ; assurez-vous que le contexte commercial est clair
- Pas de scraping (interdit) ; construction de liste manuelle seulement

### Phase 8 : Cadence hebdomadaire et mesure

**Allocation de temps (en supposant 2-3 heures/jour sur LinkedIn) :**

```
Lundi : 
  - Construire une liste de cibles hebdomadaire (30 min, 150 personnes)
  - Engagement sur les 5 messages récents des prospects (30 min)

Mardi-Vendredi :
  - Prospection : 8-12 demandes de connexion (échelonnée, 30 min)
  - Suivi DM aux connexions chauds (20 min)
  - Engagement de contenu (10 min, rester visible)
  - Un contenu rédigé (30 min)

Vendredi :
  - Notes vocales à 5-8 prospects de haut potentiel (15 min d'enregistrement, 15 min d'écoute des réponses)
  - Métriques d'audit (15 min)

Fin de semaine :
  - Poster une fois (30 min d'écriture/édition)
  - Engagement deep-dive (1-2 commentaires sur 10 messages de l'industrie, 20 min)
  - Préparation de la liste de la semaine prochaine (15 min)
```

**Métriques à suivre hebdomadaire** (dans une feuille de calcul ou un CRM) :

| Métrique | Cible | Action si en dessous |
|--------|--------|-----------------|
| Demandes de connexion envoyées | 80-100/semaine | Vérifier la qualité du filtre |
| Taux d'acceptation de connexion | >25% | Audit spécificité de messagerie |
| Taux de réponse du premier message | >12% | Modèles de question de rotation |
| Taux de réponse de note vocale | >45% | Qualité (son, spécificité) |
| Engagement de contenu (message) | >30 engagements | Le sujet ne résonne pas |
| Score ISS | 70+ | 2 semaines d'engagement focus |
| Prospects qualifiés (pipeline) | 2-4/semaine | Vérifier le taux de conversion |
| Réunions réservées | 1-2/semaine | Compétences en vente (pas compétence LinkedIn) |

**Liste de contrôle mensuelle :**

- [ ] ISS soutenu >70 (sinon, pause de prospection ce mois-ci)
- [ ] Calendrier de contenu pour le mois prochain (4 messages minimum, thèmes définis)
- [ ] 3 meilleurs modèles de demande de connexion performants (benchmark : >30% acceptation)
- [ ] Meilleure question qui encourage les réponses (benchmark : >15%)
- [ ] Audit de sécurité : Tout quasi-violation ? Ajuster le volume.
- [ ] Taux de conversion du message à la réunion (objectif : 5-8%)

## Exemple

**Scénario réel : Outil RevOps SaaS B2B, ciblant les directeurs des opérations de revenu dans les entreprises financées en série B+.**

**Définition de l'ICP :**
- Directeur des opérations de revenus chez les entreprises de 50-500 personnes
- SaaS, FinTech ou B2B2C
- L'entreprise a levé une série B+ au cours des 18 derniers mois
- A publié sur LinkedIn au cours des 30 derniers jours

**Configuration de la semaine 1 :**

1. Audit de profil : Titre mis à jour en « Accélération des revenus du marché | Conseiller RevOps | 200+ entreprises augmentées »
2. Vérification ISS : 72 (bon, procéder)
3. Préchauffage : 5 jours d'engagement de 15 min/jour (aimer + commenter sur 10 messages RevOps/CRO)
4. Construction de liste : 150 cibles (3 listes de 50), filtre par case « publié récemment »
5. Contenu : Planifier un message : « Nous avons analysé 340 entreprises. Les organisations qui ont unifié les ventes + la planification RevOps ont vu 28% d'accélération plus rapide de quota. D'habitude séparés : silos de données, objectifs mal alignés, cycles de travail. Vaut-il la peine d'en parler en interne ? »

**Prospection des semaines 2-3 :**

Demande de connexion au premier objectif (exemple réel) :

```
Bonjour James,

J'ai vu votre message la semaine dernière sur la mise en œuvre de l'IA dans les prévisions de revenus. 
Une perspective pointue sur la partie gestion du changement — c'est exactement ce que nous 
voyons avec notre base de clients chez [Entreprise]. 

Ce serait formidable de rester connecté. - [Votre nom]
```

*Nombre de caractères : 243. Spécifique (son message), pas de pitch, référence au défi partagé.*

Envoyez à : Mardi 14h47 (heure aléatoire).

*Résultat : James accepte mercredi 11h.*

Attendez exactement 48 heures. Envoyer le message vendredi 13h33 (heure différente) :

```
Salut James,

Merci de vous être connecté ! 

J'ai remarqué que vous vous plongez profondément dans les prévisions de revenus 
et l'adoption de l'IA dernièrement. Quel est le plus gros point de friction 
que votre équipe subit en ce moment ?
```

*Nombre de mots : 29. Une question, zéro pitch.*

*Résultat : James répond samedi matin : « Le plus gros problème est d'obtenir l'alignement entre les ventes et les finances sur les intrants de prévisions. »*

Dimanche 8h15 (matin d'affaires suivant, heure aléatoire), note vocale :

```
« Salut James, c'est [Votre nom] de [Entreprise]. Donc j'ai écouté ce que vous avez dit 
sur l'alignement ventes-finance sur les prévisions — c'est la #1 chose que nous voyons aussi, 
surtout à mesure que les entreprises augmentent en série B. Question rapide : 
avez-vous cartographié qui sont les parties prenantes spécifiques qui doivent convenir 
des intrants ? C'est généralement où le bouchon est. À bientôt ! »
```

*32 secondes, spécifique à son problème, conversationnel.*

*Résultat : 58% de chance de réponse (taux de note vocale).*

S'il répond (lundi) : 

```
J'aime que vous ayez dit que la finance pousse pour un aperçu de 60 jours. 
C'est la tension exacte que nous avons résolu avec [contexte client]. 
Cela vaut la peine d'un appel de 15 min pour comparer les notes — 
mercredi 10h ou jeudi 14h vous conviennent ?
```

*C'est demande de réunion, sûr parce qu'il s'est engagé 3+ fois.*

**Résultat de fin de mois :**
- 104 demandes de connexion envoyées (5 par jour, échelonnées)
- 31 acceptées (taux de 30%, au-dessus de la cible)
- 18 réponses du premier message (taux de 12%, à la cible)
