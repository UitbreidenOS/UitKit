# Habilidad de MLflow

## Cuándo activar
- Seguimiento de experimentos de aprendizaje automático (parámetros, métricas, artefactos)
- Registro de modelos en el registro de modelos de MLflow
- Comparación de ejecuciones entre búsquedas de hiperparámetros
- Servicio de un modelo registrado como punto de extremo REST
- Configuración del servidor de seguimiento de MLflow con backend PostgreSQL y almacén de artefactos S3
- Integración de MLflow con scikit-learn, PyTorch, TensorFlow o XGBoost autolog
- Implementación del ciclo de vida de gestión de modelos (preparación, producción, archivado)
