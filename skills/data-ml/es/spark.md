# Habilidad de Apache Spark

## Cuándo activar
- Procesamiento de conjuntos de datos demasiado grandes para caber en memoria (DataFrames de PySpark)

- Construcción de pipelines de Structured Streaming desde Kafka o S3
- Optimización de trabajos de Spark: particionamiento, caché, broadcast joins, manejo de sesgos
- Trabajo con Delta Lake (transacciones ACID, viaje en el tiempo, evolución de esquema)
- Migración de código de Pandas a PySpark para escalabilidad
