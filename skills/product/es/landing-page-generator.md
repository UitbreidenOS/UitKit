# Competencia Generador de Página de Destino

## Cuándo activar
- Construir una nueva página de destino de marketing o producto desde cero
- Convertir un resumen de diseño en código Next.js/React funcionando
- Crear una página de lanzamiento de producto, página de campaña o página de captura de clientes potenciales
- Construir una página de precios o secciones de comparación de características
- Generar variantes de prueba A/B para una página de destino

## Cuándo NO usar
- Aplicaciones web complejas (usar competencias de andamiaje de proyectos)
- Sitios de blog o contenido (usar enfoque basado en CMS)
- Paneles internos (audiencia incorrecta y patrones)

## Instrucciones

### Generación de página completa

```
Genere una página de destino para [producto].

Producto: [nombre y descripción de una línea]
Audiencia objetivo: [quién visita esta página]
CTA primario: [qué desea que hagan — registrarse / demostración / comprar / descargar]
Propuestas de valor clave: [3-5 cosas más importantes a comunicar]
Precios: [niveles o "contactar ventas"]
Tono: [profesional / startup / amigable / audaz / mínimo]
Stack: [Next.js 15 App Router / React + Vite]

Genere una página de destino completa con estas secciones:

1. Sección hero
2. Prueba social (logotipos o estadísticas)
3. Características / beneficios
4. Precios (si aplica)
5. Preguntas frecuentes
6. Sección CTA
7. Pie de página

SALIDA: Componente TSX completo con clases Tailwind CSS.
Objetivo: LCP < 1s, CLS < 0.1, responsivo mobile-first.
```

[Las secciones posteriores siguen el mismo patrón que la versión francesa con detalles técnicos en español]
