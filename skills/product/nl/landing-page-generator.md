# Competentie Landing Page Generator

## Wanneer activeren
- Een nieuwe marketing- of product-landing page helemaal opnieuw bouwen
- Een ontwerpontwerpschets converteren naar werkende Next.js/React-code
- Een productlancer-page, campagnepagina of lead-capture-pagina maken
- Een prijspagina of feature-vergelijkingssecties bouwen
- A/B-testvarianten voor een landing page genereren

## Wanneer niet gebruiken
- Complexe webapplicaties (projectscaffold-competenties gebruiken)
- Blog- of contentsites (een CMS-gebaseerde benadering gebruiken)
- Interne dashboards (verkeerd publiek en patronen)

## Instructies

### Volledige paginagenerering

```
Genereer een landing page voor [product].

Product: [naam en one-line beschrijving]
Doelgroep: [wie deze pagina bezoekt]
Primaire CTA: [wat u wilt dat zij doen — aanmelden / demo / koop / download]
Sleutelwaardestellingen: [3-5 belangrijkste dingen om mee te delen]
Prijsstelling: [lagen of "neem contact op met verkoop"]
Toon: [professioneel / startup / vriendelijk / stoutmoedig / minimaal]
Stack: [Next.js 15 App Router / React + Vite]

Genereer een volledige landing page met deze secties:

1. Heliosectie
2. Sociaal bewijs (logo's of statistieken)
3. Functies / voordelen
4. Prijsstelling (als van toepassing)
5. Veelgestelde vragen
6. CTA-sectie
7. Voettekst

UITVOER: Volledige TSX-component met Tailwind CSS-klassen.
Doel: LCP < 1s, CLS < 0.1, Mobiel-eerst responsief.
```
