# Kompetenz Landing Page Generator

## Wann aktivieren
- Eine neue Marketing- oder Produkt-Landing Page von Grund auf erstellen
- Ein Design-Brief in funktionierenden Next.js/React-Code konvertieren
- Eine Produktstart-Page, Kampagnen-Page oder Lead-Capture-Page erstellen
- Preis-Page oder Feature-Vergleichs-Abschnitte erstellen
- A/B-Test-Varianten für eine Landing Page generieren

## Wann nicht verwenden
- Komplexe Webanwendungen (verwenden Sie die Project-Scaffolding-Kompetenzen)
- Blog- oder Content-Sites (verwenden Sie einen CMS-basierten Ansatz)
- Interne Dashboards (falsches Publikum und Muster)

## Anweisungen

### Vollständige Seitengenerierung

```
Generieren Sie eine Landing Page für [Produkt].

Produkt: [Name und One-Line-Beschreibung]
Zielgruppe: [wer diese Seite besucht]
Primärer CTA: [was Sie möchten, dass sie tun — Anmeldung / Demo / Kauf / Download]
Wichtigste Value Props: [3-5 wichtigste Dinge zum Kommunizieren]
Preisgestaltung: [Niveaus oder „Sales kontaktieren"]
Ton: [professionell / Startup / freundlich / kühn / minimal]
Stack: [Next.js 15 App Router / React + Vite]

Generieren Sie eine vollständige Landing Page mit diesen Abschnitten:

1. Hero-Abschnitt
2. Sozialer Beweis (Logos oder Statistiken)
3. Funktionen / Vorteile
4. Preisgestaltung (falls zutreffend)
5. FAQ
6. CTA-Abschnitt
7. Footer

AUSGABE: Vollständige TSX-Komponente mit Tailwind CSS-Klassen.
Ziel: LCP < 1s, CLS < 0.1, Mobile-First-Responsive.
```
