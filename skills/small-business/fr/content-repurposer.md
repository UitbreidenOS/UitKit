# Compétence Content Repurposer

## Quand l'activer
- Vous avez un billet de blog, une transcription de podcast ou un article et voulez en tirer plus
- Créer une semaine de contenu social à partir d'une seule source
- Transformer une étude de cas client en plusieurs formats
- Réutiliser une longue vidéo en clips, sous-titres et newsletter
- Vous avez 30 minutes et avez besoin de contenu pour plusieurs canaux

## Quand ne PAS l'utiliser
- Créer un nouveau contenu à partir de zéro — décrivez plutôt ce que vous voulez
- Réutiliser le contenu des concurrents — risque juridique
- Contenu datant de plus de 2 ans — vérifier d'abord l'exactitude

## Instructions

### Le flux de travail un-vers-plusieurs

Donnez Claude votre contenu source et les résultats dont vous avez besoin :

```
Voici un billet de blog que j'ai écrit sur [sujet] : [collez contenu]

S'il vous plaît, créez :
1. 5 messages LinkedIn (professionnel, une idée par message)
2. 5 légendes Instagram avec suggestions de hashtag
3. Une version newsletter (300 mots, conversationnelle)
4. Un titre et une description Google Ads
5. Un script vidéo de 60 secondes
```

### Formats que Claude peut produire

| Format | Ce qu'il faut spécifier |
|---|---|
| Message LinkedIn | Ton (professionnel/personnel), longueur, inclure une question ? |
| Légende Instagram | Vibe (inspirant/éducatif/amusant), nombre de hashtags |
| Newsletter par email | Longueur, ligne d'objet, CTA |
| Thread Twitter/X | Nombre de tweets, style d'accroche |
| Message Facebook | Communauté-centré, conversationnel |
| Google Ads | Limites de caractères appliquées automatiquement |
| Description YouTube | SEO-focalisé, avec horodatages |
| Script vidéo | Durée, présentateur ou voix off |
| Notes de podcast | Résumé + points clés + horodatages |
| Broche Pinterest | Description + riche en mots-clés |

### Pour une entreprise de services

```
Je suis plombier. Voici une étude de cas d'un travail que j'ai fait la semaine dernière :
"Réparation d'une canalisation éclatée pour un propriétaire à 2h du matin un dimanche, sauvé leur cuisine d'une inondation"

Transformez ceci en :
- Une demande d'avis Google (à envoyer au client)
- Un message Facebook pour ma page commerciale
- Un message Google Business
- Un court email à ma liste de diffusion
```

### Pour une entreprise de produits

```
Voici ma description de nouveau produit : [collez]

Créer :
- 3 légendes Instagram différentes (éducatif, style témoignage, promotionnel)
- Options de ligne d'objet email (5 variations)
- Texte publicitaire pour Facebook (sensibilisation) et Google (intention de recherche)
- Une annonce de lancement de produit pour ma liste de diffusion
```

### Voix de marque cohérente

```
Voici un exemple de contenu que j'ai écrit avant qui sonne comme moi : [collez exemple]

Maintenant, réutilisez ceci [nouveau contenu] dans le même ton et style.
```

### Plan de contenu à partir d'une pièce

```
J'ai écrit cet article : [collez]

Créez un plan de contenu média social de 2 semaines en utilisant uniquement cet article comme source.
Mélangez les formats : conseils, devis, questions, coulisses.
Plateforme : LinkedIn + Instagram.
```

## Exemple

**Vous collez un article de blog de 600 mots sur « 5 raisons pour lesquelles les entreprises locales perdent des clients »**

**Claude produit :**
- 5 messages LinkedIn (un par raison, chacun avec un accroche + idée + question)
- Email : ligne d'objet « Commettez-vous ces 5 erreurs ? » + version newsletter
- 3 légendes Instagram avec angles différents
- Google Ads : 3 combinaisons titre + description
- Script 60 secondes : « Si vous êtes propriétaire d'une entreprise locale et que vous perdez des clients, voici ce qui se passe probablement... »
