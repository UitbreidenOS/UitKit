# Compétence Job Description

## Quand l'activer
- Rédiger une offre d'emploi pour un poste
- Examiner un tas de CV et identifier les meilleurs candidats
- Créer des questions d'entrevue pour un rôle spécifique
- Rédiger une lettre d'offre ou un email de rejet
- Définir ce qu'une nouvelle embauche doit vraiment faire au jour le jour

## Quand ne PAS l'utiliser
- Contrats d'emploi légaux — utilisez un avocat du travail
- Processus de redondance ou de licenciement — consultez un professionnel RH/juridique
- Documents de politique RH au niveau de l'entreprise

## Instructions

### Écrire une description de poste

```
Écrivez une description de poste pour : [titre de poste]

Mon entreprise : [type et brève description]
Emplacement : [ville / distant / hybride]
Temps plein / temps partiel : [lequel, et heures]
Fourchette salariale : $[X]–$[Y] ou [taux horaire] (si vous êtes disposé à partager)
Incontournables : [3-5 exigences non négociables]
Appréciables : [compétences ou expérience optionnelles]
Ce qu'ils vont réellement faire au jour le jour : [décrivez les tâches principales]
Qui ils travailleront avec : [à qui ils rendent des comptes, qui est sur l'équipe]
Ce qui rend le travail ici bon : [culture, avantages, opportunité de croissance]
Ton : [professionnel / casual / startup-y / chaleureux]
```

### Passez un tas de CV

```
J'embauche pour [titre du poste]. Les exigences clés sont : [liste 3-5 incontournables].
J'ai reçu [X] CV. Je vais coller chacun ci-dessous.
Pour chacun : les noter de 1-5, résumé de leur expérience pertinente en 2 phrases,
et dites-moi qui je dois prioriser pour les entrevues.

[coller les CV un par un]
```

### Créer des questions d'entrevue

```
J'interviewe des candidats pour [titre du poste].
Les choses clés que j'ai besoin d'évaluer : [liste ce qui compte le plus pour ce rôle]
Type d'entreprise : [décrivez]

Créer:
- 5 questions pour évaluer les compétences/expérience pertinentes
- 3 questions situationnelles (« dites-moi une fois où... »)
- 2 questions de culture/culture
- 1 question qui révélera s'il sont vraiment intéressés par le rôle

Aussi suggérer à quoi ressemble une « bonne réponse » pour chacun.
```

### Rédiger une lettre d'offre

```
Écrivez une lettre d'offre pour [nom], que j'embauche comme [rôle].
Date de début : [date]
Salaire/taux : $[X] [par heure/par année]
Heures : [temps plein/temps partiel, heures spécifiques le cas échéant]
Période de probation : [le cas échéant — par exemple 3 mois]
Conditions spécifiques : [période de probation, non-concurrence, etc.]
Nom de mon entreprise : [nom]
Ton : [chaud mais professionnel]
```

### Rédiger un email de rejet

```
Écrivez un email de rejet gentil mais clair pour un candidat qui a postulé pour [rôle].
Ils : [note brève — par exemple, avaient les compétences mais moins d'expérience qu'un autre candidat]
Je veux : [laisser la porte ouverte / la fermer poliment]
Gardez-le : sous 100 mots, authentique, pas robotique.
```

### Définir le rôle avant de publier

```
J'ai besoin d'embaucher quelqu'un mais je ne suis pas sûr exactement de quel rôle j'ai besoin.
Je souffre actuellement de : [décrivez les plus grands points de douleur]
Les choses que je veux enlever mon assiette : [liste]
Budget : $[X]/heure ou $[X]/année

Aidez-moi à chiffrer : quel type de rôle j'ai réellement besoin ?
Quel devrait être leur titre ? Que ne devraient-ils pas faire ?
```

## Exemple

**Vous dites :**
```
Je dirige une petite entreprise de comptabilité (3 personnes).
J'ai besoin d'embaucher un administrateur/coordinateur de client.
Ils gèreraient la planification, les courriels des clients, les documents de chasse, garder la base de données client ordonnée, répondre au téléphone.
Salaire : 45 000–55 000 $
Incontournables : organisé, bon communicateur, fiable, connaît Google Workspace
Pas obligatoire mais super : connaissance de la comptabilité
Ton : professionnel mais amical — nous sommes une petite équipe, pas d'entreprise
```

**Claude écrit une offre d'emploi complète** avec : titre clair (« Coordonnateur Client »), paragraphe d'introduction attrayant sur l'entreprise, aperçu de rôle en anglais courant, liste des responsabilités (8-10 points), exigences divisées en incontournables/appréciables, ce qu'ils aimeront du rôle, fourchette salariale et instruction d'application directe.
