# SOP Writer Skill

## Quand activer
- Vous devez documenter un processus pour qu'un nouvel employé puisse le suivre
- Rédiger un manuel d'employé ou un manuel d'opérations
- Créer un script de service client ou un guide de réponse FAQ
- Documenter votre routine d'ouverture/fermeture, liste de contrôle de nettoyage ou liste de contrôle de qualité
- Préparer votre entreprise à fonctionner sans vous

## Quand ne PAS utiliser
- Les procédures requises par la loi ou la conformité — faire examiner par un avocat
- Les protocoles médicaux ou de sécurité — exiger un examen professionnel
- Les accords de franchise — utiliser les modèles de votre franchiseur

## Instructions

### SOP de base à partir d'une description

```
Écrire un SOP pour : [décrire le processus en anglais simple]

Type d'entreprise : [votre entreprise]
Qui suit cela : [rôle/position]
À quelle fréquence : [quotidiennement/hebdomadairement/quand X se produit]
Quels outils/systèmes ils utilisent : [lister]
Toute note importante ou chose qui va souvent mal : [optionnel]
```

### Procédure d'ouverture/fermeture

```
Écrire une liste de contrôle de procédure d'ouverture pour mon [restaurant/magasin/salon/bureau].
Heure d'ouverture : [heure]
Étapes que vous faites actuellement (dans un ordre approximatif) : [décrire votre routine]
Personnel impliqué : [rôles]
Format : liste numérotée avec temps estimé pour chaque étape
```

### Script de service client

```
Écrire un script de service client pour mon [type d'entreprise] qui couvre :
1. Accueillir un nouveau client
2. Gérer la question la plus courante : [c'est quoi]
3. Gérer une plainte concernant [problème courant]
4. Upselling/recommandation [produit/service]
5. Fermer l'interaction

Mon ton : [friendly/professional/upbeat]
Les valeurs de mon entreprise : [1-2 choses clés]
```

### Liste de contrôle d'intégration du nouvel employé

```
Créer une liste de contrôle d'intégration de première semaine pour un nouveau [rôle] chez mon [type d'entreprise].

Ce qu'ils doivent savoir :
- Nos outils principaux : [lister]
- Leurs responsabilités principales : [décrire]
- À qui ils rapportent : [rôle]
- La chose la plus importante à bien faire la semaine 1 : [décrire]

Format : liste de contrôle jour après jour pour les jours 1-5
```

### Documentation de processus à partir d'une entrevue

```
Je vais décrire comment je fais [processus] et je veux que vous la transformiez en un SOP clair et étape par étape que quelqu'un d'autre pourrait suivre.

Voici comment je le fais : [décrire avec vos propres mots, aussi désordonné que vous voulez]
```

### Liste de contrôle de contrôle de qualité

```
Créer une liste de contrôle de contrôle de qualité pour [produit/service] avant qu'il ne soit livré à un client.

À quoi « bon » ressemble : [décrire vos normes]
Les problèmes courants que nous attrapons : [décrire]
Qui vérifie : [rôle]
Format : liste de contrôle avec critères de réussite/échec
```

## Exemple

**Vous dites :**
```
Écrire un SOP pour gérer une plainte de client dans mon salon de coiffure.
Plaintes courantes : temps d'attente, couleur qui ne correspond pas aux attentes, surprise de prix.
Je veux que le personnel reste calme, offre des solutions, ne jamais argumenter.
Nous pouvons offrir : refaire le service, réduction à la prochaine visite, petit remboursement pour problèmes mineurs.
Ton : empathique, professionnel.
```

**Claude produit :**

**SOP: Gérer les plaintes de clients — Salon de coiffure**

**Objectif :** Résoudre les plaintes professionnellement, conserver le client et protéger la réputation du salon.

**Étape 1 — Écouter sans interrompre**
- Faire contact oculaire, hocher la tête, la laisser finir complètement
- Ne pas défendre ou expliquer encore
- Dire : « Je suis vraiment désolée d'entendre cela — merci de me l'avoir dit »

**Étape 2 — Reconnaître le sentiment**
- « Je comprends complètement pourquoi c'est frustrant »
- Ne pas dire : « Ce n'est pas notre politique » / « Mais nous vous l'avions dit... » / « Cela n'arrive jamais »

**Étape 3 — Identifier le problème**
- Demander : « Pouvez-vous m'aider à comprendre exactement ce qui s'est passé? »
- Clarifier : s'agit-il du résultat, du prix, de l'attente ou de l'expérience?

**Étape 4 — Offrir une résolution**
- Problème de couleur/résultat : offrir un ajustement gratuit dans les 7 jours
- Surprise de prix : revoir la conversation de tarification ; si notre erreur, honorer le prix inférieur
- Temps d'attente : s'excuser sincèrement + offrir une petite réduction sur le service d'aujourd'hui

**Étape 5 — Confirmer la résolution**
- « Cela vous semble-t-il une solution juste? »
- Documenter : noter le nom du client, la date, le type de plainte, la résolution dans le système de réservation

**Étape 6 — Suivi (optionnel mais puissant)**
- Pour les plaintes graves : le responsable appelle ou envoie un SMS dans les 24 heures

**Ne jamais :** argumenter, impliquer d'autres clients, offrir des remboursements sans approbation du responsable (au-dessus de $50)
