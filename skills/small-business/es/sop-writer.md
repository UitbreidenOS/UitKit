# Skill SOP Writer

## Cuándo activar
- Necesitas documentar un proceso para que un nuevo empleado pueda seguirlo
- Escribir un manual de empleados u operaciones
- Crear un script de servicio al cliente o guía de respuesta a preguntas frecuentes
- Documentar tu rutina de apertura/cierre, lista de verificación de limpieza, o lista de verificación de calidad
- Preparar tu negocio para ejecutarse sin ti

## Cuándo NO usar
- Procedimientos requeridos por ley o cumplimiento — haz que un abogado revise
- Protocolos médicos o de seguridad — requieren revisión profesional
- Acuerdos de franquicia — usar plantillas de tu franquiciador

## Instrucciones

### SOP básico desde descripción

```
Escribir un SOP para: [describir el proceso en inglés plano]

Tipo de negocio: [tu negocio]
Quién lo sigue: [rol/posición]
Con qué frecuencia: [diario/semanal/cuando X sucede]
Qué herramientas/sistemas usan: [listar]
Notas importantes o cosas que a menudo salen mal: [opcional]
```

### Procedimiento de apertura/cierre

```
Escribir un procedimiento de apertura lista de verificación para mi [restaurante/tienda/salón/oficina].
Hora de apertura: [hora]
Pasos que actualmente hago (en orden aproximado): [describir tu rutina]
Personal involucrado: [roles]
Formato: lista de verificación numerada con tiempo estimado para cada paso
```

### Script de servicio al cliente

```
Escribir un script de servicio al cliente para mi [tipo de negocio] que cubra:
1. Saludar a un nuevo cliente
2. Manejar la pregunta más común: [cuál es]
3. Lidiar con una queja sobre [problema común]
4. Venta adicional/recomendación [producto/servicio]
5. Cerrar la interacción

Mi tono: [amistoso/profesional/alegre]
Valores de mi negocio: [1-2 cosas clave]
```

### Lista de verificación de incorporación de nuevo empleado

```
Crear una lista de verificación de incorporación de la primera semana para un nuevo [rol] en mi [tipo de negocio].

Lo que necesitan saber:
- Nuestras herramientas principales: [listar]
- Sus responsabilidades principales: [describir]
- A quién reportan: [rol]
- Lo más importante en la semana 1: [describir]

Formato: lista de verificación día por día para días 1-5
```

### Documentación de proceso desde entrevista

```
Voy a describir cómo hago [proceso] y quiero que lo conviertas en un SOP claro, 
paso a paso que alguien más podría seguir.

Aquí está cómo lo hago: [describir con tus propias palabras, tan desordenado como quieras]
```

### Lista de verificación de control de calidad

```
Crear una lista de verificación de control de calidad para [producto/servicio] antes de que se entregue a un cliente.

Cómo se ve "bien": [describir tus estándares]
Problemas comunes que detectamos: [describir]
Quién verifica: [rol]
Formato: lista de verificación con criterios de aprobación/rechazo
```

## Ejemplo

**Tú dices:**
```
Escribir un SOP para manejar una queja de cliente en mi salón de peluquería.
Quejas comunes: tiempo de espera, color no coincide con expectativas, sorpresa de precio.
Quiero que el personal mantenga la calma, ofrezca soluciones, nunca discuta.
Podemos ofrecer: rehacer el servicio, descuento en próxima visita, pequeño reembolso para problemas menores.
Tono: empático, profesional.
```

**Claude produce:**

**SOP: Manejar Quejas de Clientes — Salón de Peluquería**

**Propósito:** Resolver quejas profesionalmente, retener al cliente y proteger la reputación del salón.

**Paso 1 — Escuchar sin interrumpir**
- Hacer contacto visual, asentir, dejar que terminen completamente
- No defender ni explicar aún
- Decir: "Lamento mucho escuchar eso — gracias por decírmelo"

**Paso 2 — Reconocer el sentimiento**
- "Completamente entiendo por qué eso es frustrante"
- No decir: "Esa no es nuestra política" / "Pero te dijimos..." / "Eso nunca sucede"

**Paso 3 — Identificar el problema**
- Preguntar: "¿Puedes ayudarme a entender exactamente qué sucedió?"
- Aclarar: ¿es sobre el resultado, el precio, la espera o la experiencia?

**Paso 4 — Ofrecer una solución**
- Problema de color/resultado: ofrecer ajuste sin cargo dentro de 7 días
- Sorpresa de precio: revisar la conversación de precios; si es nuestro error, honrar el precio más bajo
- Tiempo de espera: disculparse sinceramente + ofrecer pequeño descuento en el servicio de hoy

**Paso 5 — Confirmar solución**
- "¿Se siente eso como una solución justa?"
- Documentar: anotar nombre del cliente, fecha, tipo de queja, resolución en el sistema de reservas

**Paso 6 — Seguimiento (opcional pero poderoso)**
- Para quejas serias: gerente llama o envía mensaje de texto dentro de 24 horas

**Nunca:** discutir, involucrar a otros clientes, ofrecer reembolsos sin aprobación del gerente (por encima de $50)
