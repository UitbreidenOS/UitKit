# Habilidad Content Repurposer

## Cuándo activar
- Tiene una publicación de blog, transcripción de podcast o artículo y desea obtener más de él
- Crear una semana de contenido social a partir de una única fuente
- Transformar un estudio de caso de cliente en múltiples formatos
- Reutilizar un video largo en clips, subtítulos y boletín
- Tiene 30 minutos y necesita contenido para múltiples canales

## Cuándo NO usar
- Crear contenido completamente nuevo desde cero — describa lo que desea en su lugar
- Reutilizar contenido competidor — riesgo legal
- Contenido con más de 2 años — verifique la precisión primero

## Instrucciones

### El flujo de trabajo uno a muchos

Proporcione a Claude su contenido fuente y los resultados que necesita:

```
Aquí hay una publicación de blog que escribí sobre [tema]: [pegue contenido]

Por favor, cree:
1. 5 publicaciones de LinkedIn (profesional, una idea por publicación)
2. 5 subtítulos de Instagram con sugerencias de hashtag
3. Una versión de boletín por correo electrónico (300 palabras, conversacional)
4. Titular de Google Ads y descripción
5. Un script de video de 60 segundos
```

### Formatos que Claude puede producir

| Formato | Qué especificar |
|---|---|
| Publicación LinkedIn | Tono (profesional/personal), longitud, ¿incluir una pregunta? |
| Subtítulo Instagram | Vibe (inspirador/educativo/divertido), cantidad de hashtags |
| Boletín por correo electrónico | Longitud, línea de asunto, CTA |
| Hilo Twitter/X | Cantidad de tweets, estilo de gancho |
| Publicación Facebook | Orientado a la comunidad, conversacional |
| Google Ads | Límites de caracteres aplicados automáticamente |
| Descripción de YouTube | Enfocado en SEO, con marcas de tiempo |
| Script de video | Duración, presentador o voz en off |
| Notas del programa de podcast | Resumen + puntos clave + marcas de tiempo |
| Pin de Pinterest | Descripción + rico en palabras clave |

### Para un negocio de servicios

```
Soy fontanero. Aquí hay un estudio de caso sobre un trabajo que hice la semana pasada:
"Reparé una tubería reventada para un propietario a las 2 a.m. el domingo, salvé su cocina de una inundación"

Convierta esto en:
- Una solicitud de reseña de Google (para enviar al cliente)
- Una publicación de Facebook para mi página de negocio
- Una publicación de Google Business
- Un correo corto a mi lista de boletín
```

### Para un negocio de productos

```
Aquí está mi nueva descripción de producto: [pegue]

Crear:
- 3 subtítulos de Instagram diferentes (educativo, estilo de testimonio, promocional)
- Opciones de línea de asunto de correo (5 variaciones)
- Texto de anuncio para Facebook (conciencia) y Google (intención de búsqueda)
- Un anuncio de lanzamiento de producto para mi lista de correo
```

### Voz de marca consistente

```
Aquí hay un ejemplo de contenido que escribí anteriormente que suena como yo: [pegue ejemplo]

Ahora reutilice esto [nuevo contenido] con la misma voz y estilo.
```

### Plan de contenido de una pieza

```
Escribí este artículo: [pegue]

Crear un plan de contenido de redes sociales de 2 semanas usando solo este artículo como fuente.
Formatos de mezcla: consejos, citas, preguntas, perspectivas entre bastidores.
Plataforma: LinkedIn + Instagram.
```

## Ejemplo

**Pegue una publicación de blog de 600 palabras sobre "5 razones por las que los negocios locales pierden clientes"**

**Claude produce:**
- 5 publicaciones de LinkedIn (una por razón, cada una con gancho + perspectiva + pregunta)
- Correo: línea de asunto "¿Estás cometiendo estos 5 errores?" + versión de boletín
- 3 subtítulos de Instagram con diferentes ángulos
- Google Ads: 3 combinaciones de titular + descripción
- Script de 60 segundos: "Si es propietario de un negocio local y está perdiendo clientes, esto es lo que probablemente está sucediendo..."
