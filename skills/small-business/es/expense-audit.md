# Habilidad Expense Audit

## Cuándo activar
- Desea encontrar gastos que puede reducir o renegociar
- Sus costos se sienten altos pero no está seguro de dónde va el dinero
- Realizar una revisión anual de todos los suscripciones comerciales
- Preparándose para una reunión con su contador
- Ha crecido rápidamente y sospecha que algunos gastos ya no son necesarios

## Cuándo NO usar
- Auditar gastos de empleados individuales (proceso diferente)
- Revisiones de cumplimiento fiscal — use un contador autorizado
- Investigación de fraude — consulte un profesional

## Instrucciones

### Auditoría de suscripción y software

```
Aquí están todas las suscripciones/herramientas de software que pago:
[pegue su lista — nombre, costo mensual, qué hace, frecuencia de uso]

¿Cuál debería:
1. Cancelar (no se usa o duplicado)
2. Reducir (pagar por funciones que no uso)
3. Consolidar (dos herramientas hacen lo mismo)
4. Renegociar (pagando mucho, existen alternativas)
```

### Revisión de gastos de estado de cuenta bancario

```
Aquí está mi estado de cuenta bancario comercial para [mes] [pegue o describa categorías]:

Categorice todos los gastos, identifique:
1. Cualquier cargo recurrente que podría haber olvidado
2. Artículos inusuales o puntuales que necesitan revisión
3. Cualquier monto que parezca más alto de lo esperado
4. Posibles clasificaciones incorrectas para propósitos fiscales
```

### La lista de "¿todavía lo necesito?"

```
Quiero revisar cada gasto comercial mayor a $100/mes.
Aquí está mi lista: [pegue]

Para cada uno, hágame 3 preguntas para ayudarme a decidir si aún lo necesito,
luego dame una recomendación: mantener / cancelar / revisar.
```

### Encontrar ganancias rápidas

```
Mis gastos comerciales mensuales totales son $[X].
Mis ingresos son $[Y].
Mi margen de ganancia se siente delgado.

¿Qué categorías de gastos debo revisar primero para ahorros rápidos?
¿Cuál es un benchmark razonable para [tipo de negocio] a mi nivel de ingresos?
```

### Reinicio de suscripción anual

```
Quiero auditar todo lo que mi negocio paga anualmente.
Mis cuentas incluyen: QuickBooks, Stripe, Slack, Google Workspace, [otros]

Ayúdeme a:
1. Listar cada herramienta y su costo aproximado
2. Identificar alternativas gratuitas para cualquier cosa que use mínimamente
3. Encontrar cualquier cosa que esté pagando doble
4. Priorizar qué cancelaría primero
```

### Antes de una fase de crecimiento

```
Estoy a punto de contratar mi primer empleado (costo: ~$[X]/mes).
Mis gastos actuales son $[Y]/mes, ingresos $[Z]/mes.

¿Qué gastos debería considerar cortar o reducir para dejar espacio para esta contratación
sin dañar las operaciones?
```

## Ejemplo

**Pegue sus gastos del mes pasado:**
```
Shopify: $79
Canva Pro: $17
Adobe CC: $55
Mailchimp: $35
ConvertKit: $29 (prueba)
Zoom: $15
Google Meet (gratis pero pagar Google Workspace): $12
Slack: $0 (nivel gratuito)
Airtable: $24
Notion: $16
ClickUp: $10
Dropbox: $12
Google Drive (incluido en Workspace)
QuickBooks: $30
FreshBooks: $17 (cuenta antigua, no usada)
```

**Claude señala:**
- **Cancelar inmediatamente:** FreshBooks ($17) — no usado, tengo QuickBooks
- **Cancelar inmediatamente:** ConvertKit ($29) — en prueba, ya pago Mailchimp
- **Consolidar:** Airtable + Notion + ClickUp — 3 herramientas de proyecto, elegir una (~$30-34 ahorros)
- **Revisar:** Dropbox ($12) — ya tiene Google Drive incluido en Workspace
- **Revisar:** Adobe CC ($55) — si solo usa Photoshop, podría usar Canva Pro en su lugar
- **Ahorros inmediatos totales: $46-80/mes**
