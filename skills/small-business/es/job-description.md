# Habilidad Job Description

## Cuándo activar
- Redactar una publicación de oferta de empleo para cualquier puesto
- Revisar un conjunto de currículos e identificar candidatos principales
- Crear preguntas de entrevista para un puesto específico
- Redactar una carta de oferta o un correo de rechazo
- Definir lo que una nueva contratación realmente debe hacer día a día

## Cuándo NO usar
- Contratos laborales legales — use un abogado de empleo
- Procesos de redundancia o terminación — consulte HR/legal
- Documentos de política de RH a nivel empresarial

## Instrucciones

### Redacte una descripción de puesto

```
Redacte una descripción de puesto para: [título de puesto]

Mi negocio: [tipo y descripción breve]
Ubicación: [ciudad / remoto / híbrido]
Tiempo completo / tiempo parcial: [cuál, y horas]
Rango salarial: $[X]–$[Y] o [tarifa por hora] (si está dispuesto a compartir)
Imprescindibles: [3-5 requisitos innegociables]
Seria agradable tener: [habilidades o experiencia opcionales]
Lo que realmente harán día a día: [describa las tareas principales]
Con quién trabajarán: [a quién reportan, quién está en el equipo]
Qué hace que trabajar aquí sea bueno: [cultura, beneficios, oportunidad de crecimiento]
Tono: [profesional / casual / startup-y / cálido]
```

### Revise un conjunto de currículos

```
Estoy contratando para [título de puesto]. Los requisitos clave son: [enumere 3-5 imprescindibles].
Recibí [X] currículos. Pegaré cada uno a continuación.
Para cada uno: califique 1-5, resuma su experiencia relevante en 2 oraciones,
y dígame a quién debería priorizar para entrevistas.

[pegue cada currículum uno por uno]
```

### Cree preguntas de entrevista

```
Estoy entrevistando candidatos para [título de puesto].
Cosas clave que necesito evaluar: [enumere lo más importante para este puesto]
Tipo de negocio: [describa]

Crear:
- 5 preguntas para evaluar habilidades/experiencia relevantes
- 3 preguntas situacionales (« cuéntame sobre un momento en que... »)
- 2 preguntas de ajuste de cultura/valores
- 1 pregunta que revelaré si están realmente interesados en el puesto

Sugiera también cómo se ve una « buena respuesta » para cada una.
```

### Redacte una carta de oferta

```
Redacte una carta de oferta para [nombre], a quien estoy contratando como [puesto].
Fecha de inicio: [fecha]
Salario/tarifa: $[X] [por hora/por año]
Horas: [tiempo completo/tiempo parcial, horas específicas si aplica]
Período de prueba: [si aplica — p. ej. 3 meses]
Términos específicos: [período de prueba, no competencia, etc.]
Nombre de mi negocio: [nombre]
Tono: [cálido pero profesional]
```

### Redacte un correo de rechazo

```
Redacte un correo de rechazo amable pero claro para un candidato que solicitó [puesto].
Ellos: [nota breve — p. ej. tenían las habilidades pero menos experiencia que otro candidato]
Quiero: [dejar la puerta abierta / cerrarla cortésmente]
Manténgalo: menos de 100 palabras, auténtico, no robótico.
```

### Defina el puesto antes de publicar

```
Necesito contratar a alguien pero no estoy seguro de qué puesto necesito exactamente.
Actualmente estoy teniendo dificultades con: [describa los mayores puntos débiles]
Cosas que quiero sacar de mi plato: [lista]
Presupuesto: $[X]/hora o $[X]/año

Ayúdeme a descubrir: ¿qué tipo de puesto realmente necesito?
¿Cuál debería ser su título? ¿Qué no deberían hacer?
```

## Ejemplo

**Usted dice:**
```
Dirijo una pequeña empresa de contabilidad (3 personas).
Necesito contratar un administrador/coordinador de cliente.
Manejarían programación, correos de clientes, documentos de persecución, mantener ordenada la base de datos de clientes, responder el teléfono.
Salario: $45.000–$55.000
Imprescindibles: organizado, buen comunicador, confiable, conoce Google Workspace
No requerido pero excelente: conocimiento de contabilidad
Tono: profesional pero amable — somos un equipo pequeño, no corporativo
```

**Claude redacta una publicación de trabajo completa** con: título claro (« Coordinador de Cliente »), párrafo introductorio convincente sobre la empresa, descripción general del puesto en inglés claro, lista de responsabilidades (8-10 puntos), requisitos divididos en imprescindibles/agradables, lo que les encantará del puesto, rango salarial e instrucciones directas de solicitud.
