# Habilidad Review Response

## Cuándo activar
- Tiene nuevas reseñas de Google o Yelp que necesitan respuestas
- Una reseña negativa necesita una respuesta tranquila y profesional
- Desea responder a cada reseña pero no tiene tiempo de escribir desde cero
- Construir una plantilla de respuesta de reseña consistente para su equipo

## Cuándo NO usar
- Reseñas que implican amenazas legales o disputas en curso — consulte un abogado
- Reseñas que cree que son falsas — repórtelas a la plataforma primero
- Responder con ira o defensivamente — aléjese y regrese

## Instrucciones

### Responder a una reseña positiva

```
Responda a esta reseña de Google de 5 estrellas para mi [tipo de negocio]:
Reseña: "[pegue texto de reseña]"

Mi nombre de negocio: [nombre]
Tono: [cálido y personal / profesional / divertido]
Incluir: agradecerles, mencionar algo específico que dijeron, invitarlos a regresar
```

### Responder a una reseña negativa

```
Recibí esta reseña de 1 estrella: "[pegue reseña]"

Mi negocio: [tipo]
Lo que realmente pasó (su lado): [breve explicación]
Mi objetivo: reconocer su experiencia, explicar sin ser defensivo, ofrecer resolución

Escriba una respuesta que:
- No discuta ni culpe al cliente
- Reconozca sus sentimientos
- Explique brevemente nuestra perspectiva
- Ofrezca resolver sin conexión (teléfono/correo)
- Se mantenga bajo 150 palabras
```

### Respuestas de reseña a granel

```
Tengo 12 reseñas que necesitan respuestas. Aquí están con calificaciones:
1. ⭐⭐⭐⭐⭐ - "[texto]"
2. ⭐⭐⭐ - "[texto]"
3. ⭐ - "[texto]"
...

Mi negocio: [tipo]. Tono: [describa].
Escriba respuestas para todas. Varíe el lenguaje para que no suenen todas idénticas.
```

### Manejo de situaciones específicas

**Reseña falsa o injusta:**
```
Recibí esta reseña pero creo que la persona nunca visitó nuestro negocio:
"[texto de reseña]"

Escriba una respuesta profesional y no acusatoria que señale con elegancia que no tenemos registro
de esta visita e invítelos a contactarnos directamente para aclarar.
```

**Queja sobre algo que ya ha resuelto:**
```
Esta reseña de 2 estrellas menciona servicio lento. Desde entonces hemos contratado personal adicional
y hemos reducido los tiempos de espera. Escriba una respuesta que:
- Los agradezca por la retroalimentación
- Reconozca que fue un problema
- Mencione que lo hemos resuelto específicamente
- Los invite a intentarlo de nuevo
```

### Plantillas de respuesta de reseña para su equipo

```
Cree 5 plantillas de respuesta diferentes para reseñas de 5 estrellas para mi [restaurante/salón/tienda].
Cada una debe sonar natural y diferente. Incluya espacios para el nombre del reseñador
y algo específico que mencionaron.
```

## Ejemplo

**La reseña:**
```
★★★ "La comida estaba bien pero el servicio fue muy lento. Esperamos 40 minutos nuestro plato principal.
El bistec fue bueno sin embargo. Podría intentarlo nuevamente en una noche más tranquila."
```

**Claude responde:**
```
Gracias por venir y tomarse el tiempo para compartir su experiencia, [Nombre].

Estamos realmente felices de que el bistec haya hecho la diferencia — nuestro equipo de cocina está realmente orgulloso de eso.
Tiene toda la razón de que esperara demasiado su plato principal, y lamentamos
que eso haya arruinado la noche. Estamos mirando de cerca nuestro ritmo en noches ocupadas
y estamos haciendo cambios activamente.

Nos encantaría tener la oportunidad de mostrarle una visita mucho más fluida.
Si le gustaría volver, no dude en mencionar esta reseña y nos encargaremos de usted.
```
