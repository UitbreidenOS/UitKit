# Review Response Skill

## Wann aktivieren
- Sie haben neue Google- oder Yelp-Bewertungen, die Antworten benötigen
- Eine negative Bewertung benötigt eine ruhige, professionelle Antwort
- Sie möchten auf jede Bewertung antworten, aber haben keine Zeit, von Grund auf zu schreiben
- Aufbau einer konsistenten Bewertungsantwort-Vorlage für Ihr Team

## Wann NICHT verwenden
- Bewertungen mit rechtlichen Drohungen oder laufenden Streitigkeiten — konsultieren Sie einen Anwalt
- Bewertungen, von denen Sie glauben, dass sie gefälscht sind — melden Sie sie der Plattform zuerst
- Wütend oder defensiv antworten — treten Sie zurück und kommen Sie zurück

## Anleitung

### Auf eine positive Bewertung antworten

```
Antworten Sie auf diese 5-Sterne-Google-Bewertung für mein [Geschäftstyp]:
Bewertung: "[fügen Sie Bewertungstext ein]"

Mein Geschäftsname: [Name]
Ton: [warm und persönlich / professionell / lustig]
Einschließen: ihnen danken, etwas Spezifisches erwähnen, das sie sagten, sie zur Rückkehr einladen
```

### Auf eine negative Bewertung antworten

```
Ich habe diese 1-Sterne-Bewertung erhalten: "[fügen Sie Bewertung ein]"

Mein Unternehmen: [Typ]
Was tatsächlich passiert ist (Ihre Seite): [kurze Erklärung]
Mein Ziel: ihre Erfahrung anerkennen, erklären ohne defensiv zu sein, Lösung anbieten

Verfassen Sie eine Antwort, die:
- Nicht argumentiert oder den Kunden beschuldigt
- Ihre Gefühle anerkennt
- Kurz unsere Perspektive erklärt
- Anbietet offline zu lösen (Telefon/Email)
- Unter 150 Wörtern bleibt
```

### Massen-Bewertungsantworten

```
Ich habe 12 Bewertungen, die Antworten benötigen. Hier sind sie mit Sternbewertungen:
1. ⭐⭐⭐⭐⭐ - "[Text]"
2. ⭐⭐⭐ - "[Text]"
3. ⭐ - "[Text]"
...

Mein Unternehmen: [Typ]. Ton: [beschreiben].
Schreiben Sie Antworten für alle. Variieren Sie die Sprache, damit sie nicht alle identisch klingen.
```

### Spezifische Situationen handhaben

**Falsche oder unfaire Bewertung:**
```
Ich habe diese Bewertung erhalten, aber ich glaube, die Person hat unser Unternehmen nie besucht:
"[Bewertungstext]"

Schreiben Sie eine professionelle, nicht anklagende Antwort, die sanft bemerkt, dass wir keine Aufzeichnung
dieses Besuchs haben und sie einlade, uns direkt zu kontaktieren, um es zu klären.
```

**Beschwerde über etwas, das Sie bereits behoben haben:**
```
Diese 2-Sterne-Bewertung erwähnt langsamen Service. Wir haben seitdem zusätzliches Personal eingestellt
und die Wartezeiten reduziert. Schreiben Sie eine Antwort, die:
- Ihnen für das Feedback dankt
- Anerkennt, dass es ein Problem war
- Erwähnt, dass wir es speziell behoben haben
- Sie einlade, uns erneut zu versuchen
```

### Bewertungsantwort-Vorlagen für Ihr Team

```
Erstellen Sie 5 verschiedene Antwortvorlagen für 5-Sterne-Bewertungen für mein [Restaurant/Salon/Shop].
Jede sollte natürlich und unterschiedlich klingen. Fügen Sie Platzhalter für den Namen des Rezensenten
und etwas Spezifisches ein, das sie erwähnten.
```

## Beispiel

**Die Bewertung:**
```
★★★ "Das Essen war okay, aber der Service war wirklich langsam. Wir mussten 40 Minuten auf unseren Hauptgang warten.
Das Steak war aber gut. Könnten an einem ruhigeren Abend wiederkommen."
```

**Claude antwortet:**
```
Danke, dass Sie vorbeikamen und sich Zeit nahmen, Ihre Erfahrung zu teilen, [Name].

Wir sind wirklich froh, dass das Steak den Unterschied machte — unser Küchenpersonal ist wirklich stolz darauf.
Sie haben absolut recht, dass der Wartezeit auf Ihren Hauptgang zu lange war, und wir entschuldigen uns,
dass dies den Abend getrübt hat. Wir schauen genau auf unser Tempo an beschäftigen Abenden
und treffen aktiv Änderungen.

Wir würden Ihnen gerne die Gelegenheit zeigen, einen viel reibungsloser Besuch zu haben.
Wenn Sie wiederkommen möchten, können Sie gerne diese Bewertung erwähnen und wir kümmern uns um Sie.
```
