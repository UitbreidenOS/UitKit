# Customer Inquiry Skill

## Wann aktivieren
- Schreiben von Antworten auf häufige Kundenfragen
- Erstellung einer FAQ für Ihre Website oder Google Business-Profil
- Einrichtung von After-Hours-Auto-Reply-Nachrichten
- Antwort auf Anfragen per E-Mail, DM oder Kontaktformular
- Schulung eines neuen Mitarbeiters zur Kundenbetreuung

## Wann NICHT verwenden
- Komplexe Streitigkeiten oder Beschwerden (verwenden Sie die Review-Response-Fähigkeit)
- Rechtliche oder formelle Kundenkommunikation
- Automatisiertes Live-Chat-Setup (benötigt einen Entwickler oder Chat-Tool)

## Anleitung

### Bauen Sie Ihre FAQ auf

```
Helfen Sie mir, eine FAQ für mein [Geschäftstyp] zu schreiben.
Die häufigsten Fragen, die ich bekomme:
1. [Frage]
2. [Frage]
3. [Frage]
[fügen Sie so viele hinzu, wie Sie haben]

Meine Antworten (grob): [beschreiben Sie, was Sie sagen würden]
Format: saubere Q&A für meine Website
Ton: [freundlich/professionell/beiläufig]
```

### Anfrageantworten schreiben

```
Ein Kunde fragte: "[fügen Sie ihre Nachricht oder Frage ein]"

Mein Unternehmen: [Typ]
Die Antwort: [beschreiben Sie auf Englisch]
Ton: [freundlich/professionell]

Schreiben Sie eine Antwort, die ich direkt versenden kann.
```

### After-Hours-Auto-Reply

```
Schreiben Sie eine After-Hours-Auto-Reply für mein [Geschäftstyp].
Meine Geschäftszeiten: [Öffnungszeiten auflisten]
Wie Kunden buchen/kontaktieren können: [beschreiben — Online-Buchungslink, Anruf, E-Mail]
Versprechener Reaktionszeitpunkt: [ich werde Ihnen innerhalb von X Stunden antworten]
Ton: [warm/professionell]
```

### Nachrichtenfluss für Terminbuchung

```
Erstellen Sie einen Nachrichtenfluss für Terminbuchungen bei meinem [Salon/Klinik/Studio/Service].
Wenn jemand anfragt, muss ich:
1. Verfügbarkeit bestätigen
2. Erklären Sie, was zu erwarten ist
3. Ihre Details sammeln
4. Bestätigung versenden

Schreiben Sie Vorlagen für jeden Schritt.
```

### Umgang mit "Wie viel kostet es?" Anfragen

```
Mich fragen ständig nach Preisen, aber meine Preise variieren.
Mein Service: [beschreiben]
Warum sich die Preise unterscheiden: [erklären — es hängt von X, Y, Z ab]

Schreiben Sie eine Antwort, die:
- Nicht einen Pauschalpreis gibt, der für manche Menschen falsch ist
- Die richtigen Qualifizierungsfragen stellt
- Hilfreich klingt, nicht ausweichend
- Mit einem einfachen nächsten Schritt endet (eine Beratung buchen / schicken Sie mir mehr Details)
```

### Häufige Einwand-Antworten

```
Schreiben Sie Antworten für diese häufigen Einwände bei meinem [Geschäftstyp]:
1. „Ihre Preise sind zu hoch"
2. „Ich muss darüber nachdenken"
3. „Ich habe jemanden Billigeres gefunden"
4. „Ich werde es einfach selbst tun"

Ton: selbstbewusst aber nicht aufdringlich. Halten Sie jede Antwort unter 100 Wörtern.
```

## Beispiel

**Sie sagen:**
```
Ich bin Personal Trainer. Schreiben Sie mir eine FAQ für meine Website.
Die häufigsten Fragen:
- Wie viel kostet es?
- Bieten Sie Online-Training an?
- Wie lange, bis ich Ergebnisse sehe?
- Was ist, wenn ich ein absoluter Anfänger bin?
- Brauche ich meine eigene Ausrüstung?
- Wie beginne ich?
Ton: motivierend und erreichbar, nicht unternehmensgerecht.
```

**Claude schreibt:**

**Häufig Gestellte Fragen**

**Wie viel kostet es?**
Sitzungen beginnen ab $[X] für 1-zu-1 Coaching und $[X] für Online-Programme. Ich biete Pakete an, die die Kosten pro Sitzung reduzieren — sprechen wir und finden das beste für Ihr Budget und Ihre Ziele.

**Bieten Sie Online-Training an?**
Ja! Ich arbeite mit Kunden bundesweit über Videoanruf. Sie erhalten die gleiche personalisierte Aufmerksamkeit wie persönlich, plus die Flexibilität, überall zu trainieren.

**Wie lange, bis ich Ergebnisse sehe?**
Die meisten Klienten bemerken erhöhte Energie und verbesserte Fitness innerhalb von 3-4 Wochen. Sichtbare physische Veränderungen beginnen typischerweise bei 6-8 Wochen — aber das hängt von Konsistenz, Ernährung und Ihrem Startpunkt ab. Ich werde realistische Meilensteine setzen, damit Sie immer wissen, dass Sie Fortschritt machen.

**Was ist, wenn ich ein absoluter Anfänger bin?**
Perfekt — ich spezialisiere mich auf die Unterstützung von Menschen, die noch nie konsequent trainiert haben. Wir beginnen in Ihrem Tempo, konzentrieren uns auf Selbstbewusstsein und ich erkläre alles klar, damit Sie sich nie verloren fühlen.

**Brauche ich meine eigene Ausrüstung?**
Nein. Ich gestalte Ihr Programm um das herum, was Sie haben — ob Fitnessstudio, Heimgeräte oder nur Körpergewicht.

**Wie beginne ich?**
Buchen Sie einen kostenlosen 20-Minuten-Anruf und wir sprechen über Ihre Ziele, aktuellen Fitnesslevel und den besten Ansatz für Sie. Keine Verpflichtung. → [Hier buchen]
