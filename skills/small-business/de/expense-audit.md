# Expense Audit Skill

## Wann aktivieren
- Sie möchten Ausgaben finden, die Sie kürzen oder neu verhandeln können
- Ihre Kosten scheinen hoch, aber Sie sind sich nicht sicher, wohin das Geld geht
- Durchführung einer jährlichen Überprüfung aller Geschäftsabonnements
- Vorbereitung auf ein Treffen mit Ihrem Buchhalter
- Sie sind schnell gewachsen und vermuten, dass einige Ausgaben nicht mehr notwendig sind

## Wann NICHT verwenden
- Prüfung einzelner Mitarbeiterausgaben (anderer Prozess)
- Steuerkonformitätsprüfungen — verwenden Sie einen lizenzierten Buchhalter
- Betrugsverdacht — beziehen Sie einen Fachmann ein

## Anleitung

### Abonnement- und Software-Audit

```
Hier sind alle Software/Tool-Abonnements, die ich zahle:
[fügen Sie Ihre Liste ein — Name, monatliche Kosten, Funktion, Nutzungshäufigkeit]

Welche sollte ich:
1. Kündigen (nicht verwendet oder doppelt)
2. Downgrade (zahle für Funktionen, die ich nicht brauche)
3. Konsolidieren (zwei Tools machen dasselbe)
4. Nachverhandeln (zahle zu viel, Alternativen existieren)
```

### Überprüfung Bankkontoauszug Ausgaben

```
Hier ist mein Geschäftsbankkontoauszug für [Monat] [fügen Sie ein oder beschreiben Sie Kategorien]:

Klassifizieren Sie alle Ausgaben, identifizieren Sie:
1. Alle wiederkehrenden Gebühren, die ich vielleicht vergessen habe
2. Ungewöhnliche oder einmalige Posten, die überprüft werden müssen
3. Beträge, die höher als erwartet scheinen
4. Mögliche Missklassifizierungen für Steuerzwecke
```

### Die "brauche ich das noch?" Checkliste

```
Ich möchte jede Geschäftsausgabe über $100/Monat überprüfen.
Hier ist meine Liste: [fügen Sie ein]

Stellen Sie mir für jede 3 Fragen, um mir zu entscheiden, ob ich sie noch brauche,
dann geben Sie mir eine Empfehlung: behalten / kündigen / prüfen.
```

### Schnelle Gewinne finden

```
Meine gesamten monatlichen Geschäftsausgaben sind $[X].
Mein Umsatz ist $[Y].
Meine Gewinnmarge scheint dünn.

Welche Ausgabenkategorien sollte ich zuerst für schnelle Einsparungen überprüfen?
Was ist ein angemessener Benchmark für [Geschäftstyp] auf meinem Umsatzniveau?
```

### Jährliche Abonnement-Zurücksetzen

```
Ich möchte alle jährlich gezahlten Geschäftsausgaben überprüfen.
Meine Konten umfassen: QuickBooks, Stripe, Slack, Google Workspace, [andere]

Helfen Sie mir:
1. Alle Tools und ungefähre Kosten auflisten
2. Kostenlose Alternativen für alles finden, das ich minimal nutze
3. Alles finden, das ich doppelt zahle
4. Prioritäten was ich zuerst kündigen würde
```

### Vor einer Wachstumsphase

```
Ich bin dabei, meinen ersten Mitarbeiter einzustellen (Kosten: ~$[X]/Monat).
Meine aktuellen Ausgaben sind $[Y]/Monat, Umsatz $[Z]/Monat.

Welche Ausgaben sollte ich kürzen oder reduzieren, um Platz für diese Einstellung zu schaffen
ohne den Betrieb zu beeinträchtigen?
```

## Beispiel

**Sie fügen Ihre letzten Monatsausgaben ein:**
```
Shopify: $79
Canva Pro: $17
Adobe CC: $55
Mailchimp: $35
ConvertKit: $29 (Testversion)
Zoom: $15
Google Meet (kostenlos, aber Google Workspace bezahlen): $12
Slack: $0 (kostenlos)
Airtable: $24
Notion: $16
ClickUp: $10
Dropbox: $12
Google Drive (in Workspace enthalten)
QuickBooks: $30
FreshBooks: $17 (altes Konto, nicht verwendet)
```

**Claude kennzeichnet:**
- **Sofort kündigen:** FreshBooks ($17) — nicht verwendet, habe QuickBooks
- **Sofort kündigen:** ConvertKit ($29) — in Test, zahle bereits Mailchimp
- **Konsolidieren:** Airtable + Notion + ClickUp — 3 Project-Tools, eines wählen (~$30-34 Einsparungen)
- **Überprüfen:** Dropbox ($12) — Sie haben bereits Google Drive in Workspace
- **Überprüfen:** Adobe CC ($55) — wenn Sie nur Photoshop verwenden, könnten Sie Canva Pro statt dessen nutzen
- **Gesamteinsparungen sofort: $46-80/Monat**
