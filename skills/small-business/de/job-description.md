# Job Description Skill

## Wann aktivieren
- Schreiben einer Stellenanzeige für eine beliebige Position
- Überprüfung eines Stapels Lebensläufe und Identifizierung der Top-Kandidaten
- Erstellung von Interviewfragen für eine bestimmte Rolle
- Verfassung eines Angebotsbriefs oder einer Ablehnungs-E-Mail
- Festlegung, was ein neuer Mitarbeiter täglich tun muss

## Wann NICHT verwenden
- Rechtliche Arbeitsverträge — verwenden Sie einen Arbeitsanwalt
- Redundanz- oder Kündigungsprozesse — konsultieren Sie HR/Recht
- Dokumente zur Unternehmensrichtlinie auf Unternehmensebene

## Anleitung

### Schreiben Sie eine Stellenbeschreibung

```
Schreiben Sie eine Stellenbeschreibung für: [Stellentitel]

Mein Unternehmen: [Typ und kurze Beschreibung]
Ort: [Stadt / Remote / Hybrid]
Vollzeit / Teilzeit: [welcher, und Stunden]
Gehaltsbereich: $[X]–$[Y] oder [Stundensatz] (falls Sie bereit sind zu teilen)
Must-Haves: [3-5 nicht verhandelbare Anforderungen]
Nice-to-Haves: [optionale Fähigkeiten oder Erfahrung]
Was sie täglich tatsächlich tun werden: [beschreiben Sie die Hauptaufgaben]
Wer sie arbeitet mit: [wem sie berichten, wer im Team ist]
Was macht das Arbeiten hier gut: [Kultur, Vergünstigungen, Wachstumschance]
Ton: [professionell / beiläufig / Startup-y / warm]
```

### Überprüfen Sie einen Stapel Lebensläufe

```
Ich stelle für [Stellentitel] ein. Die Hauptanforderungen sind: [3-5 Must-Haves auflisten].
Ich habe [X] Lebensläufe erhalten. Ich werde jeden unten einfügen.
Für jeden: mit 1-5 bewerten, ihre relevante Erfahrung in 2 Sätzen zusammenfassen,
und sagen Sie mir, wen ich für Interviews priorisieren sollte.

[jeden Lebenslauf einfügen]
```

### Erstellen Sie Interviewfragen

```
Ich interviewe Kandidaten für [Stellentitel].
Wichtige Dinge, die ich bewerten muss: [auflisten, was für diese Rolle am meisten zählt]
Art des Unternehmens: [beschreiben]

Erstellen Sie:
- 5 Fragen, um relevante Fähigkeiten/Erfahrung zu bewerten
- 3 Situationsfragen (« erzähle mir von einer Zeit, als... »)
- 2 Kultur/Werte-Fit-Fragen
- 1 Frage, die offenbart, ob sie wirklich an der Rolle interessiert sind

Schlagen Sie auch vor, wie eine « gute Antwort » für jede aussieht.
```

### Schreiben Sie einen Angebo tsbrief

```
Schreiben Sie einen Angebo tsbrief für [Name], den ich als [Rolle] einstelle.
Startdatum: [Datum]
Gehalt/Satz: $[X] [pro Stunde/pro Jahr]
Stunden: [Vollzeit/Teilzeit, spezifische Stunden, falls zutreffend]
Probezeit: [falls zutreffend — z.B. 3 Monate]
Spezifische Bedingungen: [Probezeit, Wettbewerbsverbot, etc.]
Mein Unternehmensname: [Name]
Ton: [warm aber professionell]
```

### Schreiben Sie eine Ablehnungs-E-Mail

```
Schreiben Sie eine freundliche aber klare Ablehnungs-E-Mail für einen Kandidaten, der sich für [Rolle] beworben hat.
Sie: [kurze Notiz — z.B. hatten die Fähigkeiten, aber weniger Erfahrung als ein anderer Kandidat]
Ich möchte: [Tür offen lassen / sie höflich schließen]
Halten Sie es: unter 100 Wörtern, authentisch, nicht roboterhaft.
```

### Definieren Sie die Rolle vor dem Posten

```
Ich muss jemanden einstellen, aber ich bin nicht sicher, welche Rolle ich genau brauche.
Ich habe derzeit Probleme mit: [beschreiben Sie die größten Schmerzpunkte]
Dinge, die ich von meinem Teller herunternehmen möchte: [Liste]
Budget: $[X]/Stunde oder $[X]/Jahr

Helfen Sie mir zu verstehen: Welche Art von Rolle brauche ich wirklich?
Welcher Titel sollten Sie haben? Was sollten sie nicht tun?
```

## Beispiel

**Sie sagen:**
```
Ich leite ein kleines Buchaltungsbüro (3 Personen).
Ich muss einen Administrator/Client-Koordinator einstellen.
Sie würden Zeitplanung verwalten, Kunden-E-Mails bearbeiten, Dokumente jagen, die Client-Datenbank ordentlich halten, Telefon beantworten.
Gehalt: $45.000–$55.000
Must-Haves: organisiert, guter Kommunikator, zuverlässig, kennt Google Workspace
Nicht erforderlich, aber großartig: Buchhaltungskenntnis
Ton: professionell aber freundlich — wir sind ein kleines Team, kein Konzern
```

**Claude schreibt eine vollständige Stellenanzeige** mit: klarem Titel (« Client-Koordinator »), ansprechendem Einleitungsabsatz über das Büro, Rollenübersicht auf Englisch, Verantwortungsliste (8-10 Punkte), in Must-Have/Nice-to-Have aufgeteilte Anforderungen, was sie an der Rolle lieben werden, Gehaltsbereich und direkte Bewerbungsanweisung.
