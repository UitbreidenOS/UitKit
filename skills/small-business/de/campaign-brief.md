# Campaign Brief Skill

## Wann aktivieren
- Sie haben eine Promotion- oder Startidee und benötigen einen strukturierten Plan
- Erstellung einer saisonalen Marketingkampagne (Black Friday, Sommerschlussverkauf, etc.)
- Verfassung eines Briefs für einen Freiberufler, eine Agentur oder einen Marketingmitarbeiter
- Organisation Ihrer eigenen Marketingbemühungen für die nächsten 4-8 Wochen
- Verwandlung von „wir sollten etwas für [Anlass] tun" in einen echten Plan

## Wann NICHT verwenden
- Kampagnenausführung (Claude hilft bei der Planung, dann führen Sie oder Ihr Team aus)
- Detaillierte Medienkäufe oder Kontoverwaltung
- Markenidentitätserstellung von Grund auf

## Anleitung

### Schneller Kampagnen-Brief

```
Helfen Sie mir, ein Marketingkampagnen-Brief zu erstellen.

Unternehmen: [Ihr Unternehmensname und -typ]
Was ich bewerbe: [Produkt/Service/Event/Verkauf]
Ziel: [mehr Kunden / beworbenes Produkt / saisonaler Verkauf / etwas Neues starten]
Budget: $[X] gesamt oder $[X]/Monat
Zeitplan: [wann muss dies beginnen und enden]
Zielkunde: [beschreiben Sie, wen Sie erreichen möchten]
Kanäle, die ich in Betracht ziehe: [Email / Social / Google Ads / Flyer / etc.]
Alles, das ich bereits über funktionieren weiß: [optional]
```

### Saisonale Kampagnenplanung

```
Ich möchte eine [Black Friday / Weihnachts / Sommer / Schulanfang]-Kampagne durchführen.
Mein Unternehmen: [Typ]
Zu bewerbende Produkte/Services: [Liste]
Typische Einnahmen in diesem Zeitraum ohne Kampagne: $[X]
Was ich anbieten kann: [Rabatt / kostenlos Geschenk / Bundle / frühzeitiger Zugang]
Verfügbares Budget: $[X]
Personal zur Hilfe: [nur ich / 1 Assistent / Team von X]

Erstellen Sie einen 4-Wochen-Kampagnenplan mit wöchentlichen Aktionen.
```

### E-Mail-Kampagnenplanung

```
Ich möchte eine E-Mail-Kampagne an meine Liste von [X] Abonnenten durchführen.
Ziel: [Käufe steigern / ein neues Produkt ankündigen / inaktive Kunden wieder einbinden]
Zeitplan: [X Wochen]
Was ich anbieten kann: [beschreiben]

Erstellen Sie eine [3/5/7]-Email-Sequenz mit:
- Betreffzeilen für jede
- Schlüsselbotschaft für jede E-Mail
- Wann jede zu senden ist
- Was das CTA sein sollte
```

### Social-Media-Kampagne

```
Ich möchte eine Kampagne auf [Instagram / Facebook / LinkedIn / TikTok] durchführen.
Dauer: [X Wochen]
Produkt/Angebot: [beschreiben]
Budget für bezahlte Bewerbung: $[X]
Ich kann posten: [täglich / 3x pro Woche / wöchentlich]

Erstellen Sie einen Inhaltskalender mit:
- Nachrichtenthemen nach Woche
- Inhaltstyp (Foto, Video, Karussell, Story, Reel)
- Caption-Winkel für jede
- Wann mit bezahlten Anzeigen zu steigern
```

### Messung des Erfolgs

```
Ich habe letzte Monat eine Kampagne durchgeführt: [beschreiben Sie, was Sie getan haben].
Ergebnisse: [was passiert ist — mehr Verkäufe? neue Kunden? Websitebesuche?]

Hat dies funktioniert? Worauf sollte ich messen, und was wäre für mein Unternehmen „gut"?
```

## Beispiel

**Sie sagen:**
```
Ich betreibe einen Hundefriseursalon.
Ich möchte meine stillen Dienstag-/Mittwochplätze füllen.
Budget: $200.
Meine vorhandenen Tools: Instagram (450 Anhänger), E-Mail-Liste (180 Personen), Google Business-Profil.
Ziel: 10 neue Buchungen in 3 Wochen.
```

**Claude produziert einen Brief:**

**Kampagne: Mid-Week Slots füllen**

**Ziel:** 10 neue Buchungen Di/Mi über 3 Wochen

**Angebot:** „Mid-Week-Spezial — 20% Rabatt auf alle Frisuren Dienstag oder Mittwoch gebucht"

**Woche 1 — Ankündigen**
- E-Mail (Tag 1): Betreff: „Psst — Dienstagplätze sind gerade billiger 🐾" → Angebot ankündigen, Buchungslink einschließen, auf 15 Plätze begrenzt
- Instagram-Post + Story (Tag 2): Vorher/Nachher-Foto + „Buchen Sie einen Mid-Week-Groom, sparen Sie 20%"
- Google Business-Post (Tag 3): Angebot zu Ihrem Profil hinzufügen

**Woche 2 — Sozialer Beweis**
- Instagram: 2 Kundenfotos aus Di/Mi-Buchungen teilen
- Story-Umfrage: „Würden Sie einen Mid-Week-Groom versuchen, wenn es billiger wäre?"
- Top-Instagram-Post steigern: $100 Budget, Hundebesitzer in 10 km Radius anvisieren

**Woche 3 — Dringlichkeit**
- E-Mail (Tag 1): „3 Plätze diese Woche zu 20% Rabatt"
- Instagram Story: Countdown-Sticker + „Füllt sich schnell"
- Restbudget $100: Wer Ihre Website/Instagram besucht hat, erneut anvisieren

**Erfolgskennzahlen:** 10 Buchungen = $X Umsatz zu Ihrem Durchschnittspreis. Verfolgen, von welchem Kanal jede Buchung kommt.
