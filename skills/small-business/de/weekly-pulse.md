# Weekly Pulse Skill

## Wann aktivieren
- Sie möchten eine strukturierte Zusammenfassung der Unternehmensleistung dieser Woche
- Zahlen aus QuickBooks, HubSpot, Shopify und anderen Tools zusammenbringen
- Sie möchten ein einfaches wöchentliches Ritual, das Sie über Ihre Zahlen informiert hält
- Identifizierung von Funktionen und Problemen, die vor Montagmorgen Aufmerksamkeit benötigen
- Erstellen eines Berichts zur Freigabe mit einem Partner, Investor oder Team

## Wann NICHT verwenden
- Formelle Finanzaufstellungen für Kreditgeber oder Investoren (verwenden Sie Ihren Buchhalter)
- Echtzeit-Dashboards (benötigt ein BI-Tool wie Looker oder Tableau)
- Detaillierte forensische Analyse eines bestimmten Problems

## Anleitung

### Ihr wöchentlicher Geschäftspuls

Kopieren Sie diese Vorlage und füllen Sie sie jede Woche aus:

```
Generieren Sie meinen wöchentlichen Geschäftsgesundheitsbericht. Hier sind meine Zahlen:

EINNAHMEN
- Diese Woche Einnahmen: $[X]
- Letzte Woche Einnahmen: $[X]
- Gleiche Woche letztes Jahr: $[X] (falls vorhanden)
- Monatlich bis Datum: $[X]

OPERATIONEN
- Neue Kunden/Clients diese Woche: [X]
- Wiederholungskunden: [X]
- Ausstehende Rechnungen/Außenstände: $[X]
- Beliebige große Gewinne: [beschreiben]
- Beliebige Probleme: [beschreiben]

MARKETING (was du hast)
- Website-Besuche: [X]
- Neue Leads/Anfragen: [X]
- Soziale Medien: [alles Bemerkenswerte]
- Beste Inhaltsleistung: [optional]

BARGELDUMLAUF
- Banksaldo heute: $[X]
- Beliebige großen ausstehenden Zahlungen: [beschreiben]

Sagen Sie mir:
1. Wie verglich sich diese Woche mit der letzten Woche und meinen Trend?
2. Auf welche Nummer sollte ich mich am meisten konzentrieren?
3. Was sollte ich nächste Woche anders machen?
4. Gibt es Risiken, deren ich mir bewusst sein sollte?
```

### Nur die wichtigsten Kennzahlen

```
Ich habe nur 10 Minuten. Hier sind meine Schlüsselzahlen diese Woche:
- Umsatz: $[X] (vs $[X] letzte Woche)
- Neue Kunden: [X]
- Bargeld in Bank: $[X]
- Größtes Problem diese Woche: [beschreiben]

Gib mir eine 5-Satz-Wochenzusammenfassung und 3 Prioritäten für nächste Woche.
```

**Für ein Dienstleistungsunternehmen (Salon, Klinik, Gym):**

- Keine Shows: [X]
- Umsatz pro Termin: $[X]
- Top-Service diese Woche: [Name]

Was sagt mir das über die Gesundheit meines Unternehmens diese Woche?
```

**Für ecommerce/Shopify:**
```
Diese Woche:

Analysieren Sie dies und sagen Sie mir, auf welche Konzentration ich mich nächste Woche konzentrieren sollte.

**Für ein B2B-Dienstleistungsunternehmen:**

Wie verfolge ich gegen ein gesundes Geschäft?

### Monatliche Trendübersicht

```
Hier sind meine letzten 12 Wochen Umsatz [einfügen]:
Woche 1: $X
Woche 2: $X
...

Welche Trends sehen Sie? Saisonalität? Wachstumsrate?
Bei diesem Tempo, was ist mein projizierter Umsatz für die nächsten 3 Monate?
Was sollte ich basierend auf diesem Trend anders tun?
```

## Beispiel

**Sie fügen Ihre Zahlen ein:**
```
Umsatz: $12.400 (letzte Woche $9.800, gleiche Woche letztes Jahr $7.200)
Neue Kunden: 4
Wiederholungskunden: 11
Ausstehende Rechnungen: $8.200 (2 Rechnungen über 30 Tage)
Bank: $6.100
Miete gestern bezahlt $3.200
Großer Gewinn: $6.000 Projekt von einer Überraschung gelandet
Problem: Mein bester Kunde wird ruhig, haben seit 2 Wochen nichts gehört
```

**Wöchentlicher Puls-Bericht von Claude:**

**Week Summary — Strong Revenue, Watch the AR**

Der Umsatz erreichte diese Woche $12.400 — 27% über der letzten Woche und 72% über der gleichen Woche letztes Jahr. Der $6.000 Überraschungsgewinn fällt auf und signalisiert, dass Ihre Überraschungs-Pipeline funktioniert.

**Die Nummer zu beobachten: Ihre ausstehende AR ($8.200).** Mit $6.100 in der Bank und gerade bezahlter Miete stellt diese AR ein echtes Liquiditätsrisiko dar. Verfolgen Sie beide ausstehenden Rechnungen aktiv diese Woche — beginnend mit der größeren.

**Der ruhige Kunde ist ein Risiko.** Zwei Wochen Stille von einem Schlüsselkunden wert eine proaktive Überprüfung. Eine kurze « Nur-Check-In »-E-Mail ist besser als abzuwarten, bis sie abwandern.

**Nächste Woche 3 Prioritäten:**
1. Verfolgen Sie die 2 ausstehenden Rechnungen — visieren Sie bis Freitag
2. Senden Sie einen Check-In an den ruhigen Kunden
3. Nachverfolgung auf der Überraschung, die gerade gelandet ist, um das Projekt zu starten
