# Content Repurposer Skill

## Wann aktivieren
- Sie haben einen Blogbeitrag, ein Podcast-Transkript oder einen Artikel und möchten mehr daraus herausholen
- Eine Woche Social-Inhalte aus einer einzigen Quelle erstellen
- Eine Kundenfallstudie in mehrere Formate umwandeln
- Ein langes Video in Clips, Untertitel und Newsletter umfunktionieren
- Sie haben 30 Minuten und brauchen Inhalte für mehrere Kanäle

## Wann NICHT verwenden
- Neue Inhalte von Grund auf erstellen — beschreiben Sie stattdessen, was Sie möchten
- Konkurrenzinhalte umgestalten — rechtliches Risiko
- Inhalte, die älter als 2 Jahre sind — Genauigkeit zuerst prüfen

## Anleitung

### Der Ein-zu-Viele-Arbeitsablauf

Geben Sie Claude Ihren Quellinhalt und die benötigten Ausgaben:

```
Hier ist ein Blogbeitrag, den ich über [Thema] geschrieben habe: [Inhalt einfügen]

Bitte erstellen Sie:
1. 5 LinkedIn-Posts (professionell, ein Einblick pro Post)
2. 5 Instagram-Captions mit Hashtag-Vorschlägen
3. Eine E-Mail-Newsletter-Version (300 Wörter, umgangssprachlich)
4. Google Ads Überschrift und Beschreibung
5. Ein 60-Sekunden-Videoscript
```

### Formate, die Claude erstellen kann

| Format | Was zu spezifizieren ist |
|---|---|
| LinkedIn-Post | Ton (professionell/persönlich), Länge, Frage einschließen? |
| Instagram-Caption | Vibe (inspirierend/lehrreich/lustig), Hashtag-Anzahl |
| E-Mail-Newsletter | Länge, Betreffzeile, CTA |
| Twitter/X-Thread | Anzahl Tweets, Hook-Stil |
| Facebook-Post | Gemeinschaftsorientiert, umgangssprachlich |
| Google Ads | Zeichenlimits automatisch angewendet |
| YouTube-Beschreibung | SEO-fokussiert, mit Zeitstempeln |
| Videoscript | Dauer, Talking Head oder Voice-over |
| Podcast-Shownotizen | Zusammenfassung + Schlüsselpunkte + Zeitstempel |
| Pinterest Pin | Beschreibung + Schlüsselwort-reich |

### Für ein Dienstleistungsunternehmen

```
Ich bin Klempner. Hier ist eine Fallstudie über eine Arbeit, die ich letzte Woche gemacht habe:
"Reparatur einer geflühlten Leitung für einen Hausbesitzer um 2 Uhr morgens an einem Sonntag, rettete ihre Küche vor Überschwemmung"

Verwandeln Sie dies in:
- Eine Google-Bewertungsanfrage (zum Versenden an den Kunden)
- Ein Facebook-Post für meine Geschäftsseite
- Ein Google Business Post
- Eine kurze E-Mail an meine Newsletter-Liste
```

### Für ein Produktunternehmen

```
Hier ist meine neue Produktbeschreibung: [einfügen]

Erstellen Sie:
- 3 verschiedene Instagram-Captions (lehrreich, Testimonial-Stil, Werbung)
- E-Mail-Betreffzeilenoptionen (5 Variationen)
- Anzeigentext für Facebook (Bewusstsein) und Google (Suchintention)
- Eine Produktstart-Ankündigung für meine E-Mail-Liste
```

### Konsistente Markenstimme

```
Hier ist ein Beispiel von Inhalten, die ich vorher geschrieben habe, die wie ich klingen: [Beispiel einfügen]

Jetzt verarbeiten Sie dies [neuen Inhalt] in derselben Stimme und dem gleichen Stil neu.
```

### Inhaltskalender aus einem Stück

```
Ich habe diesen Artikel geschrieben: [einfügen]

Erstellen Sie einen 2-Wochen-Plan für Social-Media-Inhalte nur mit diesem Artikel als Quelle.
Mischformate: Tipps, Zitate, Fragen, Hinter-den-Kulissen-Takes.
Plattform: LinkedIn + Instagram.
```

## Beispiel

**Sie fügen einen 600-Wort-Blogbeitrag über "5 Gründe, warum lokale Unternehmen Kunden verlieren" ein**

**Claude produziert:**
- 5 LinkedIn-Posts (einer pro Grund, jeder mit Hook + Einblick + Frage)
- E-Mail: Betreffzeile "Begehen Sie diese 5 Fehler?" + Newsletter-Version
- 3 Instagram-Captions mit verschiedenen Winkeln
- Google Ads: 3 Überschriften + Beschreibungskombinationen
- 60-Sekunden-Script: "Wenn Sie ein Kleinunternehmer sind und Kunden verlieren, ist hier, was wahrscheinlich passiert..."
