# Job Description Skill

## Wanneer activeren
- Schrijf een vacatureadvertentie voor een rol
- Controleer een stapel CV's en identificeer topcandidaten
- Maak interviewvragen voor een specifieke rol
- Schrijf een aanbiedenbrief of afwijzings-e-mail
- Definieer wat een nieuwe aangestelde daadwerkelijk moet doen

## Wanneer NIET gebruiken
- Juridische arbeidscontracten — gebruik een arbeidsadvocaat
- Redundantie- of beëindigingsprocessen — raadpleeg HR/juridisch
- Bedrijfsniveaubeleidsdo cumenten

## Instructies

### Schrijf een vacaturetekst

```
Schrijf een vacaturetekst voor: [functtitel]

Mijn bedrijf: [type en korte beschrijving]
Locatie: [stad / remote / hybride]
Voltalijds / deeltijd: [welke, en uren]
Salarisbereik: $[X]–$[Y] of [uurloon] (als je bereid bent te delen)
Must-haves: [3-5 niet-onderhandelbare vereisten]
Nice-to-haves: [optionele vaardigheden of ervaring]
Wat ze daadwerkelijk dagelijks zullen doen: [beschrijf de hoofdtaken]
Met wie ze zullen werken: [aan wie rapporteren, wie is op het team]
Wat maakt werken hier goed: [cultuur, voordelen, groeikans]
Toon: [professioneel / terloops / startup-y / warm]
```

### Controleer een stapel CV's

```
Ik ben aan het inhuren voor [functietitel]. De sleutelvaardigheden zijn: [3-5 must-haves opsommen].
Ik heb [X] CV's ontvangen. Ik zal elk hieronder plakken.
Voor elk: score 1-5, samenvat hun relevante ervaring in 2 zinnen,
en zeg me wie ik moet prioriteren voor interviews.

[elke CV eenmaaal plakken]
```

### Maak interviewvragen

```
Ik interview kandidaten voor [functietitel].
Belangrijke dingen die ik moet beoordelen: [wat het meest telt voor deze rol]
Type bedrijf: [beschrijf]

Maak:
- 5 vragen om relevante vaardigheden/ervaring te beoordelen
- 3 situatievragen (« vertel me van een moment waarin... »)
- 2 cultuur/waardevragen
- 1 vraag die onthult of ze echt geïnteresseerd zijn in de rol

Stel ook voor hoe een « goed antwoord » eruitziet voor elk.
```

### Schrijf een aanbiedenbrief

```
Schrijf een aanbiedenbrief voor [naam], die ik als [rol] inhuur.
Startdatum: [datum]
Salaris/tarief: $[X] [per uur/per jaar]
Uren: [voltijd/deeltijd, specifieke uren indien van toepassing]
Proeftijd: [indien van toepassing — bijv. 3 maanden]
Specifieke voorwaarden: [proeftijd, concurrentiebeding, etc.]
Mijn bedrijfsnaam: [naam]
Toon: [warm maar professioneel]
```

### Schrijf een afwijzings-e-mail

```
Schrijf een vriendelijke maar duidelijke afwijzings-e-mail voor een kandidaat die soliciteerde voor [rol].
Zij: [korte notitie — bijv. hadden vaardigheden maar minder ervaring dan andere kandidaat]
Ik wil: [deur openhouden / het beleefd sluiten]
Houd het: onder 100 woorden, authentiek, niet robotachtig.
```

### Definieer de rol voordat u post

```
Ik moet iemand inhuren maar ben niet zeker welke rol ik precies nodig heb.
Ik heb momenteel moeite met: [beschrijf de grootste pijnpunten]
Dingen die ik van mijn bord af wil: [lijst]
Budget: $[X]/uur of $[X]/jaar

Help me erachter te komen: welk soort rol heb ik echt nodig?
Welke titel zouden ze moeten hebben? Wat zouden ze niet moeten doen?
```

## Voorbeeld

**U zegt:**
```
Ik leid een klein boekhoudkantoor (3 personen).
Ik heb een admin/klantcoördinator nodig.
Zij zouden planning beheren, clientmails afhandelen, documenten jagen, clientdatabase netjes houden, telefoon beantwoorden.
Salaris: $45.000–$55.000
Must-haves: georganiseerd, goed communicator, betrouwbaar, kent Google Workspace
Niet vereist maar geweldig: boekhoudkundige kennis
Toon: professioneel maar vriendelijk — wij zijn een klein team, geen bedrijf
```

**Claude schrijft een volledige vacaturetekst** met: duidelijke titel (« Klantcoördinator »), aantrekkelijke inleidingsparagraaf over het kantoor, roleoverzicht in duidelijk Engels, verantwoordelijkheidlijst (8-10 punten), vereisten onderverdeeld in must-have/nice-to-have, wat ze leuk vinden aan de rol, salarisbereik en directe sollicitatierichtlijnen.
