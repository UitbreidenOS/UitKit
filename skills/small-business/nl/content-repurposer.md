# Content Repurposer Skill

## Wanneer activeren
- U hebt een blogpost, podcast-transcript of artikel en wilt er meer uit halen
- Een week sociale inhoud van een enkele bron maken
- Een klantcasestudy in meerdere formaten omzetten
- Een lange video in clips, ondertitels en nieuwsbrief hergebruiken
- U hebt 30 minuten en hebt inhoud voor meerdere kanalen nodig

## Wanneer NIET gebruiken
- Geheel nieuwe inhoud van nul af aan maken — beschrijf liever wat u wilt
- Concurrentie-inhoud hergebruiken — juridisch risico
- Inhoud ouder dan 2 jaar — eerst nauwkeurigheid controleren

## Instructies

### De één-naar-veel workflow

Geef Claude uw bron-inhoud en benodigde outputs:

```
Hier is een blogpost die ik over [onderwerp] heb geschreven: [plak inhoud]

Alstublieft, maak:
1. 5 LinkedIn-berichten (professioneel, één inzicht per bericht)
2. 5 Instagram-onderschriften met hashtag-suggesties
3. Een e-mailnieuwsbrief-versie (300 woorden, conversationeel)
4. Google Ads headline en beschrijving
5. Een 60-seconden videoscript
```

### Formaten die Claude kan produceren

| Format | Wat te specificeren |
|---|---|
| LinkedIn-bericht | Toon (professioneel/persoonlijk), lengte, vraag opnemen? |
| Instagram-onderschrift | Vibe (inspirerend/educatief/leuk), hashtag-aantal |
| E-mailnieuwsbrief | Lengte, onderwerpregel, CTA |
| Twitter/X-thread | Aantal tweets, hook-stijl |
| Facebook-bericht | Gemeenschapsgericht, conversationeel |
| Google Ads | Tekenlimits automatisch toegepast |
| YouTube-beschrijving | SEO-gericht, met tijdstempels |
| Videoscript | Duur, sprekende kop of voice-over |
| Podcast-shownotes | Samenvatting + sleutelpunten + tijdstempels |
| Pinterest pin | Beschrijving + trefwoord-rijk |

### Voor een servicebedrijf

```
Ik ben loodgieter. Hier is een caseStudy over een klus die ik vorige week heb gedaan:
"Reparatie van een gebarsten pijp voor een huiseigenaar om 2 uur 's ochtends op zondagmorgen, redde hun keuken voor overstroming"

Zet dit om in:
- Een Google-reviewverzoek (om naar de klant te sturen)
- Een Facebook-bericht voor mijn bedrijfspagina
- Een Google Bedrijfspost
- Een korte e-mail naar mijn mailinglijst
```

### Voor een productbedrijf

```
Hier is mijn nieuwe productbeschrijving: [plak]

Maak:
- 3 verschillende Instagram-onderschriften (educatief, testimonial-stijl, promotioneel)
- E-mail onderwerpregel opties (5 variaties)
- Advertentietekst voor Facebook (bewustzijn) en Google (zoekvraag)
- Een productlanceringsaankondiging voor mijn mailinglijst
```

### Consistente merkstem

```
Hier is een voorbeeld van inhoud die ik eerder heb geschreven die klinkt als ik: [plak voorbeeld]

Hergebruik nu dit [nieuwe inhoud] in dezelfde stem en stijl.
```

### Inhoudskalender van één stuk

```
Ik heb dit artikel geschreven: [plak]

Maak een 2-weken plan voor sociale media inhoud met alleen dit artikel als bron.
Mix formaten: tips, quotes, vragen, behind-the-scenes takes.
Platform: LinkedIn + Instagram.
```

## Voorbeeld

**U plakt een 600-woord blogpost over "5 redenen waarom lokale bedrijven klanten verliezen"**

**Claude produceert:**
- 5 LinkedIn-berichten (één per reden, elk met hook + inzicht + vraag)
- E-mail: onderwerpregel "Begaat u deze 5 fouten?" + nieuwsbriefversie
- 3 Instagram-onderschriften met verschillende invalshoeken
- Google Ads: 3 headline + beschrijvingscombinaties
- 60-seconden script: "Als u een kleine bedrijfsigenaar bent die klanten verliest, dit is wat waarschijnlijk gebeurt..."
