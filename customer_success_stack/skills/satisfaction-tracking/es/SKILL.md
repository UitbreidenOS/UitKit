# Skill: Seguimiento de Satisfacción

## Cuándo activar

Después de la interacción o encuesta trimestral; agregar NPS y sentimiento.

## Cuándo NO usar

No rastrear comentarios anónimos sin consentimiento del cliente.

## Instrucciones

1. Recopilar puntuaciones de NPS, CSAT y esfuerzo
2. Analizar comentarios de texto para sentimiento
3. Agregar por cuenta y cohorte
4. Marcar cuentas con tendencias decrecientes

## Ejemplo

```
/nps-check --customer=acme-corp
→ NPS: 42 (↓ -8 desde Q1)
→ Tema clave: "incorporación lenta"
```
