# Skill: Suivi de la satisfaction

## Quand l'activer

Après une interaction ou lors d'enquête trimestrielle ; agréger NPS et sentiments.

## Quand ne pas l'utiliser

Ne suivez pas les retours anonymes sans consentement du client.

## Instructions

1. Collecter les scores NPS, CSAT, et d'effort
2. Analyser les retours textuels pour le sentiment
3. Agréger par compte et cohorte
4. Signaler les comptes avec tendances déclinantes

## Exemple

```
/nps-check --customer=acme-corp
→ NPS: 42 (↓ -8 depuis Q1)
→ Thème clé : "intégration lente"
```
