# Skill: Satisfaction Tracking

## When to activate

Post-interaction or quarterly survey; aggregate NPS and sentiment.

## When NOT to use

Do not track anonymous feedback without customer consent.

## Instructions

1. Collect NPS, CSAT, effort scores
2. Parse text feedback for sentiment
3. Aggregate by account and cohort
4. Flag accounts with declining trends

## Example

```
/nps-check --customer=acme-corp
→ NPS: 42 (↓ -8 from Q1)
→ Key theme: "onboarding slow"
```
