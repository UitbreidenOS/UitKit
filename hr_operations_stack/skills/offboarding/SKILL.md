# Offboarding Conductor

## When to activate

Resignation notice received, termination approved by counsel, or retirement triggered. Requires exit date, reason for departure, final payroll data, and compliance review completion.

## When NOT to use

Not for internal transfer (use onboarding-optimizer). Not without compliance auditor review if termination. Not without legal approval for severance or disputed departures.

## Offboarding Framework

### Exit Checklist by Departure Type

#### Resignation (Voluntary, Good Standing)
- [ ] Exit interview scheduled (HR, neutral third party, within 2 weeks)
- [ ] Knowledge transfer plan created (30–60 day window, documented)
- [ ] Access deprovisioning scheduled (last day COB)
- [ ] Final paycheck prepared (includes earned PTO payout per state law)
- [ ] Benefits notification (COBRA eligibility, 401(k) rollover, insurance continuation)
- [ ] Reference check form signed (optional; document if declined)
- [ ] Exit survey sent (anonymous feedback, retention learnings)
- [ ] Alumni network opt-in offered (referral incentive if applicable)
- [ ] Final paperwork signed (confidentiality reaffirmation if applicable, non-disparagement)

#### Termination (Performance, Reduction in Force, Layoff)
- [ ] Compliance auditor approval completed (legal review signed off)
- [ ] Severance package approved (amount, release of claims, reference contingency)
- [ ] Termination meeting script prepared (HR, manager, timing, message)
- [ ] Final paycheck prepared (including severance, accrued PTO, taxes calculated)
- [ ] WARN Act notice if applicable (60 days for mass layoffs, federal + state requirements)
- [ ] Benefits notification (COBRA continuation, coverage end date)
- [ ] Unemployment insurance claim filed (documentation for challenge, if applicable)
- [ ] Reference policy documented (no reference, limited reference, or neutral reference)
- [ ] Separation agreement signed (severance conditional on release of claims)
- [ ] Exit interview (optional; may not be appropriate for performance termination)
- [ ] Legal hold if applicable (preserve emails, documents per counsel instruction)
- [ ] Intra-company communication script (team announcement, impact assessment)

#### Retirement
- [ ] Retirement meeting scheduled (congratulations, transition plan, legacy discussion)
- [ ] Pension/401(k) rollover coordinated (timeline, beneficiary confirmation)
- [ ] Phased retirement option if applicable (part-time consulting, mentorship, board access)
- [ ] Knowledge transfer plan (final 30–60 days, mentoring successor)
- [ ] Benefits continuation (health insurance bridge, COBRA election)
- [ ] LinkedIn/public profile review (company page mention, alumni network)
- [ ] Exit celebration/recognition (team gathering, farewell email)
- [ ] Final paperwork (beneficiary confirmations, non-disparagement optional)

### Knowledge Transfer Protocol
- **Timing:** Begin 2–4 weeks before departure; critical for 30–60 days post-exit if complex role
- **Documentation:** Wikis, runbooks, project status, password/tool access handoff
- **Pairing sessions:** Successor shadows or works alongside departing employee (recorded if permitted)
- **Customer/stakeholder handoff:** Departing employee attends final meetings if appropriate
- **Lessons learned:** Document key decisions, vendor relationships, strategic context

### Data & Legal Holds
- **Legal hold:** If litigation or investigation pending, notify IT to preserve emails/files
- **Confidentiality:** Reaffirm non-disclosure agreements, non-solicitation, non-compete (if applicable, state-specific)
- **Intellectual property:** Assign any work product, patents, or inventions per employment agreement
- **Personal data deletion:** Honor data subject requests if applicable (GDPR, CCPA); delete upon exit if consented

### Payroll & Final Payment
- **Calculation:** Base salary (through last day), accrued PTO (per state law), bonuses (if earned), severance (if applicable)
- **Deductions:** Normal taxes, 401(k) contributions, benefit premiums, outstanding loans (if applicable)
- **Timing:** Final check within 30 days of termination (state-dependent; some require immediate)
- **Documentation:** Itemized pay stub, severance letter, benefits summary, tax forms (W-2, 1099 if applicable)
