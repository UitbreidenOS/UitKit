# script-optimizer

## When to activate

After topic analysis (GO score ≥60) and competitor research. Generates SEO-optimized script with hook, chapters, keywords, CTAs, and pacing.

## When NOT to use

- Before topic validation (use topic-analyzer first).
- On topics that failed viability check (score <40).
- Without competitor analysis (use competitor-analyzer first).

## Instructions

1. **Input:** Topic, keywords, target audience, video length (8–20 min), competitor hooks.
2. **Generate hook (0–15 sec):** Promise specific outcome. No clickbait. Reference data or real example.
3. **Build script structure:**
   - Hook (15 sec)
   - Problem statement (30 sec)
   - 3–5 main sections (chunks of 1–3 min each)
   - Case study or real example (embedded in section)
   - Call-to-action (15 sec)
4. **SEO optimization:**
   - Primary keyword in title (1st 3 words if possible)
   - Secondary keywords naturally in script (2–3 mentions)
   - Long-tail keywords in chapters
   - Keyword density 1–3% (avoid stuffing)
5. **Pacing:** ~150 words per minute; conversational tone; vary sentence length.
6. **CTAs:** Subscribe + bell, click link in description, join community — max 2 per video.
7. **Output:** Full script with chapter timestamps, embedded keywords (highlighted), keyword density report, and 3 thumbnail concepts.

## Example

**Input:** Topic: "AI video editing automation"; 12-min video; Keywords: AI editing, automation, content creators

**Output:**

```
[10:30] Outro
```
