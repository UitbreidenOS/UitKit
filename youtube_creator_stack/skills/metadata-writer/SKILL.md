# metadata-writer

## When to activate

After script finalization and thumbnail approval. Generates SEO-optimized title, description, tags, and chapters.

## When NOT to use

- Before script is complete (metadata must reflect final content).
- Without keyword research (use topic-analyzer for keyword list).
- On unpublished videos (metadata aligns with final script).

## Instructions

1. **Input:** Final script, keywords (primary, secondary, long-tail), video topic, audience hooks.
2. **Generate title (50–60 characters):**
   - Primary keyword in first 3 words
   - Question or number format (increases CTR)
   - Avoid clickbait; promise specific outcome
   - Examples: "AI Edits Videos For You (20 Hours Saved)" or "How to Use ChatGPT for Coding (2024)"
3. **Generate description (2000 characters max):**
   - First 150 characters visible before "read more" (hook here)
   - Keywords naturally distributed (2–3 primary, 2–3 secondary)
   - Timestamps for chapters
   - Links to resources mentioned in video
   - Call-to-action (subscribe, join community)
   - Disclaimer or attribution if applicable
4. **Generate tags (10–15 tags):**
   - Exact match tags (primary keywords)
   - Phrase match tags (variations)
   - Broad match tags (audience demographic)
   - Competitor tags (if appropriate)
5. **Generate chapters (3–8 chapters):**
   - 0:00 Intro
   - Key section timestamps
   - Chapters drive 50% longer watch time
6. **Output:** Title, full description with timestamps, tag list (ranked by search volume), chapters.
